# Генератор логотипов

![Готовые макеты логотипов](/assets/docs-pages/blog/update-8.2-1.png)

Генератор логотипов позволяет быстро и просто создать логотип, используя магию **img2img**. Модель смотрит на ваш макет и изменяет его по промпту, а вы получаете готовый логотип.

Генератор использует модели, которые генерируют текст на русском языке — они отмечены категорией **RU TEXT** в каталоге моделей.

## Как это работает

1. Введите кириллическое название вашего бренда
2. Выберите один из готовых макетов
3. При необходимости отредактируйте макет

![Редактор макета логотипа](/assets/docs-pages/blog/update-8.2-2.png)

4. Выберите модель из категории **RU TEXT** — эти модели поддерживают генерацию текста на русском языке
5. Напишите промпт для стилизации и запустите генерацию

![Стилизация логотипа](/assets/docs-pages/blog/update-8.2-3.png)

![Готовый логотип](/assets/docs-pages/blog/update-8.2-4.png)
