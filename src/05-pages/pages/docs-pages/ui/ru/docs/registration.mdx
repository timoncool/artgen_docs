# Вход и регистрация

Добро пожаловать в удивительный мир цифрового творчества на [ArtGeneration.me](https://artgeneration.me), где каждый может стать художником с помощью искусственного интеллекта. Чтобы начать своё путешествие, вам необходимо войти или создать аккаунт.

На главной странице вы найдёте кнопку "Начать творить" – ваш первый шаг к созданию уникальных изображений.

## Вход по email (OTP)

![Форма входа](/assets/docs-pages/blog/update-8.3-1.png)

Основной способ входа — через одноразовый пароль (OTP). Вам не нужно запоминать пароль — каждый раз на почту приходит новый код.

import { Steps, Callout } from '@/src/01-shared/ui/mdx'

<Steps>
### Введите email адрес

Укажите свою почту в поле авторизации. Подойдёт любой почтовый сервис — Яндекс, Mail.ru, Gmail, Outlook и другие.

### Нажмите «Получить код»

После прохождения проверки Cloudflare на вашу почту будет отправлен одноразовый код.

### Получите код на почту

![Письмо с одноразовым кодом](/assets/docs-pages/blog/update-8.3-2.png)

На указанный email придёт письмо с 6-значным кодом. Код действителен **10 минут**.

### Введите код

![Ввод одноразового кода](/assets/docs-pages/blog/update-8.3-3.png)

Введите полученный код в поле ввода. После успешной проверки вход произойдёт автоматически.
</Steps>

<Callout type="info">
  Если вы входите впервые — аккаунт будет создан автоматически. Отдельная регистрация не требуется.
</Callout>

## Быстрая авторизация через OAuth

Помимо входа по email, доступна быстрая авторизация через социальные сети:

- **Яндекс ID** — нажмите «Войти с Яндекс ID»
- **VK ID** — нажмите «Войти с VK ID»
- **Одноклассники** — нажмите иконку ОК
- **Mail.ru** — нажмите иконку @

<Callout type="info">
  **Преимущества OAuth:** вход в один клик, автоматическое заполнение профиля, можно привязать несколько способов входа к одному аккаунту.
</Callout>

## Зачем регистрироваться?

Создание учетной записи даёт вам возможность:

- Сохранять ваши работы
- Получать обновления
- Участвовать в сообществе
- Наслаждаться персонализированным опытом на платформе
- Подписываться на других участников и следить за их обновлениями

Если вам понадобится помощь — наша служба поддержки support@artgeneration.me всегда к вашим услугам.
