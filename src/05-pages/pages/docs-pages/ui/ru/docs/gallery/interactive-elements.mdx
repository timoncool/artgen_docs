# Интерактивные элементы

![Интерактивные элементы](/assets/docs-pages/screens/image-rate-popup.png)

Галерея содержит множество интерактивных элементов для взаимодействия с контентом.

## Кнопки действий

- **Лайк** — отметить изображение как понравившееся
- **Избранное** — добавить в коллекцию избранного
- **Поделиться** — отправить ссылку на изображение
- **Скачать** — загрузить изображение на устройство

## Социальные функции

<Callout type="info">
  Подписывайтесь на понравившихся авторов, чтобы не пропустить их новые работы.
</Callout>

- **Подписка на автора** — следить за новыми работами пользователя
- **Комментарии** — оставить отзыв под изображением
- **Репост** — поделиться в социальных сетях

## Массовые операции

Галерея поддерживает массовые операции для управления несколькими изображениями одновременно.

### Выбор нескольких изображений

![Режим массового выбора](/assets/docs-pages/screens/bulk-delete-selected.png)

Для выбора нескольких изображений удерживайте клавишу Ctrl (Cmd на Mac) и кликайте на нужные изображения. Выбранные изображения будут выделены рамкой.

### Панель выбора

![Панель массового выбора](/assets/docs-pages/screens/bulk-selection-bar.png)

При выборе нескольких изображений появляется панель выбора:
- **Счетчик выбранных** — количество выбранных изображений
- **Кнопка удаления** — удалить выбранные изображения из коллекции
- **Иконки быстрых действий** — удалить, увеличить, показать/скрыть, развернуть

### Меню массовых действий

![Меню массовых действий](/assets/docs-pages/screens/gallery-bulk-select.png)

Нажмите на выпадающее меню «Действие» для доступа к массовым операциям:
- **Удалить** — удалить все выбранные изображения сразу
- Подтвердите или отмените выбор с помощью кнопок галочки и крестика

## Фильтр по цвету

![Диалог выбора цветов](/assets/docs-pages/screens/gallery-bulk-delete-menu.png)

Используйте фильтр по цвету для поиска изображений с определенной цветовой палитрой:
- Выберите до 4 цветов для фильтрации
- Нажмите «Применить» для применения фильтра
- Используйте опцию прозрачности для изображений с прозрачным фоном

## Контекстное меню

Правый клик на изображении открывает дополнительные опции:

- Открыть в новой вкладке
- Скопировать ссылку
- Пожаловаться на контент
