# Получение генераций

![Получение генераций](/assets/docs-pages/screens/daily-reward-popup.png)

Генерации — это основная валюта на платформе ArtGeneration.me. Каждая генерация позволяет создать одно изображение с помощью искусственного интеллекта.

## Способы получения генераций

### Бесплатные генерации

- **Ежедневные генерации** — каждый день вы получаете определённое количество бесплатных генераций
- **Приветственный бонус** — новые пользователи получают 100 несгораемых генераций и **3 дня PRO подписки** при регистрации

<Callout type="info">
  Приветственный бонус включает 100 несгораемых генераций и 3 дня PRO подписки — отличный способ познакомиться со всеми возможностями сервиса без каких-либо затрат!
</Callout>

### Платные генерации

![Тарифы Pro и Max](/assets/docs-pages/guide/guide-01.png)

- **Подписка MAX** — 600 генераций в день + 182 модели + до 8 изображений за раз
- **Подписка PRO** — 300 генераций в день + 154 модели + до 4 изображений за раз
- **Покупка пакетов** — дополнительные генерации можно приобрести отдельно

## Типы генераций

1. **Ежедневные** — обновляются каждый день, не накапливаются
2. **Несгораемые** — сохраняются до использования, не имеют срока действия

<Callout type="warning">
  Ежедневные генерации не накапливаются! Обязательно забирайте их каждый день, чтобы не потерять.
</Callout>
