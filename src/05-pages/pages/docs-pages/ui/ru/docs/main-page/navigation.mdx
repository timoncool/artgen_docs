# Навигация

![Навигация](/assets/docs-pages/screens/gallery-header-full.png)

Навигационное меню расположено в верхней части страницы и обеспечивает быстрый доступ ко всем основным разделам сайта.

<Callout type="info">
  Клик на логотип ArtGeneration.me из любого места сайта вернёт вас на главную страницу.
</Callout>

## Элементы навигации

- **Логотип** — возврат на главную страницу
- **Галерея** — просмотр работ сообщества
- **Создать** — переход к генератору изображений
- **Редактор** — инструменты редактирования изображений
- **О сервисе** — информация о платформе
- **Руководство** — документация и справка

## Меню пользователя

В правой части навигации находится меню пользователя с доступом к:

- Личному кабинету
- Настройкам аккаунта
- Истории генераций
- Балансу и подпискам
