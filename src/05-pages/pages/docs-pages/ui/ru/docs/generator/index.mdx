# Генератор изображений

![Генератор изображений ArtGeneration.me](/assets/docs-pages/screens/generator-right-panel.png)

Генератор изображений — это основной инструмент ArtGeneration.me, позволяющий создавать уникальные изображения с помощью искусственного интеллекта.

Здесь вы можете воплотить любые творческие идеи в визуальную форму.

## Интерфейс генератора

Интерфейс разделён на три основные области:

1. **Центральная область** — результаты генерации и поле ввода запроса
2. **Левая панель** — история генераций и быстрые действия
3. **Правая панель** — настройки модели, стилей и параметров

![Правая панель](/assets/docs-pages/screens/generator-right-panel2.png)

## Поле ввода промпта

![Поле ввода промпта](/assets/docs-pages/screens/prompt-input.png)

Нижняя панель генератора содержит:
- **Поле для ввода промпта** — опишите желаемое изображение
- **Кнопка «Сбросить»** — очистить текущие настройки
- **Кнопка «Создать»** — запустить генерацию

### Вспомогательные кнопки

Рядом с полем ввода расположены дополнительные кнопки:

- **Кубик (🎲)** — генерирует случайный промпт. Полезно для вдохновения или когда не знаете, с чего начать.
- **Ластик (🧹)** — очищает поле ввода промпта одним кликом.
- **Автопереводчик** — автоматически переводит ваш промпт на английский язык.

<Callout emoji="💡">
**Автопереводчик** особенно полезен, если вы пишете на русском языке. Модели генерации лучше понимают английские промпты, поэтому автоматический перевод может улучшить качество результата.
</Callout>

<Callout type="info">
**Советы:**
- Используйте **Ctrl + Enter** для быстрого запуска генерации
- Рекомендуется писать промпты на **английском языке** для лучших результатов
- Промпты на русском тоже поддерживаются — используйте автопереводчик для конвертации
</Callout>

## Просмотр результата

![Просмотр одного изображения](/assets/docs-pages/screens/single-image-view.png)

После генерации изображение отображается в центральной области. Доступны:
- Кнопка «Обрезать» для кадрирования
- Поле промпта для создания вариаций
- Кнопка «Создать» для новой генерации

## Оценка изображений

![Оценка изображения](/assets/docs-pages/screens/image-rate-popup.png)

В галерее вы можете оценивать изображения других пользователей:
- **Лайк** — отметить понравившееся изображение
- **Дизлайк** — если изображение не понравилось

## Видимость изображений

![Переключатель видимости](/assets/docs-pages/screens/visibility-toggle.png)

Для каждого изображения можно настроить видимость:
- **Видно всем** — изображение отображается в публичной галерее
- **Приватное** — видно только вам

## Язык запросов

Генератор поддерживает запросы на **русском** и **английском** языках.

### Примеры запросов

**На русском:**
- "Красивый закат над морем в стиле импрессионизма"
- "Портрет девушки с цветами в волосах"

**На английском:**
- "Beautiful sunset over the ocean in impressionist style"
- "Portrait of a girl with flowers in her hair"

## Советы по запросам

<Steps>
### Будьте конкретны
Чем детальнее описание, тем точнее результат.

### Указывайте стиль
Добавляйте художественный стиль или технику.

### Описывайте освещение
Используйте термины: "мягкий свет", "контровой свет", "золотой час".

### Добавляйте детали
Описывайте текстуры, материалы, атмосферу.

### Используйте негативный промпт
Исключайте нежелательные элементы.
</Steps>

## Дополнительные действия

После генерации изображения доступны следующие действия:

- **Скачать** — сохранить изображение на устройство
- **Улучшить** — увеличить разрешение с помощью AI
- **Редактировать** — открыть в редакторе для доработки
- **Face Swap** — заменить лицо на изображении
- **Создать вариации** — сгенерировать похожие изображения
- **Поделиться** — опубликовать в галерее

## Генератор логотипов

![Генератор логотипов](/assets/docs-pages/blog/update-8.2-5.png)

Генератор логотипов позволяет быстро и просто создать логотип, используя магию **img2img**. Модель смотрит на ваш макет и изменяет его по промпту, а вы получаете готовый логотип.

### Как это работает

1. Введите кириллическое название вашего бренда
2. Выберите один из готовых макетов
3. При необходимости отредактируйте макет
4. Выберите модель из категории **RU TEXT** — эти модели поддерживают генерацию текста на русском языке
5. Напишите промпт для стилизации и запустите генерацию

<Callout emoji="💡">
Экспериментируйте с разными моделями категории **RU TEXT** и промптами — каждая модель даёт свой уникальный стиль логотипа.
</Callout>
