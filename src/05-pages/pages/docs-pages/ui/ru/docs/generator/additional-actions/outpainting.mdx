# Расширение границ (Outpainting)

![Outpainting](/assets/docs-pages/screens/outpainting-menu.png)

Outpainting позволяет расширить изображение за его текущие границы, добавляя новый контент в выбранных направлениях.

## Как работает

AI анализирует существующее изображение и генерирует продолжение:
- Сохраняет стиль и атмосферу
- Продолжает композицию
- Добавляет логичные элементы

## Направления расширения

Вы можете расширить изображение в любом направлении:
- **Дорисовать сверху** — добавить контент над изображением
- **Дорисовать слева** — расширить влево
- **Дорисовать справа** — расширить вправо
- **Дорисовать снизу** — добавить контент под изображением

## Как использовать

<Steps>
### Наведите на изображение
Выберите изображение для расширения.

### Нажмите кнопку Outpainting
Откроется меню выбора направления.

### Выберите направление
Укажите, в какую сторону расширить изображение.

### Измените промпт (опционально)
При необходимости вы можете изменить промпт, чтобы указать желаемое содержимое расширенной области.

### Получите результат
AI сгенерирует продолжение изображения.
</Steps>

## Ограничения по количеству расширений

<Callout type="warning">
**Важно:** Качество изображения снижается с каждым последовательным расширением.

- Максимум **~4 расширения** возможно для одного изображения
- **Рекомендуется не более 2 расширений** для сохранения хорошего качества
- При большем количестве расширений качество значительно ухудшается
</Callout>

## Применение

- Изменение соотношения сторон
- Добавление пространства для текста
- Расширение композиции
- Создание панорамных изображений

## Советы

<Callout type="info">
- Начинайте с небольших расширений
- Используйте модификацию промпта для контроля содержимого
- Ограничьтесь 1-2 расширениями для лучшего качества
- Проверяйте согласованность добавленного контента
</Callout>
