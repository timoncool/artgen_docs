# Своё изображение

![Своё изображение](/assets/docs-pages/screens/own-image-panel.png)

Функция "Своё изображение" позволяет использовать сгенерированное или загруженное изображение как основу для новой генерации (img2img).

## Как использовать

<Steps>
### Наведите на изображение
Выберите изображение в результатах генерации.

### Нажмите кнопку "Своё изображение"
Изображение будет добавлено в панель настроек.

### Настройте силу влияния
Определите степень влияния исходного изображения.

### Введите промпт
Добавьте новый или измените существующий промпт.

### Запустите генерацию
Нажмите кнопку "Создать".
</Steps>

## Сила влияния

Параметр "Сила влияния" определяет, насколько результат будет похож на исходное изображение:

<Tabs items={['0.1-0.3', '0.4-0.6', '0.7-0.9']}>
  <Tab>
    **Минимальные изменения.** Сохранение структуры и композиции оригинала.
  </Tab>
  <Tab>
    **Умеренные изменения.** Баланс между оригиналом и промптом.
  </Tab>
  <Tab>
    **Значительные изменения.** Промпт доминирует над исходным изображением.
  </Tab>
</Tabs>

## Применение

- Создание вариаций понравившегося изображения
- Изменение стиля существующего изображения
- Добавление или удаление элементов
- Улучшение композиции
