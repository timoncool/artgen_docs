# Восстановление + Увеличение

![Восстановление + Увеличение](/assets/docs-pages/screens/image-actions-toolbar.png)

Комбинированная функция, объединяющая восстановление лица и увеличение разрешения в одно действие.

## Преимущества

<Callout type="info">
**Одним кликом** выполняются сразу две операции:
1. Восстановление лиц
2. Увеличение разрешения в 2 раза
</Callout>

Это экономит время и упрощает процесс обработки портретных изображений.

## Как использовать

<Steps>
### Наведите на изображение
Выберите изображение с лицами, которое нужно улучшить.

### Нажмите кнопку "Восстановление + Увеличение"
Одним кликом запустите обе операции.

### Дождитесь обработки
Система последовательно выполнит восстановление и увеличение.

### Получите результат
Результат откроется во всплывающем окне для скачивания.
</Steps>

## Когда использовать

- Портретные фотографии для печати
- Изображения с людьми в высоком разрешении
- Когда нужно и восстановить лица, и увеличить размер
- Для экономии времени при обработке портретов

## Порядок обработки

<Steps>
### Восстановление лиц
Сначала восстанавливаются лица на изображении.

### Увеличение разрешения
Затем изображение увеличивается в 2 раза.
</Steps>

Такой порядок обеспечивает лучшее качество, чем обратная последовательность.

## Результат

Вы получаете изображение:
- С восстановленными лицами
- В увеличенном разрешении (2x)
- С сохранением общего качества

## Стоимость

Расходует генерации из вашего баланса.
