# Язык запросов

Текстовый запрос (промпт) — это основной способ описать желаемое изображение для AI. От качества запроса напрямую зависит результат генерации.

## Рекомендуемая структура промпта

Для достижения лучших результатов рекомендуется следовать этой структуре:

<Steps>
### Тип изображения
Фотография, иллюстрация, 3D-рендер, картина.

### Основной объект
Что изображено: человек, животное, пейзаж, предмет.

### Внешний вид
Детали объекта: цвет, размер, материал, текстура.

### Элементы сцены
Дополнительные объекты и детали окружения.

### Локация
Где происходит действие: студия, улица, природа, интерьер.

### Эффекты
Освещение, атмосфера, погода, время суток.

### Стиль
Художественное направление: реализм, аниме, киберпанк.
</Steps>

### Примеры запросов

**Простой запрос:**
```
красивый закат над морем
```

**Детальный запрос по структуре:**
```
фотография, молодая женщина, длинные рыжие волосы, зелёное платье,
букет цветов, городской парк, мягкий вечерний свет,
золотой час, портретная фотография
```

## Синтаксис весов

Веса позволяют усилить или ослабить влияние отдельных слов в промпте.

<Callout type="warning">
**Важно:** Веса работают только в круглых скобках `()`, а не в квадратных `[]`!
</Callout>

### Увеличение веса

| Синтаксис | Эффект | Описание |
|-----------|--------|----------|
| `(слово)` | ×1.1 | Увеличение на 10% |
| `((слово))` | ×1.21 | Увеличение на 21% |
| `(((слово)))` | ×1.33 | Увеличение на 33% |
| `(слово:1.5)` | ×1.5 | Точное значение 1.5 |

### Уменьшение веса

| Синтаксис | Эффект | Описание |
|-----------|--------|----------|
| `[слово]` | ÷1.1 | Уменьшение на 10% |
| `(слово:0.5)` | ×0.5 | Уменьшение в 2 раза |
| `(слово:0.25)` | ×0.25 | Уменьшение в 4 раза |

### Буквальные скобки

Если вам нужны скобки как часть текста, экранируйте их:
```
\\(текст в скобках\\)
```

### Примеры использования весов

```
(красные:1.3) волосы, голубые глаза, ((улыбка))
```

В этом примере:
- «красные» усилено до 1.3
- «улыбка» усилена дважды (×1.21)

<Callout type="info">
**Рекомендация:** Начинайте с небольших корректировок (0.1-0.2). Используйте умеренные веса — слишком высокие значения могут вызвать артефакты.
</Callout>

## Язык запросов

<Callout type="info">
Запросы можно писать на русском или английском языке. Английский язык часто даёт более точные результаты, так как большинство моделей обучены на англоязычных данных.
</Callout>

## Негативный промпт

Негативный промпт указывается в поле «Избегать» и определяет, чего следует избегать в изображении:

```
размытое, низкое качество, деформированные руки, лишние конечности
```

Подробнее о негативном промпте читайте в [соответствующем разделе](/docs/generator/right-panel/negative-prompt).
