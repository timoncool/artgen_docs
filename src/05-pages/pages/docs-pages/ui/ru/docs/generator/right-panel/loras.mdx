# LoRA

![LoRA](/assets/docs-pages/screens/lora-weight-slider.png)

LoRA (Low-Rank Adaptation) — это дополнительные мини-модели, которые добавляют новую информацию к генерации. Каждая LoRA обучена на одном конкретном концепте, что позволяет точно воспроизводить определённые стили, персонажей или эффекты.

## Что такое LoRA

LoRA — это небольшие дообученные модели, которые добавляют к базовой модели новые знания. В ArtGeneration.me доступны следующие категории LoRA:

- **Жесты и позы** — специфические позы, жесты рук, ракурсы
- **Персонажи** — конкретные персонажи из игр, фильмов, аниме
- **Художественные стили** — уникальные стили известных художников и направлений
- **Спецэффекты** — визуальные эффекты, освещение, атмосфера
- **Служебные функции** — вспомогательные LoRA для улучшения качества

LoRA работают поверх основной модели, добавляя к ней дополнительные «знания». Это позволяет получить результаты, которые невозможно достичь только с помощью промпта.

## Как использовать LoRA

<Steps>
### Откройте раздел LoRA
Откройте раздел «LoRA» на правой панели генератора

### Найдите нужную LoRA
Просмотрите доступные LoRA или воспользуйтесь поиском

### Добавьте LoRA
Нажмите на карточку LoRA для добавления

### Настройте вес (Weight)
Настройте силу влияния с помощью ползунка. Значение по умолчанию — **1.0**

### Триггер-слова добавятся автоматически
Триггер-слова активируются автоматически при добавлении LoRA
</Steps>

## Триггер-слова

<Callout type="info">
Триггер-слова активируются автоматически при добавлении LoRA. Вам не нужно вручную добавлять их в промпт — система сделает это за вас.
</Callout>

Триггер-слова — это специальные ключевые слова, которые активируют обученный концепт LoRA. Например, если LoRA обучена на персонаже, триггер-слово будет содержать имя этого персонажа.

## Вес LoRA (Weight)

Параметр веса определяет, насколько сильно LoRA влияет на результат. Значение по умолчанию — **1.0**.

- **0.1-0.3** — минимальное влияние, лёгкий намёк на стиль
- **0.4-0.6** — умеренное влияние, баланс между базовой моделью и LoRA
- **0.7-0.9** — сильное влияние, LoRA доминирует
- **1.0** — стандартное максимальное влияние

<Callout type="info">
На моделях GE (Generation Enhanced) поддерживаются отрицательные значения веса. Это позволяет «убирать» влияние LoRA вместо добавления.
</Callout>

## Включение и выключение LoRA

Вы можете быстро включать и выключать добавленные LoRA без их удаления. Это удобно для сравнения результатов с LoRA и без неё, а также для экспериментов с разными комбинациями.

## Комбинирование нескольких LoRA

![Настройка веса для нескольких LoRA](/assets/docs-pages/guide/init-image-02.png)

Вы можете использовать несколько LoRA одновременно для комбинирования эффектов. Однако учитывайте:

- Общая сумма весов всех LoRA не должна превышать **1.5-2.0**
- Конфликтующие LoRA (например, два разных стиля) могут давать непредсказуемые результаты
- Начните с одной LoRA, затем добавляйте другие постепенно

## Совместимость с моделями

<Callout type="warning">
Не все LoRA совместимы со всеми базовыми моделями:

- LoRA для **SDXL** работают только с SDXL-моделями
- LoRA для **FLUX** работают только с FLUX-моделями

Использование несовместимой LoRA может привести к ошибкам или некачественным результатам.
</Callout>

Информация о совместимости указана в описании каждой LoRA.

## Поиск LoRA

В ArtGeneration.me предусмотрен удобный поиск по LoRA:

- Поиск по названию
- Фильтрация по категориям (жесты/позы, персонажи, стили, спецэффекты, служебные)
- Сортировка по популярности
- Просмотр примеров работ

## Советы по использованию

- Изучите примеры работ с LoRA перед использованием
- Триггер-слова добавляются автоматически — не дублируйте их
- Экспериментируйте с весом для достижения нужного эффекта
- На GE моделях попробуйте отрицательные значения веса
- Используйте переключатель для быстрого сравнения результатов
