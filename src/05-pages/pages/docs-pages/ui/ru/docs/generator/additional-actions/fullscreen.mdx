# Полноэкранный режим

![Полноэкранный режим](/assets/docs-pages/screens/image-actions-toolbar.png)

Полноэкранный режим позволяет просматривать изображение в максимальном размере с возможностями масштабирования и панорамирования.

## Как активировать

<Steps>
### Наведите на изображение
Выберите нужное изображение.

### Нажмите кнопку полноэкранного режима
Или дважды кликните на изображение.

### Готово
Изображение откроется на весь экран.
</Steps>

## Возможности

<Callout type="info">
В полноэкранном режиме доступны:
- Просмотр в максимальном разрешении
- **Масштабирование (zoom)** — приближение и отдаление
- **Панорамирование (pan)** — перемещение по увеличенному изображению
- Навигация между изображениями
</Callout>

## Управление

- **Стрелки** — переход к следующему/предыдущему изображению
- **+/-** или колесо мыши — масштабирование
- **Перетаскивание** — перемещение по увеличенному изображению
- **Escape** или клик вне изображения — выход

## Когда использовать

- Для детального изучения результата
- Для проверки качества перед скачиванием
- Для демонстрации работ
- Для сравнения деталей разных генераций
