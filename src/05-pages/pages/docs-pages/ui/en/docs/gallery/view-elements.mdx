# View Elements

![View Elements](/assets/docs-pages/images/blog/img_005.png)

The image viewing page contains many informative elements.

## Top Panel

- **Work title** - if specified by the author
- **Creation date** - when the image was generated
- **Close button** - return to gallery

## Side Panel

- **Author avatar** - profile photo
- **Author name** - link to profile
- **Subscribe button** - subscribe to author
- **Statistics** - likes, views, comments

## Bottom Panel

- **Action buttons** - like, favorites, download, share
- **"Create version" button** - use as a basis for your own generation
