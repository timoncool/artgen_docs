# Interactive Elements

![Interactive Elements](/assets/docs-pages/images/blog/img_001.png)

The gallery contains many interactive elements for interacting with content.

## Action Buttons

- **Like** - mark an image as liked
- **Favorites** - add to favorites collection
- **Share** - send a link to the image
- **Download** - download the image to your device

## Social Features

- **Subscribe to author** - follow a user's new works
- **Comments** - leave feedback under an image
- **Repost** - share on social networks

## Bulk Operations

The gallery supports bulk operations for managing multiple images at once.

### Selecting Multiple Images

![Bulk Selection Mode](/assets/docs-pages/screens/bulk-delete-selected.png)

To select multiple images, hold the Ctrl key (Cmd on Mac) and click on images you want to select. Selected images will be highlighted with a border.

### Selection Bar

![Bulk Selection Bar](/assets/docs-pages/screens/bulk-selection-bar.png)

When you select multiple images, a selection bar appears showing:
- **Selection count** - number of currently selected images
- **Delete button** - remove selected images from your collection
- **Quick action icons** - delete, zoom, show/hide, expand

### Bulk Actions Menu

![Bulk Actions Menu](/assets/docs-pages/screens/gallery-bulk-select.png)

Click the "Action" dropdown to access bulk operations:
- **Delete** - remove all selected images at once
- Confirm or cancel your selection using the checkmark and X buttons

## Color Filter

![Color Selection Dialog](/assets/docs-pages/screens/gallery-bulk-delete-menu.png)

Use the color filter to find images with specific color palettes:
- Select up to 4 colors to filter by
- Click "Apply" to filter results
- Use transparency option for images with transparent backgrounds

## Context Menu

Right-clicking on an image opens additional options:

- Open in new tab
- Copy link
- Report content
