# Gallery View

![Gallery View](/assets/docs-pages/screens/gallery-grid.png)

The ArtGeneration.me gallery showcases user-created images in an interactive grid optimized for convenient viewing on any device.

## Image Count Slider

At the top of the gallery, there is a slider for adjusting the number of displayed images. This allows you to customize the grid density to your preferences:
- More images — compact overview of many works
- Fewer images — larger previews for detailed examination

## Interactive Cards

![Gallery Cards](/assets/docs-pages/screens/gallery-thumbnails-with-dates.png)

Each image is presented as an interactive card. When hovering over a thumbnail, the following is displayed:

- **Likes count** — the number of users who rated the work
- **Favorites icon** — quick addition to your favorites collection
- **Brief prompt with copy button** — prompt text and ability to copy it with one click
- **Creator username with link** — navigate to the creator's profile

## Card Actions

<Steps>
### Copy prompt
Click the copy button to save the prompt to your clipboard and use it for your own generation.

### Open fullsize
Click on the image to view it in full-size mode with all generation parameters.

### Navigate to creator profile
Click on the username below the image to visit the creator's profile page.
</Steps>

## Categories and Filters

![Category and Model Filters](/assets/docs-pages/screens/image-metadata-panel.png)

The gallery supports filtering by various categories:
- **Anime** — works in Japanese animation style
- **Characters** — portraits and character images
- **Classical styles** — works in traditional artistic styles
- **Fantasy** — fantastical worlds and creatures
- **Landscapes** — natural and urban views

## Community Voting

<Callout type="info">
  User likes determine gallery covers. Works with the most votes become the main images for sections.
</Callout>

The voting system allows the community to highlight the best works:
- Like images you enjoy
- Works with more likes receive priority in display
- The most popular works become category covers

## Responsiveness

The gallery automatically adapts to the screen size:

- **Desktop (1920px+)** — 5-6 columns
- **Laptop (1200-1920px)** — 4-5 columns
- **Tablet (768-1200px)** — 3-4 columns
- **Mobile (up to 768px)** — 2 columns

## Gallery Navigation

![Gallery page with image grid](/assets/docs-pages/screens/gallery-pagination.png)

**Infinite scroll** — new images are automatically loaded when scrolling down.

**Quick return to top** — button in the bottom right corner to return to the beginning of the gallery.
