# Registration & Login Process

![Authorization](/assets/docs-pages/blog/update-8.3-1.png)

Logging into ArtGeneration.me is a quick and simple process that takes less than a minute. The platform offers several authorization methods to choose from.

## Email Login (OTP)

The primary login method uses a one-time password (OTP). You don't need to remember a password — a new code is sent to your email each time you log in.

<Steps>
### Enter your email address

Type your email in the login field. Any email provider works — Yandex, Mail.ru, Gmail, Outlook, and others.

### Click "Get Code"

After passing the Cloudflare verification, a one-time code will be sent to your email.

### Receive the code

![Email with one-time code](/assets/docs-pages/blog/update-8.3-2.png)

You'll receive an email with a 6-digit code. The code is valid for **10 minutes**.

### Enter the code

![Entering the one-time code](/assets/docs-pages/blog/update-8.3-3.png)

Type the received code into the input field. After successful verification, you'll be logged in automatically.
</Steps>

<Callout type="info">
  If this is your first login — your account will be created automatically. No separate registration required.
</Callout>

<Callout type="warning">
  **Not receiving the code?**
  - Check your "Spam" folder
  - Make sure the email was entered correctly
  - Wait a couple of minutes — delivery sometimes takes time
  - Click "Resend" (available after 2 minutes)
</Callout>

## Quick Login via OAuth

In addition to email login, quick authorization via social networks and services is available:

### Yandex ID

1. Click the "Sign in with Yandex ID" button
2. Confirm access in the Yandex window
3. Done — you're logged in

### VK ID

1. Click the "Sign in with VK ID" button
2. Confirm access in the VKontakte window
3. Account is created and ready to use

### Odnoklassniki and Mail.ru

Authorization via **Odnoklassniki** and **Mail.ru** is also available — click the corresponding icon on the login page.

<Callout type="info">
  **OAuth advantages:**
  - One-click login without entering a code
  - Automatic profile filling (name, avatar)
  - You can link multiple login methods to one account
</Callout>

## Account Linking

You can link multiple login methods to your account simultaneously. For example, log in via email, then link Yandex or VK in your profile settings for quick future access. You can also disable login methods you don't need.

## Bot Protection

If the system suspects automated registration, you may be asked to complete a Cloudflare verification. This is a standard security measure to protect the platform.

## After Login

After successful login, you will automatically:
- Receive a welcome bonus on first login (3 days of PRO + 100 non-expiring generations)
- Be redirected to the generator page
- Be able to start creating images right away

## Possible Problems

<Callout type="warning">
  **Error during social network authorization:**
  - Check if pop-ups are blocked in your browser
  - Disable your ad blocker — it may block authorization widgets
  - Try a different browser
  - Contact support if the problem persists
</Callout>
