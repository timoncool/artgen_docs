# Logo Generator

![Logo Templates](/assets/docs-pages/blog/update-8.2-1.png)

The Logo Generator allows you to quickly and easily create a logo using **img2img** magic. The model looks at your template and transforms it according to the prompt, giving you a ready-made logo.

The generator uses models that generate Russian text — they are marked with the **RU TEXT** category in the model catalog.

## How It Works

1. Enter your Cyrillic brand name
2. Choose one of the ready-made templates
3. Edit the template if needed

![Logo Template Editor](/assets/docs-pages/blog/update-8.2-2.png)

4. Select a model from the **RU TEXT** category — these models support Russian text generation
5. Write a stylization prompt and start generation

![Logo Stylization](/assets/docs-pages/blog/update-8.2-3.png)

![Finished Logo](/assets/docs-pages/blog/update-8.2-4.png)
