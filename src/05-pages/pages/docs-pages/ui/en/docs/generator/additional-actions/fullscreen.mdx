# Fullscreen Mode

![Fullscreen Mode](/assets/docs-pages/screens/single-image-view.png)

Fullscreen mode allows you to view an image at maximum size with zoom and pan capabilities.

## How to Activate

<Steps>
### Hover over the image
Select the desired image.

### Click the fullscreen button
Or double-click the image.

### Done
The image will open in fullscreen.
</Steps>

## Features

<Callout type="info">
In fullscreen mode you can access:
- Maximum resolution viewing
- **Zoom** — zoom in and out
- **Pan** — move across the zoomed image
- Navigate between images
</Callout>

## Controls

- **Arrows** — go to next/previous image
- **+/-** or mouse wheel — zoom
- **Drag** — pan across zoomed image
- **Escape** or click outside image — exit

## When to Use

- For detailed result examination
- For quality check before downloading
- For showcasing works
- For comparing details of different generations
