# Left Panel

The generator's left panel displays all your created images and provides tools for working with them.

**Chronological Order** â€” all your created images are displayed in chronological order.

**Likes and Favorites Counters** â€” each image displays the number of likes and favorites.

## Reset Button (green)

At the top of the left panel, there is a **green reset button**. It allows you to quickly reset all current generator settings to defaults:

- Clears the prompt field
- Resets the selected model to default
- Disables all LoRAs
- Resets parameters (CFG, Steps, Seed, etc.)
- Removes the init image
