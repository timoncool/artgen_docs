# Image Generator

![ArtGeneration.me Image Generator](/assets/docs-pages/screens/generator-right-panel.png)

The Image Generator is the main tool of ArtGeneration.me, allowing you to create unique images using artificial intelligence.

Here you can bring any creative ideas to visual form.

## Generator Interface

The interface is divided into three main areas:

1. **Central area** — generation results and prompt input field
2. **Left panel** — generation history and quick actions
3. **Right panel** — model settings, styles, and parameters

![Right panel](/assets/docs-pages/screens/generator-right-panel2.png)

## Prompt Input Field

![Prompt input field](/assets/docs-pages/screens/prompt-input.png)

The bottom panel of the generator contains:
- **Prompt input field** — describe the desired image
- **"Reset" button** — clear current settings
- **"Create" button** — start generation

### Helper Buttons

Next to the input field, there are additional buttons:

- **Dice (🎲)** — generates a random prompt. Useful for inspiration or when you don't know where to start.
- **Eraser (🧹)** — clears the prompt input field with one click.
- **Auto-translator** — automatically translates your prompt to English.

<Callout emoji="💡">
**Auto-translator** is especially useful if you write in Russian. Generation models understand English prompts better, so automatic translation can improve the quality of results.
</Callout>

<Callout type="info">
**Tips:**
- Use **Ctrl + Enter** to quickly start generation
- It is recommended to write prompts in **English** for best results
- Prompts in Russian are also supported — use the auto-translator to convert them
</Callout>

## Viewing Results

![Single image view](/assets/docs-pages/screens/single-image-view.png)

After generation, the image is displayed in the central area. Available options:
- "Crop" button for cropping
- Prompt field for creating variations
- "Create" button for new generation

## Rating Images

![Image rating](/assets/docs-pages/screens/image-rate-popup.png)

In the gallery, you can rate images from other users:
- **Like** — mark an image you liked
- **Dislike** — if you didn't like the image

## Image Visibility

![Visibility toggle](/assets/docs-pages/screens/visibility-toggle.png)

For each image, you can configure visibility:
- **Visible to all** — the image is displayed in the public gallery
- **Private** — visible only to you

## Query Language

The generator supports queries in **Russian** and **English**.

### Query Examples

**In Russian:**
- "Красивый закат над морем в стиле импрессионизма"
- "Портрет девушки с цветами в волосах"

**In English:**
- "Beautiful sunset over the ocean in impressionist style"
- "Portrait of a girl with flowers in her hair"

## Query Tips

<Steps>
### Be Specific
The more detailed the description, the more accurate the result.

### Specify the Style
Add an artistic style or technique.

### Describe the Lighting
Use terms like: "soft light", "backlight", "golden hour".

### Add Details
Describe textures, materials, atmosphere.

### Use Negative Prompts
Exclude unwanted elements.
</Steps>

## Additional Actions

After image generation, the following actions are available:

- **Download** — save the image to your device
- **Enhance** — increase resolution using AI
- **Edit** — open in editor for refinement
- **Face Swap** — replace a face in the image
- **Create Variations** — generate similar images
- **Share** — publish to the gallery

## Logo Generator

![Logo Generator](/assets/docs-pages/blog/update-8.2-5.png)

The Logo Generator allows you to quickly and easily create a logo using **img2img** magic. The model looks at your template and transforms it according to the prompt, giving you a ready-made logo.

### How It Works

1. Enter your Cyrillic brand name
2. Choose one of the ready-made templates
3. Edit the template if needed
4. Select a model from the **RU TEXT** category — these models support Russian text generation
5. Write a stylization prompt and start generation

<Callout emoji="💡">
Experiment with different **RU TEXT** category models and prompts — each model gives its own unique logo style.
</Callout>
