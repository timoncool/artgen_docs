# Image Generator

![ArtGeneration.me Image Generator](/assets/docs-pages/screens/generator-right-panel.png)

The Image Generator is the main tool of ArtGeneration.me, allowing you to create unique images using artificial intelligence.

Here you can bring any creative ideas to visual form.

## Generator Interface

The interface is divided into three main areas:

1. **Central area** ‚Äî generation results and prompt input field
2. **Left panel** ‚Äî generation history and quick actions
3. **Right panel** ‚Äî model settings, styles, and parameters

![Right panel](/assets/docs-pages/screens/generator-right-panel2.png)

## Prompt Input Field

![Prompt input field](/assets/docs-pages/screens/prompt-input.png)

The bottom panel of the generator contains:
- **Prompt input field** ‚Äî describe the desired image
- **"Reset" button** ‚Äî clear current settings
- **"Create" button** ‚Äî start generation

### Helper Buttons

Next to the input field, there are additional buttons:

- **Dice (üé≤)** ‚Äî generates a random prompt. Useful for inspiration or when you don't know where to start.
- **Eraser (üßπ)** ‚Äî clears the prompt input field with one click.
- **Auto-translator** ‚Äî automatically translates your prompt to English.

<Callout emoji="üí°">
**Auto-translator** is especially useful if you write in Russian. Generation models understand English prompts better, so automatic translation can improve the quality of results.
</Callout>

<Callout type="info">
**Tips:**
- Use **Ctrl + Enter** to quickly start generation
- It is recommended to write prompts in **English** for best results
- Prompts in Russian are also supported ‚Äî use the auto-translator to convert them
</Callout>

## Viewing Results

![Single image view](/assets/docs-pages/screens/single-image-view.png)

After generation, the image is displayed in the central area. Available options:
- "Crop" button for cropping
- Prompt field for creating variations
- "Create" button for new generation

## Rating Images

![Image rating](/assets/docs-pages/screens/image-rate-popup.png)

In the gallery, you can rate images from other users:
- **Like** ‚Äî mark an image you liked
- **Dislike** ‚Äî if you didn't like the image

## Image Visibility

![Visibility toggle](/assets/docs-pages/screens/visibility-toggle.png)

For each image, you can configure visibility:
- **Visible to all** ‚Äî the image is displayed in the public gallery
- **Private** ‚Äî visible only to you

## Query Tips

<Steps>
### Be Specific
The more detailed the description, the more accurate the result.

### Specify the Style
Add an artistic style or technique.

### Describe the Lighting
Use terms like: "soft light", "backlight", "golden hour".

### Add Details
Describe textures, materials, atmosphere.

### Use Negative Prompts
Exclude unwanted elements.
</Steps>

## Additional Actions

After image generation, the following actions are available:

- **Download** ‚Äî save the image to your device
- **Enhance** ‚Äî increase resolution using AI
- **Edit** ‚Äî open in editor for refinement
- **Face Swap** ‚Äî replace a face in the image
- **Create Variations** ‚Äî generate similar images
- **Share** ‚Äî publish to the gallery
