# Copy Link

![Copy Link](/assets/docs-pages/screens/image-actions-toolbar.png)

The copy link function allows you to get a direct URL to the image for quick sharing.

## How to Use

<Steps>
### Hover over the image
Find the desired image in the generation results.

### Click the "Copy Link" button
The button will appear on the action toolbar.

### Done
The link is copied to clipboard.
</Steps>

## Link Features

<Callout type="info">
The copied link supports **automatic preview** in social networks and messengers. When pasting the link, the recipient will immediately see an image preview.
</Callout>

## Applications

- Share the image on social media
- Send the link via messenger with auto-preview
- Insert into a document or presentation
- Save for later access

## Link Format

The link leads to the image viewing page on ArtGeneration.me, where you can access:
- Full-size image
- Generation parameters information
- Option to create your own version

## Privacy

<Callout type="warning">
If the image was created in private mode, the link will only be accessible to you.
</Callout>
