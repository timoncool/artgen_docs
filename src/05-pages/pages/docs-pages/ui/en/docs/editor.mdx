# Image Editor

The editor allows you to modify generated images without the need for third-party programs. It's a powerful tool for finalizing your generations.

![ArtGeneration.me Image Editor](/assets/docs-pages/guide/editor-02.png)

## How to Open the Editor

There are several ways to open an image in the editor:

<Steps>
### From the generator gallery
Hover over an image and click the "To Editor" button.

### From the main gallery
Open an image and select "Edit".

### Directly
Go to the "Editor" section in the navigation and upload an image.
</Steps>

## Main Tools

### Crop

Crop the image to the desired size or aspect ratio. Available presets:
- Free crop
- 1:1 (square)
- 16:9 (widescreen)
- 9:16 (vertical)
- 4:3 (classic)

![Image crop tool](/assets/docs-pages/screens/crop-image-tool.png)

### Scale

Resize the image while maintaining proportions. Useful for preparing images for social media publication.

### Rotate

Rotate the image to any angle. Quick rotations at 90Â° and precise angle adjustment are available.

### Flip

Flip the image horizontally or vertically. Often used to fix composition.

## Drawing and Masks

![Annotation tools in editor](/assets/docs-pages/images/blog/img_022.png)

### Brush

Draw over the image with various brushes:
- Brush size adjustment
- Edge hardness adjustment
- Color selection

![Brush color picker](/assets/docs-pages/screens/color-picker.png)

### Mask for Inpainting

Create masks for selective editing:

<Steps>
### Select the tool
Select the "Mask" tool from the toolbar.

### Paint the area
Paint over the area you want to change.

### Send to generator
Click "Send to Generator".

### Write a prompt
Write a prompt for the new area.

### Start generation
Start generation and wait for the result.
</Steps>

![Mask tool](/assets/docs-pages/screens/mask-paint-tool.png)

![Mask settings panel](/assets/docs-pages/screens/mask-options-panel.png)

<Callout type="warning">
For quality inpainting, it's recommended to paint the area with a small margin beyond the object you want to change.
</Callout>

## Integration with Generator

The edited image can be used as an Init Image for further generation. This allows you to create complex compositions iteratively:

<Steps>
### Generate a base image
Create an initial image in the generator.

### Edit in the editor
Open the image in the editor and make the necessary changes.

### Send back to the generator
Send the edited image back to the generator.

### Add new elements
Add new elements through inpainting.

### Repeat the process
Repeat the process until you achieve the desired result.
</Steps>
