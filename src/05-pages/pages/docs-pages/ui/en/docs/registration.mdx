# Login & Registration

Welcome to the amazing world of digital creativity at [ArtGeneration.me](https://artgeneration.me), where anyone can become an artist with the help of artificial intelligence. To begin your journey, you need to log in or create an account.

On the main page, you'll find the "Start Creating" button — your first step towards creating unique images.

## Email Login (OTP)

![Login Form](/assets/docs-pages/blog/update-8.3-1.png)

The primary login method uses a one-time password (OTP). You don't need to remember a password — a new code is sent to your email each time.

import { Steps, Callout } from '@/src/01-shared/ui/mdx'

<Steps>
### Enter your email address

Type your email in the login field. Any email provider works — Yandex, Mail.ru, Gmail, Outlook, and others.

### Click "Get Code"

After passing the Cloudflare verification, a one-time code will be sent to your email.

### Receive the code

![Email with one-time code](/assets/docs-pages/blog/update-8.3-2.png)

You'll receive an email with a 6-digit code. The code is valid for **10 minutes**.

### Enter the code

![Entering the one-time code](/assets/docs-pages/blog/update-8.3-3.png)

Type the received code into the input field. After successful verification, you'll be logged in automatically.
</Steps>

<Callout type="info">
  If this is your first login — your account will be created automatically. No separate registration required.
</Callout>

## Quick Login via OAuth

In addition to email login, quick authorization via social networks is available:

- **Yandex ID** — click "Sign in with Yandex ID"
- **VK ID** — click "Sign in with VK ID"
- **Odnoklassniki** — click the OK icon
- **Mail.ru** — click the @ icon

<Callout type="info">
  **OAuth advantages:** one-click login, automatic profile filling, you can link multiple login methods to one account.
</Callout>

## Why Register?

Creating an account gives you the opportunity to:

- Save your work
- Receive updates
- Participate in the community
- Enjoy a personalized experience on the platform
- Subscribe to other users and follow their updates

If you need help — our support team support@artgeneration.me is always at your service.
