# Technical FAQ

Technical questions and troubleshooting.

## Which browsers are supported?

Recommended browsers:
- Google Chrome (latest)
- Mozilla Firefox (latest)
- Microsoft Edge (latest)
- Safari (latest)

## Why is the site loading slowly?

Possible causes:
- Slow internet connection
- Browser cache issues (try clearing cache)
- Server maintenance

Try refreshing the page or using a different browser.

## Can I use the site on mobile?

Yes! The site is fully responsive and works on:
- Smartphones
- Tablets
- Desktop computers

## Why can't I see my images?

Check:
- You're logged into the correct account
- Images aren't filtered by search/category
- Try refreshing the page

## How do I download images?

1. Open the image
2. Click the download button
3. Choose format (WebP, PNG, or JPG)

## What image formats are supported?

- **WebP** — Default format, best quality/size ratio
- **PNG** — Lossless, supports transparency
- **JPG** — Smaller files, good for photos

## Is my data secure?

Yes. We use:
- Encrypted connections (HTTPS)
- Secure payment processing
- No storage of payment card details

## How do I report a bug?

Contact support@artgeneration.me with:
- Description of the issue
- Steps to reproduce
- Browser and device info
- Screenshots if possible
