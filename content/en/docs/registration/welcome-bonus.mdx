# Welcome Bonus

![Welcome Bonus](/images/blog/img_027.png)

New users receive a welcome bonus upon registration.

## What's Included

### Free Generations
A certain number of generations to get started with the service.

### Access to Features
Ability to try all basic platform features.

### No Time Limit
Bonus generations don't expire.

## How to Get

1. Register on the website
2. Confirm your email
3. Bonus is credited automatically

## Using the Bonus

Use bonus generations to:
- Try different models
- Experiment with styles
- Learn the interface
