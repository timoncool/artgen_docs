# Welcome & Registration

Welcome to ArtGeneration! This section will help you get started with the platform.

## What is ArtGeneration?

ArtGeneration is an online service for creating images using artificial intelligence. You describe the desired image in text, and the neural network creates it for you.

## Getting Started

1. Register on the website
2. Receive welcome bonus generations
3. Start creating!
