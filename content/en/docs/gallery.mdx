# Gallery

The Gallery is a section where you can view works by other users and find inspiration for your generations.

## Gallery Types

### Public Gallery
Shows all public works by platform users. Here you can:
- View new works
- Filter by categories
- Search by parameters

### Personal Gallery
Your generated images are saved in your personal gallery. Available after authorization.

### Category Gallery
Works are grouped by thematic categories for convenient searching.

## Gallery Features

Click on an image to see the generation parameters and use them to create similar works.

### View Parameters
When you click on an image, you will see:
- The prompt used
- Generation model
- Style
- Seed
- Other parameters

### Copy Settings
You can copy the settings of an image you like and use them in the generator.

## Filtering and Search

### By Model
Filter works by the generation model used.

### By Color
Search for images by dominant color.

### By Seed
Find image variations with the same seed.

## Organizing Works

### Folders
Create folders to organize your images by projects or themes.

### Batch Operations
Select multiple images for mass moving or deletion.
