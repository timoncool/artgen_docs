# Batch Operations

Batch operations allow you to work with multiple images at once.

## Selecting Images

- **Click + Ctrl** — add to selection
- **Click + Shift** — select range
- **Ctrl + A** — select all

## Available Operations

### Download
Download selected images as archive.

### Delete
Delete all selected images.

### Move
Move to selected folder.

### Add Tags
Add tags to all selected images.

## Canceling Selection

Press Esc or click on empty area to cancel selection.
