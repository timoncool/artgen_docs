# Filters and Search

Use filters and search to quickly find the images you need.

## Text Search

Enter text to search by:
- Prompt
- Image name
- Tags

## Filters

### By Date
Select a date range for generations.

### By Model
Filter by the model used.

### By Style
Filter by applied styles.

### By Resolution
Filter by image size.

## Sorting

Sort images by:
- Date (newest/oldest)
- Popularity
- Name

## Saving Filters

Save frequently used filter combinations for quick access.
