# Main Gallery

![Main Gallery](/images/blog/img_028.png)

The gallery is the heart of the ArtGeneration.me community, where the best user works are collected. Here you can find inspiration, study techniques of other artists, and discover new styles.

## Search and Categories

![Search and Categories](/images/blog/img_030.png)

At the top of the gallery is a search bar and category filters. You can search for images by keywords or select a category of interest from the list.

**Available categories:**
- All
- Portraits
- Landscapes
- Fantasy
- Sci-Fi
- Anime
- Abstract
- And many more

## Cover Interaction

When hovering over an image in the gallery, interactive elements appear:
- **Heart** - add to favorites
- **Eye** - quick view
- **Info** - generation details

## Gallery View

![Gallery View](/images/blog/img_032.png)

The gallery is displayed as a grid of images. You can scroll down to load new works. Images are automatically loaded as you scroll.

### Sorting

The following sorting options are available:
- **New** - the freshest works
- **Popular** - works with the most likes
- **Discussed** - works with active comments

## Image View

![Image View](/images/blog/img_008.png)

Clicking on an image opens full-screen view with detailed generation information.

### View Elements

In viewing mode, the following are available:
- **Full-size image** - high quality
- **Author information** - creator's profile
- **Generation parameters** - prompt, model, settings
- **Actions** - download, share, create version

### Generation Details

![Generation Details](/images/blog/img_008.png)

The right panel displays all parameters used to create the image:
- Text prompt
- Negative prompt
- Model
- Image size
- Seed
- Number of steps
- CFG Scale

### Similar Images

Below the main image, similar works that may interest you are displayed. The algorithm selects images based on style, theme, and parameters used.

### Create Your Version

![Create Your Version](/images/blog/img_013.png)

Click the "Create your version" button to use the parameters of a liked image as a basis for your own generation. All settings will be automatically transferred to the generator.
