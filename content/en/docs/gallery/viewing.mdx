# Viewing Images

Detailed information about viewing images in the gallery.

## Opening an Image

Click on an image to open it in full-screen mode.

## Full-Screen Mode

In full-screen mode you can:
- View the image in full resolution
- See generation parameters
- Navigate between images
- Perform actions with the image

## Image Information

For each image you can see:
- Prompt used
- Model and style
- Resolution
- Generation date
- Seed and other parameters

## Navigation

- **Arrows** — previous/next image
- **Esc** — close full-screen mode
- **Scroll** — zoom
