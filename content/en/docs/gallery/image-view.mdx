# Image View

![Image View](/images/blog/img_029.png)

When clicking on an image in the gallery, full-screen viewing mode opens. Here you can study the work in detail, learn the generation parameters, and interact with the image.

## Viewing Interface

The image viewing page consists of several main areas:

**Image area** - the central part of the screen with the high-resolution image. The image is scaled for optimal display on your screen.

**Information panel** - on the right or bottom (depending on device) is a panel with detailed information about the work.

**Action panel** - buttons for interacting with the image: like, favorites, download, create version.

## Image Information

The viewing page displays:

**Author** - avatar, username, and link to the creator's profile. You can go to the profile to see other works by this author.

**Creation date** - when the image was created.

**Statistics** - number of views, likes, and favorites additions.

**Prompt** - the full text prompt used for generation. You can copy the prompt with one click.

**Generation parameters** - model, style, resolution, seed, and other technical parameters.

## Image Actions

**Like** - mark the image as liked. This helps the author and affects the work's rating.

**Favorites** - add the image to your collection for quick access in the future.

**Download** - download the image in full resolution to your device.

**Create version** - go to the generator with this image's parameters to create your own variation.

**Share** - copy the image link to send to friends.

## Navigation Between Images

**On-screen arrows** - buttons to the left and right of the image for moving to the previous or next image in the gallery.

**Keyboard** - use left/right arrows on the keyboard for quick navigation.

**Swipe** - on mobile devices, swipe left or right to move between images.

## Closing View

You can return to the gallery in several ways:

- Click the **X** button in the corner of the screen
- Press the **Escape** key on the keyboard
- Click on the darkened area outside the image

## Similar Images

Below the main image is a section with similar works. This helps find inspiration and explore various interpretations of similar themes.

## Zooming

For detailed image study:
- Click on the image to enlarge
- Use the mouse wheel for zooming
- On mobile devices, use pinch gesture to zoom
