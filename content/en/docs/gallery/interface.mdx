# Gallery Interface

The gallery interface is designed for convenient viewing and managing images.

## Main Elements

### Image Grid
Images are displayed in a responsive grid that adapts to screen size.

### Sidebar
Contains filters, folders, and navigation.

### Top Panel
Search, sorting, and view mode selection.

## View Modes

- **Grid** — compact display of many images
- **List** — detailed view with metadata
- **Large** — enlarged thumbnails

## Navigation

Use keyboard arrows or mouse scroll for navigation.
