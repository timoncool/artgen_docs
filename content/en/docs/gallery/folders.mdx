# Folders

Folders help organize images by projects or themes.

## Creating a Folder

1. Click "Create Folder"
2. Enter folder name
3. Click "Create"

## Managing Folders

### Rename
Right-click on folder → "Rename"

### Delete
Right-click on folder → "Delete"

### Move
Drag and drop to change hierarchy

## Adding Images to Folder

- Drag image to folder
- Select images → "Move to Folder"
- In image menu → "Add to Folder"

## Nested Folders

Create subfolders for more detailed organization.
