# Moderation

Information about content moderation in the gallery.

## Automatic Moderation

All images go through automatic check for:
- Prohibited content
- Rule violations
- Quality

## Manual Moderation

Some images may be sent for manual review.

## Moderation Results

- **Approved** — image is available
- **Rejected** — image is hidden
- **Requires Changes** — need to edit

## Appeal

If you disagree with the moderation decision, you can file an appeal through support.
