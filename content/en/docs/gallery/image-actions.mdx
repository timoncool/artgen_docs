# Image Actions

Available actions for images in the gallery.

## Basic Actions

- **Download** — save image to device
- **Delete** — remove from gallery
- **Edit** — open in editor
- **Share** — get link for sharing

## Generation Actions

- **Create Similar** — generate with same parameters
- **Upscale** — increase resolution
- **Face Swap** — replace face

## Organization

- **Add to Favorites** — mark as favorite
- **Move to Folder** — organize into folders
- **Add Tags** — add tags for search

## Context Menu

Right-click on an image to open the context menu with all available actions.
