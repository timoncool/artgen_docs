# Interactive Elements

![Interactive Elements](/images/blog/img_001.png)

The gallery contains many interactive elements for interacting with content.

## Action Buttons

- **Like** - mark an image as liked
- **Favorites** - add to favorites collection
- **Share** - send a link to the image
- **Download** - download the image to your device

## Social Features

- **Subscribe to author** - follow a user's new works
- **Comments** - leave feedback under an image
- **Repost** - share on social networks

## Context Menu

Right-clicking on an image opens additional options:

- Open in new tab
- Copy link
- Report content
