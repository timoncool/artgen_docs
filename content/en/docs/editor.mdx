# Image Editor

The editor allows you to modify generated images without the need for third-party programs.

## Main Tools

### Crop
Crop the image to the desired size or aspect ratio.

### Scale
Resize the image while maintaining proportions.

### Rotate
Rotate the image to any angle.

### Flip
Flip the image horizontally or vertically.

### Available Filters
- Brightness and contrast
- Saturation
- Blur
- Sharpness
- Vignette

## Drawing

### Brush
Draw over the image with various brushes.

### Mask
Create masks for selective editing or inpainting.

## Saving

After editing, you can:
- Save as a new image
- Replace the original
- Download to your device

## Integration with Generator

The edited image can be used as an Init Image for further generation, allowing you to create complex compositions iteratively.
