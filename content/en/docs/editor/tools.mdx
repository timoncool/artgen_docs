# Tools

![Editor Tools](/images/blog/img_024.png)

Available tools in the image editor.

## Drawing Tools

### Brush
Freehand drawing with customizable size and opacity.

### Pencil
Precise pixel-level drawing.

### Eraser
Remove parts of the image or layer.

### Fill
Fill areas with solid color.

## Selection Tools

### Rectangle Select
Select rectangular areas.

### Lasso
Freehand selection.

### Magic Wand
Select areas by color similarity.

## AI Tools

### Inpainting
Edit selected areas using AI.

### Background Removal
Automatically remove background.

## Transform Tools

### Move
Reposition elements.

### Scale
Resize elements.

### Rotate
Rotate elements.
