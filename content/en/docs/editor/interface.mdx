# Editor Interface

Overview of the image editor interface.

## Main Areas

### Toolbar
Located at the top, contains all drawing and editing tools.

### Canvas
Central area where you work with the image.

### Layers Panel
Right side panel for layer management.

### Properties Panel
Shows properties of selected tool or object.

## Keyboard Shortcuts

- **Ctrl+Z** — Undo
- **Ctrl+Y** — Redo
- **Ctrl+S** — Save
- **Space+Drag** — Pan canvas
- **Scroll** — Zoom

## Navigation

- Zoom in/out with scroll wheel
- Pan by holding Space and dragging
- Reset view with Home key
