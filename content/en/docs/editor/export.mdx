# Exporting Results

Save your edited images in various formats.

## Export Formats

### PNG
- Lossless compression
- Supports transparency
- Best for graphics and illustrations

### JPG
- Lossy compression
- Smaller file size
- Best for photographs

### WEBP
- Modern format
- Good compression with quality
- Supports transparency

## Export Settings

### Quality
Adjust compression level (for JPG/WEBP).

### Resolution
Choose output dimensions.

### Background
Include or exclude background.

## Quick Export

- **Ctrl+Shift+E** — Export with last settings
- **Ctrl+E** — Export dialog

## Saving to Gallery

Exported images are automatically saved to your gallery.
