# Working with Layers

Layers allow non-destructive editing by separating elements.

## Layer Basics

Each layer is independent and can be edited without affecting others.

## Layer Operations

### Create Layer
Click "+" in the layers panel.

### Delete Layer
Select layer and click trash icon.

### Duplicate Layer
Right-click → Duplicate.

### Merge Layers
Select multiple layers → Right-click → Merge.

## Layer Properties

- **Opacity** — Transparency level
- **Blend Mode** — How layer interacts with layers below
- **Visibility** — Show/hide layer
- **Lock** — Prevent accidental edits

## Layer Order

Drag layers to reorder. Upper layers appear on top.

## Tips

- Use separate layers for different elements
- Name layers for organization
- Lock finished layers to prevent accidents
