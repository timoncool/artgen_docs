The ArtGeneration.me image editor is a powerful tool for refining and enhancing generated images. It combines classic editing tools with artificial intelligence capabilities.

![Image Editor Interface](/guide/editor-02.png)

## AI Tools

**Inpainting (redrawing)** â€” select an area on the image and describe what should appear in its place. AI will redraw the selected part while preserving the style and lighting of the rest of the image. Ideal for fixing defects, replacing elements, or adding new details.

![Mask tool for inpainting](/screens/mask-paint-tool.png)

**Outpainting (expansion)** â€” extend the image boundaries, and AI will fill in the missing parts. Useful when you need to change the aspect ratio or add space around an object.

![Outpainting menu](/screens/outpainting-menu.png)

**Object Removal** â€” select an unwanted element, and AI will automatically remove it, filling the area with appropriate background.

## How to Open the Editor

### From the Generator

<Steps>

Hover over the result and click the "Edit" button.

### Work in the editor
The image will open in the editor.

### From History or Gallery

<Steps>

### Directly

<Steps>

Upload your image or select from history.
</Steps>

## Workflow

Typical workflow in the editor:

<Steps>
### Upload an image
Upload or open an image for editing.

### Select an area
Use selection tools to mark the editing area.

### Choose an AI tool
Select the needed AI tool (inpainting, outpainting, removal).

### Configure parameters
Enter a prompt (for inpainting) or adjust parameters.

### Start processing
Start processing and wait for the result.

### Repeat if necessary
Repeat for other areas if needed.

### Export the result
Export the finished result.
</Steps>
