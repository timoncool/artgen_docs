## How to Open the Editor

### From the Generator

<Steps>
### Generate an image
Create an image using the generator.

### Click the edit button
Hover over the result and click the "Edit" button.

### Work in the editor
The image will open in the editor.
</Steps>

### From History or Gallery

<Steps>
### Open the image
Open the image in full-size mode.

### Go to the editor
Click the "Edit" button on the action panel.
</Steps>

### Directly

<Steps>
### Go to the editor page
Open the editor page: artgeneration.me/editor

### Upload an image
Upload your image or select from history.
</Steps>

## Workflow

Typical workflow in the editor:

<Steps>
### Upload an image
Upload or open an image for editing.

### Select an area
Use selection tools to mark the editing area.

### Choose an AI tool
Select the needed AI tool (inpainting, outpainting, removal).

### Configure parameters
Enter a prompt (for inpainting) or adjust parameters.

### Start processing
Start processing and wait for the result.

### Repeat if necessary
Repeat for other areas if needed.

### Export the result
Export the finished result.
</Steps>
