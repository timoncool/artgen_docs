# Getting Generations

![Getting Generations](/screens/daily-reward-popup.png)

Generations are the main currency on the ArtGeneration.me platform. Each generation allows you to create one image using artificial intelligence.

## Ways to Get Generations

### Free Generations

- **Daily generations** — every day you receive a certain number of free generations
- **Welcome bonus** — new users receive 100 non-expiring generations upon registration

<Callout type="info">
  The welcome bonus of 100 generations is a great way to explore the service's features without any costs!
</Callout>

### Paid Generations

- **PRO subscription** — 300 generations per day
- **Package purchases** — additional generations can be purchased separately

## Types of Generations

1. **Daily** — refresh every day, do not accumulate
2. **Non-expiring** — saved until used, have no expiration date

<Callout type="warning">
  Daily generations do not accumulate! Be sure to claim them every day so you don't lose them.
</Callout>
