# Right Panel

The right panel contains all settings for image generation.

## Panel Sections

- **Models** — base model selection
- **Styles** — artistic styles
- **LoRA** — additional adapters
- **Init Image** — image-to-image generation
- **Face Swap** — face replacement
- **Parameters** — fine-tuning
- **Negative Prompt** — exclusions
- **Resolution** — image size
- **Performance Mode** — speed vs quality
