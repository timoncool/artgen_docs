# Init Image

Init Image allows generating based on an existing image.

## How It Works

Upload an image as a starting point, and the model will transform it according to your prompt.

## Uploading Image

- Drag and drop
- Click to select file
- Paste from clipboard

## Denoising Strength

Controls how much the original image is changed:
- **0.1-0.3** — minimal changes
- **0.4-0.6** — moderate transformation
- **0.7-1.0** — significant changes

## Use Cases

- Style transfer
- Image enhancement
- Concept variations
- Color correction

## Supported Formats

PNG, JPG, WEBP up to 10MB.
