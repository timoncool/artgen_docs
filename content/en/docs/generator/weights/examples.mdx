# Weight Examples

Practical examples of using weights.

## Portrait with Emphasis

```
portrait of a woman, (green eyes:1.4), (red lipstick:1.2), soft lighting
```

## Landscape with Mood

```
mountain landscape, (dramatic sky:1.3), (fog:1.1), (trees:0.8)
```

## Style Emphasis

```
digital art, (cyberpunk style:1.5), neon lights, futuristic city
```

## Balancing Elements

```
(cat:1.2) and (dog:1.2) playing together, garden background
```

## Reducing Background

```
portrait, detailed face, (background:0.6), studio lighting
```
