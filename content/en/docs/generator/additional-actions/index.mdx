# Additional Actions

Additional actions for working with generated images.

## Available Actions

- **Upscale** — increase resolution
- **Variations** — create similar images
- **Image to Image** — transform existing images
- **Inpainting** — edit parts of image
- **Outpainting** — extend image boundaries
