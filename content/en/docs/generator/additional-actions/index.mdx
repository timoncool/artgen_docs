# Additional Actions

After generating an image, a number of additional actions are available for improving and working with the result.

## Available Actions

When hovering over a generated image, an action button panel appears:

- **Own Image** — use as a base for new generation
- **Create Similar** — generate a variation
- **Upscale 2x** — improve resolution
- **Face Restoration** — improve face quality
- **Restoration + Upscale** — restore faces and upscale 2x in one click
- **Outpainting** — add content beyond image boundaries
- **Copy Link** — get a link to the image
- **Download** — save to device
- **Fullscreen** — view in full size
