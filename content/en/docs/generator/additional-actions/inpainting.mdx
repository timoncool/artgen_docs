# Inpainting

Edit specific parts of an image while keeping the rest unchanged.

## How It Works

1. Select the area to edit (mask)
2. Describe what should appear there
3. Generate the edit

## Creating Masks

- Brush tool for painting
- Selection tools for shapes
- Automatic detection for objects

## Use Cases

- Remove unwanted objects
- Add new elements
- Fix imperfections
- Change specific details

## Tips

- Make masks slightly larger than the area
- Use clear descriptions
- Match lighting and style
