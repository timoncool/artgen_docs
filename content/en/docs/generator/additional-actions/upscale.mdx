# Upscale

Upscale increases image resolution while preserving quality.

## How It Works

AI algorithms analyze the image and intelligently increase resolution, adding details.

## Upscale Options

- **2x** — double the resolution
- **4x** — quadruple the resolution

## When to Use

- Preparing images for print
- Creating wallpapers
- Enlarging small generations

## Quality Tips

- Start with best quality generation
- Use appropriate upscale level
- Check result for artifacts

## Cost

Upscaling uses additional generations depending on the scale factor.
