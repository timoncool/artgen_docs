# Variations

Create variations of an existing image while maintaining the overall concept.

## How It Works

The model uses the original image as reference and creates similar but different versions.

## Variation Strength

- **Low** — minimal changes, very similar
- **Medium** — noticeable differences
- **High** — significant variations

## Use Cases

- Exploring different interpretations
- Finding the best version
- Creating series of related images

## Tips

- Start with medium strength
- Generate multiple variations at once
- Save favorites for comparison
