# Create Similar

![Create Similar](/screens/image-actions-toolbar.png)

The "Create Similar" function generates variations of an image with the same parameters but a different seed.

## How It Works

When clicking the "Create Similar" button:

<Steps>
### Copying Parameters
All parameters of the original generation are copied.

### Generating New Seed
A new random seed is created.

### Starting Generation
A new generation is automatically started.
</Steps>

## Result

You will get an image:
- With the same prompt
- With the same model and style
- With the same technical parameters
- But with a different random variation

## When to Use

- When the result is almost perfect, but you want to try other options
- For creating a series of similar images
- For finding the best interpretation of a prompt

## Tips

<Callout type="info">
If you like the general direction but not the specific result, use this function several times to find the perfect variant.
</Callout>
