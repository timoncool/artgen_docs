# Copy Link

![Copy Link](/images/blog/img_013.png)

The copy link function allows you to get a direct URL to the image.

## How to Use

1. Hover over the image
2. Click the "Copy Link" button
3. The link has been copied to the clipboard

## Use Cases

- Share the image on social networks
- Send the link via messenger
- Insert into a document or presentation
- Save for future access

## Link Format

The link leads to an image viewing page on ArtGeneration.me, where you have access to:
- Full-size image
- Information about generation parameters
- Ability to create your own version

## Privacy

If the image was created in private mode, the link will only be accessible to you.
