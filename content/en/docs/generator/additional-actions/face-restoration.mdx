# Face Restoration

![Face Restoration](/screens/image-actions-toolbar.png)

The face restoration function improves the quality of faces in generated images.

## Problem

AI generation sometimes creates faces with:
- Blurred features
- Asymmetry
- Unnatural details
- Artifacts

## Solution

Face restoration uses specialized algorithms to:
- Improve clarity of facial features
- Correct asymmetry
- Add realistic details
- Eliminate artifacts

## How to Use

1. Hover over an image with a face
2. Click the "Face Restoration" button
3. Wait for processing
4. Get the improved result

## When to Apply

- Portrait images
- Group photos
- Any images with people
- When faces look unnatural

## Limitations

- Works better on large faces
- May change some features
- Doesn't always improve stylized images (anime, art)
