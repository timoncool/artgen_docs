# Download

![Download](/images/blog/img_013.png)

The download function allows you to save images to your device.

## How to Download

1. Hover over the image
2. Click the "Download" button
3. The file will be downloaded to your downloads folder

## File Format

Images are downloaded in PNG format with maximum quality.

## File Name

The file name contains:
- Generation identifier
- Creation date
- .png extension

## Usage Rights

Downloaded images can be used for:
- Personal projects
- Commercial purposes (with an appropriate subscription)
- Publication with source attribution

## Tips

- Download images after applying enhancements (upscale, restoration)
- Save your favorite works, as they may be deleted over time
- Use the bulk download feature in your account for multiple images
