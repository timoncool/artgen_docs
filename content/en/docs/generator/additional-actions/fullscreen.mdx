# Fullscreen Mode

![Fullscreen Mode](/images/blog/img_013.png)

Fullscreen mode allows you to view images at maximum size.

## How to Activate

1. Hover over the image
2. Click the fullscreen button (or double-click the image)
3. The image will open in fullscreen

## Features

The following features are available in fullscreen mode:
- View at maximum resolution
- Zooming
- Image panning
- Navigation between images

## Controls

- **Arrow keys** — navigate to the next/previous image
- **+/-** or mouse wheel — zoom in/out
- **Drag** — pan across the zoomed image
- **Escape** or click outside the image — exit

## When to Use

- For detailed review of results
- To check quality before downloading
- For showcasing work
- For comparing details across different generations
