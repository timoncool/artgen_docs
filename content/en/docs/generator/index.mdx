# Image Generator

![Image Generator](/images/blog/img_005.png)

The image generator is the main tool for creating images using artificial intelligence.

## How It Works

1. Enter a text description (prompt)
2. Select a model and style
3. Configure parameters
4. Click "Generate"
5. Get the result

## Generator Interface

The generator consists of several areas:
- **Left Panel** — history, favorites, folders
- **Central Area** — prompt input and result
- **Right Panel** — models, styles, parameters
