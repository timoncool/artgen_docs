# Favorites

The favorites section allows you to save liked images for quick access.

## Favorites Counter

Each image in the left panel displays a counter showing how many users have added it to favorites.

## Adding to Favorites

To add an image to favorites, use the corresponding icon when viewing the image.

## Working with Favorites

When working with images in favorites, standard actions are available:

- **Deletion** — delete the image
- **Preview** — hover over the magnifying glass icon for a quick preview
- **Fullscreen Mode** — open the image in fullscreen
- **Privacy Settings** — eye icon for managing visibility

<Callout type="info">
  On mobile devices, the section is hidden by default. Click to expand.
</Callout>
