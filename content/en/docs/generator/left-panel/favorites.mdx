# Favorites

Favorites allow you to save the best generations for quick access.

## Adding to Favorites

Click the heart icon on any image to add it to favorites.

## Managing Favorites

- View all favorites in one place
- Remove from favorites
- Organize into collections

## Using Favorites

From favorites you can:
- Quickly access best works
- Use as reference for new generations
- Share with others

## Syncing

Favorites sync across all your devices.
