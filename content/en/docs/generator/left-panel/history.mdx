# Generation History

![Generation History](/images/blog/img_007.png)

History stores all your recent generations for quick access.

## Viewing History

History displays generations in chronological order, newest first.

## History Features

- View generation parameters
- Repeat generation with same settings
- Add to favorites
- Delete from history

## Filtering

Filter history by:
- Date
- Model
- Style

## Search

Use text search to find generations by prompt.

## History Limits

History stores the last 1000 generations. Older ones are automatically archived.
