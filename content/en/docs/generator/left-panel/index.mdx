# Left Panel

The generator's left panel displays all your created images and provides tools for working with them.

## Main Features

**Chronological Order** — all your created images are displayed in chronological order.

**Search by Prompt** — the search function allows you to find images by prompt text.

**Likes and Favorites Counters** — each image displays the number of likes and favorites.

## Mobile Behavior

<Callout type="info">
  On mobile devices and small screens, the left panel is hidden by default. Click on it to expand.
</Callout>

## Image Interactions

When working with images in the left panel, the following actions are available:

- **Deletion** — delete the image
- **Preview** — hover over the magnifying glass icon for a quick preview
- **Fullscreen Mode** — open the image in fullscreen
- **Privacy Settings** — click the eye icon to manage image visibility
