# Folders

![Folders](/screens/history-panel.png)

The folder system allows you to organize your images by any criteria: projects, themes, styles, clients. This is especially useful when you have a large number of generations.

**Project Organization** â€” create a separate folder for each project and keep all related generations together.

**Thematic Sorting** â€” group images by themes: portraits, landscapes, abstractions, etc.

**Workflow** â€” create folders "In Progress", "Needs Revision", "Done" to track status.

**Client Projects** â€” if you work with clients, create folders for each order.

## Creating a Folder

<Steps>
### Open the "Folders" Tab

Navigate to the "Folders" tab on the left panel.

### Create a New Folder

Click the "+" button or "Create Folder".

### Enter a Name

Enter the folder name.

### Confirm Creation

Press Enter or click the confirmation button.
</Steps>

The folder is created and ready to use.

## Adding Images to a Folder

**Drag & Drop:**
- Grab the image with your mouse
- Drag it to the desired folder
- Release to add

**Context Menu:**

<Steps>
### Open Context Menu

Right-click on the image.

### Select Action

Choose "Move to Folder".

### Specify Folder

Select the target folder from the list.
</Steps>

**From Image View:**

<Steps>
### Open Image

Open the image in full-size mode.

### Click Folder Icon

Click on the folder icon.

### Select Folder

Choose the destination folder.
</Steps>
