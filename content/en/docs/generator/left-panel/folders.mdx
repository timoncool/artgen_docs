# Folders

Folders help organize generations by projects or themes.

## Creating Folders

1. Click "New Folder"
2. Enter name
3. Click "Create"

## Folder Operations

- **Rename** — change folder name
- **Delete** — remove folder (images move to general gallery)
- **Move** — change folder location

## Adding Images

Drag images to folder or use context menu.

## Nested Structure

Create subfolders for detailed organization.
