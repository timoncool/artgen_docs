# Quick Actions

Quick actions provide fast access to frequently used operations.

## Available Actions

- **New Generation** — start new generation
- **Repeat Last** — repeat last generation
- **Random Prompt** — generate random prompt
- **Clear Settings** — reset all parameters

## Keyboard Shortcuts

- **Ctrl+Enter** — start generation
- **Ctrl+R** — repeat last
- **Ctrl+N** — new generation
- **Esc** — cancel current operation

## Customization

Configure which actions appear in the quick panel through settings.
