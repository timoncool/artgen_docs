# Quick Actions

![Quick Actions](/screens/image-actions-toolbar.png)

Quick actions are a set of frequently used functions available with one click. They speed up workflow and allow you to perform typical operations without navigating through menus.

## Location

The quick actions panel is located at the bottom of the generator's left panel. It's always visible and accessible regardless of the selected tab (History, Favorites, Folders).

## Main Actions

**Generate** â€” starts generation with current parameters. This is the main button for creating images. Equivalent to the Ctrl+Enter keyboard shortcut.

**New Generation** â€” clears all current parameters and starts from scratch. Useful when you want to start a completely new project without influence from previous settings.

**Repeat** â€” runs generation with the same parameters but a new seed. This creates a variation of the last image while keeping all settings.

**Random Prompt** â€” generates a random idea for a prompt. Useful for inspiration or when you don't know what to create.

## Additional Actions

**Upload Image** â€” opens a file selection dialog to upload an image. Used for img2img mode when you want to create a variation of an existing image.

**Clear Prompt** â€” removes text from the prompt field, leaving other settings unchanged.

**Reset Settings** â€” returns all parameters to their default values.

## Keyboard Shortcuts

<Callout type="info">
  For maximum efficiency, use keyboard shortcuts:
</Callout>

| Action | Windows/Linux | Mac |
|--------|---------------|-----|
| Generate | Ctrl+Enter | Cmd+Enter |
| New Generation | Ctrl+N | Cmd+N |
| Repeat | Ctrl+R | Cmd+R |
| Upload Image | Ctrl+O | Cmd+O |
| Clear Prompt | Ctrl+Backspace | Cmd+Backspace |

## Contextual Actions

Some quick actions appear depending on the context:

**After generation** â€” buttons appear for "Repeat with new seed", "Increase resolution", "Save to favorites".

**When selecting an image** â€” buttons appear for "Use parameters", "Create variation", "Delete".

**When uploading an image** â€” buttons appear for configuring img2img mode.

## Panel Customization

In the generator settings you can:

- Choose which actions to display on the panel
- Change the order of buttons
- Adjust button size
- Enable or disable labels
