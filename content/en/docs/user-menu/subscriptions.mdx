# Subscriptions

Manage your subscription and tariff plan.

## Available Tariffs

### FREE
- Daily free generations
- Basic features
- Standard queue

### LIGHT
- More generations
- Priority queue
- Additional features

### PRO
- Even more generations
- Higher priority
- Advanced features

### MAX
- Maximum generations
- Highest priority
- All features unlocked

## Managing Subscription

### Upgrade
Select a higher tier to get more features.

### Downgrade
Switch to a lower tier (takes effect next billing cycle).

### Cancel
Stop auto-renewal while keeping access until period ends.

## Payment Methods

- Bank cards
- Electronic wallets
- Fast Payment System
