# My Images

Your personal gallery with all generated images.

## Gallery Features

- View all your generations
- Organize into folders
- Search and filter
- Batch operations

## Organization

### Folders
Create folders to organize images by project or theme.

### Tags
Add tags for easier searching.

### Favorites
Mark best images for quick access.

## Batch Operations

Select multiple images to:
- Download as archive
- Move to folder
- Delete
- Add tags

## Statistics

View your generation statistics:
- Total images
- By model
- By style
- By date
