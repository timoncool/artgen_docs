# User Menu

The user menu provides access to account management and personal settings.

## Menu Sections

- **Profile** — Personal information
- **Balance** — Generations and payments
- **Subscriptions** — Tariff management
- **Settings** — Preferences
- **My Images** — Personal gallery
- **Referral** — Invite friends
- **Support** — Help and feedback
