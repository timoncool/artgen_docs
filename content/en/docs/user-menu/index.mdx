# User Menu

import { Callout, Steps, Cards, Card } from '@/components/mdx'

![User Menu](/screens/profile-menu.png)

The user menu is the control center for your account on ArtGeneration.me. Here you'll find all settings, balance information, subscriptions, and tools for working with your images.

## How to Open the Menu

The user menu is accessible from any page on the site:

<Steps>
### Find the profile icon
The profile icon is located in the upper right corner of the site header.

### Open the menu
Click on the icon to open the dropdown menu.

### Select a section
Choose the desired section from the list.
</Steps>

<Callout type="info">
If you are not logged in, a "Sign In" button will be displayed instead of the menu.
</Callout>

## My Images

The "My Images" section allows you to view and manage all your created images.

### Tabs and Categories

Images are organized by tabs:
- **All** â€” all your images
- **NSFW** â€” 18+ category images
- **Safe** â€” images without adult content

### Date Folders

Images are automatically grouped by creation date:
- Today
- Yesterday
- This week
- Earlier (by month)

This allows you to quickly find images created during a specific period.

### Bulk Operations

![Bulk Download](/screens/download-my-images-dropdown.png)

You can perform actions on multiple images at once:

- **Select images** â€” click on multiple images while holding Ctrl/Cmd
- **Bulk download** â€” download all selected images as a single archive
- **Bulk delete** â€” delete multiple images at once

<Callout type="warning">
Deleted images cannot be recovered. Be careful with bulk deletion.
</Callout>
