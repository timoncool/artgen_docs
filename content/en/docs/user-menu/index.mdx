# User Menu

import { Callout, Steps, Cards, Card } from '@/components/mdx'

![User Menu](/screens/profile-menu.png)

The user menu is the control center for your account on ArtGeneration.me. Here you'll find all settings, balance information, subscriptions, and tools for working with your images.

## How to Open the Menu

The user menu is accessible from any page on the site:

<Steps>
### Find the profile icon
The profile icon is located in the upper right corner of the site header.

### Open the menu
Click on the icon to open the dropdown menu.

### Select a section
Choose the desired section from the list.
</Steps>

<Callout type="info">
If you are not logged in, a "Sign In" button will be displayed instead of the menu.
</Callout>
