# Mass Download Images

import { Callout, Steps } from '@/components/mdx'

The "Download My Images" feature allows you to download all your generated images for a selected period in a single archive.

## How to Download Images

<Steps>
### Open Settings
Go to the user menu and select "Settings".

### Find the download function
Click on the "Download My Images" button.

### Select a period
Choose the time period for download:
- Today
- Yesterday
- Week
- Month
- 3 months
- 6 months
- Year
- All time

### Create archive
Click the "Create Archive" button. The system will start preparing your files.

### Receive the link
Once the archive is ready, a download link will be sent to your email.
</Steps>

## Important Information

<Callout type="warning">
**Please note the following limitations:**
- Archive preparation takes up to **1 hour** depending on the number of images
- The download link is valid for **72 hours**
- Only **one archive** can be active at a time â€” wait for the current one to complete before creating a new one
</Callout>

## When to Use

Mass download is useful for:
- Backing up all your work
- Transferring images to another device
- Saving images before deleting your account
- Archiving work from a specific period
