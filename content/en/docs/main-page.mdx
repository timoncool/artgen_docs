# Main Page

The ArtGeneration.me main page is your starting point for working with the platform.

## Interface Elements

### Navigation Bar
At the top of the page is a navigation bar with main sections:
- **Gallery** — view user works
- **Create** — go to the image generator
- **Editor** — editing tools
- **About** — platform information
- **Guide** — documentation (you are here)

### Quick Access
On the main page you will find:
- Examples of generated images
- Quick access to the generator
- Information about platform capabilities
- Links to social networks
