# Меню пользователя

Меню пользователя предоставляет доступ к личным настройкам, балансу и управлению аккаунтом.

## Доступ к меню

Нажмите на иконку профиля в правом верхнем углу сайта для открытия меню пользователя.

## Разделы меню

- **Профиль** — личные данные и аватар
- **Баланс** — количество генераций и история
- **Подписки** — управление тарифами
- **Настройки** — персонализация интерфейса
- **Мои изображения** — личная галерея
- **Реферальная программа** — приглашение друзей
- **Поддержка** — связь с командой
