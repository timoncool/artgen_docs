# Личное

Раздел «Личное» — это ваш персональный кабинет для управления всеми созданными изображениями, лайками, избранным и подписками.

## Информация об аккаунте

В верхней части раздела отображается основная информация о вашем профиле:

- **Имя пользователя** — ваш никнейм на платформе
- **Время подписки** — срок действия текущей подписки
- **Лайки** — количество полученных лайков на ваши работы
- **Избранное** — сколько раз ваши работы добавили в избранное
- **Подписчики** — количество пользователей, подписанных на вас

<Callout type="info">
Статистика обновляется в реальном времени и помогает отслеживать популярность ваших работ.
</Callout>

## Вкладки просмотра

Раздел «Личное» содержит четыре основные вкладки для навигации по контенту:

### Созданное

Все изображения, которые вы сгенерировали. Здесь хранится полная история ваших работ, организованная по датам создания.

### Лайки

Изображения других пользователей, которые вы отметили лайком. Удобный способ вернуться к понравившимся работам.

### Избранное

Работы, которые вы добавили в избранное для быстрого доступа. Используйте эту вкладку для сохранения особенно вдохновляющих изображений.

### Подписки

Контент от пользователей, на которых вы подписаны. Следите за творчеством любимых авторов в одном месте.

## Категории сортировки

Для удобного поиска изображений доступны четыре режима сортировки:

- **Свежие** — изображения отсортированы по дате создания, от новых к старым
- **Интересные** — работы, получившие активность за последнее время
- **Популярные** — изображения с наибольшим количеством лайков и добавлений в избранное
- **Лучшие** — топ работ по совокупности всех показателей

<Callout type="info">
Переключайте режимы сортировки для поиска нужных изображений быстрее.
</Callout>

## Папки по датам

Все ваши изображения автоматически группируются по дням создания:

- **Автоматическая группировка** — изображения организованы в папки по датам без дополнительных действий
- **Удобная навигация** — быстрый переход к работам конкретного дня
- **Удаление за день** — возможность удалить все изображения за определённый день одним действием

<Steps>
### Найдите нужную дату
Папки отображаются в хронологическом порядке.

### Разверните папку
Нажмите на дату для просмотра изображений этого дня.

### Удалите при необходимости
Используйте кнопку удаления папки для очистки всех изображений за выбранный день.
</Steps>

<Callout type="warning">
Удаление папки за день удаляет все изображения в ней. Действие можно отменить в течение часа.
</Callout>

## Массовые операции

Для работы с несколькими изображениями одновременно используйте массовые операции:

### Выбор изображений

Отмечайте нужные изображения галочками для выполнения групповых действий. Галочка появляется при наведении на миниатюру.

### Всплывающий бар действий

При выборе одного или нескольких изображений в нижней части экрана появляется панель с доступными действиями:

- **Количество выбранных** — отображается число отмеченных изображений
- **Удалить выбранные** — удаление всех отмеченных изображений
- **Снять выделение** — отмена выбора всех изображений

<Steps>
### Отметьте изображения
Нажмите на галочки нужных изображений.

### Дождитесь появления панели
Всплывающий бар появится автоматически.

### Выберите действие
Нажмите «Удалить выбранные» для удаления.

### Подтвердите действие
При необходимости подтвердите удаление.
</Steps>

## Навигация и поиск

### Пагинация

Изображения разбиты на страницы для удобной навигации:

- Переход между страницами с помощью стрелок
- Отображение номера текущей страницы
- Быстрый переход к первой и последней странице

### Фильтр по промптам

Используйте строку поиска для фильтрации изображений по тексту промпта:

- Введите ключевые слова из промпта
- Результаты обновляются мгновенно
- Поиск работает по всем вашим изображениям

<Callout type="info">
Поиск по промптам помогает быстро найти изображения, созданные с определёнными параметрами или ключевыми словами.
</Callout>

## Действия с изображениями

Для каждого изображения доступны следующие действия:

### Удаление

Удалите ненужные изображения из коллекции. После удаления изображение можно восстановить в течение одного часа.

<Callout type="warning">
По истечении часа восстановление удалённого изображения невозможно. Убедитесь, что не удаляете важные работы.
</Callout>

### Настройки приватности

Управляйте видимостью ваших изображений:

- **Публичное** (открытый глаз) — изображение видно всем пользователям в галерее
- **Приватное** (закрытый глаз) — изображение видно только вам

<Steps>
### Найдите изображение
Наведите курсор на нужное изображение.

### Нажмите на иконку глаза
Иконка находится на панели действий.

### Переключите видимость
Статус изменится мгновенно.
</Steps>

### Просмотр

Откройте изображение для детального просмотра:

- Полноразмерный режим
- Информация о параметрах генерации
- Текст промпта
- Дата создания
- Статистика лайков и избранного

<Callout type="info">
В режиме просмотра доступны все стандартные действия: скачивание, создание похожего, улучшение качества и другие.
</Callout>
