# Мои изображения

Раздел "Мои изображения" — это ваша личная галерея всех созданных работ.

## Просмотр изображений

Все ваши генерации отображаются в хронологическом порядке. Используйте фильтры для поиска:
- По дате
- По модели
- По стилю
- По промпту

## Организация

### Папки
Создавайте папки для организации изображений по проектам или темам.

### Теги
Добавляйте теги для быстрого поиска.

### Избранное
Отмечайте лучшие работы для быстрого доступа.

## Массовые операции

Выделите несколько изображений для:
- Скачивания архивом
- Перемещения в папку
- Удаления
- Публикации в галерее

## Статистика

Просматривайте статистику ваших генераций:
- Общее количество
- Популярные модели
- Использованные стили
- Расход генераций

## Экспорт данных

Скачайте все ваши изображения и метаданные в архиве.
