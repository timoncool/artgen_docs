# Мои изображения

![Мои изображения](/images/blog/blog_image-8.png)

Раздел «Мои изображения» — ваша личная галерея на ArtGeneration.me. Здесь хранятся все созданные вами генерации с возможностью организации, поиска и управления.

## Структура раздела

### История генераций

Все ваши генерации автоматически сохраняются и отображаются в хронологическом порядке — от новых к старым.

**Информация о каждом изображении:**
- Превью изображения
- Дата и время создания
- Использованная модель
- Промпт (если не скрыт)
- Статус публикации

### Избранное

Быстрый доступ к лучшим работам:
- Нажмите на сердечко, чтобы добавить в избранное
- Избранные работы отображаются в отдельной вкладке
- Удобно для сохранения референсов и удачных результатов

### Папки

Организуйте работы по проектам или темам:

**Создание папки:**
1. Нажмите «Создать папку»
2. Введите название
3. Выберите цвет или иконку (опционально)
4. Нажмите «Создать»

**Добавление в папку:**
- Перетащите изображение в папку
- Или выберите папку в контекстном меню изображения
- Одно изображение может быть в нескольких папках

## Просмотр и фильтрация

### Режимы отображения

**Сетка** — компактный просмотр множества изображений

**Список** — подробная информация о каждом изображении

**Полноэкранный** — просмотр в максимальном размере

### Фильтры

Найдите нужные изображения по параметрам:

**По дате:**
- Сегодня
- Последняя неделя
- Последний месяц
- Произвольный период

**По модели:**
- Выберите одну или несколько моделей
- Фильтруйте по типу модели

**По стилю:**
- Фильтрация по использованным стилям
- Поиск по названию стиля

**По промпту:**
- Поиск по ключевым словам в промпте
- Поддержка частичного совпадения

### Сортировка

- По дате (новые/старые)
- По популярности (лайки)
- По размеру файла
- По разрешению

## Действия с изображениями

### Одиночные действия

При клике на изображение доступно:

**Просмотр:**
- Открыть в полном размере
- Посмотреть все параметры генерации
- Увидеть промпт и негативный промпт

**Редактирование:**
- Открыть в редакторе
- Создать новую версию
- Применить Upscale
- Использовать Face Swap

**Управление:**
- Добавить в избранное
- Переместить в папку
- Скачать в разных форматах
- Удалить

**Публикация:**
- Опубликовать в галерее
- Скрыть из галереи
- Поделиться ссылкой

### Массовые операции

Выделите несколько изображений для групповых действий:

**Как выделить:**
- Зажмите Ctrl/Cmd и кликайте на изображения
- Или используйте чекбоксы в режиме выделения
- Shift+клик для выделения диапазона

**Доступные операции:**
- Скачать архивом (ZIP)
- Переместить в папку
- Добавить в избранное
- Удалить выбранные
- Опубликовать/скрыть из галереи

## Статистика

Просматривайте аналитику ваших генераций:

**Общая статистика:**
- Всего генераций за всё время
- Генераций за текущий месяц
- Опубликовано в галерее
- Получено лайков

**Популярные параметры:**
- Топ используемых моделей
- Часто применяемые стили
- Любимые разрешения

**Расход генераций:**
- График использования по дням
- Распределение по типам операций

## Управление хранилищем

### Лимиты хранения

В зависимости от тарифа:
- FREE: 1000 изображений
- LIGHT: 5000 изображений
- PRO: 20000 изображений
- MAX: Безлимит

### Очистка

Освободите место, удалив старые генерации:

1. Перейдите в «Управление хранилищем»
2. Выберите критерии (старше X дней, не в избранном)
3. Просмотрите список на удаление
4. Подтвердите очистку

**Важно:** Удалённые изображения нельзя восстановить!

## Экспорт данных

### Скачивание изображений

**Одиночное скачивание:**
- Нажмите кнопку скачивания на изображении
- Выберите формат (PNG, JPG, WebP)
- Выберите качество

**Массовое скачивание:**
- Выделите нужные изображения
- Нажмите «Скачать архивом»
- Получите ZIP-файл со всеми изображениями

### Экспорт метаданных

Скачайте информацию о генерациях:
- Промпты и параметры
- Даты создания
- Использованные модели и стили

Формат: JSON или CSV

### Полный экспорт

Скачайте все данные аккаунта:
- Все изображения в оригинальном качестве
- Полные метаданные
- Структура папок

Запрос обрабатывается до 24 часов, ссылка приходит на email.
