# Массовое скачивание изображений

import { Callout, Steps } from '@/components/mdx'

Функция «Скачать мои изображения» позволяет загрузить все ваши сгенерированные изображения за выбранный период одним архивом.

## Как скачать изображения

<Steps>
### Откройте Настройки
Перейдите в меню пользователя и выберите раздел «Настройки».

### Найдите функцию скачивания
Нажмите на кнопку «Скачать мои изображения».

### Выберите период
Укажите временной период для скачивания:
- Сегодня
- Вчера
- Неделя
- Месяц
- 3 месяца
- Полгода
- Год
- Всё время

### Создайте архив
Нажмите кнопку «Создать архив». Система начнёт подготовку файлов.

### Получите ссылку
После готовности архива ссылка для скачивания будет отправлена на ваш email.
</Steps>

## Важная информация

<Callout type="warning">
**Обратите внимание на ограничения:**
- Подготовка архива занимает до **1 часа** в зависимости от количества изображений
- Ссылка для скачивания действительна **72 часа**
- Одновременно может быть активен только **один архив** — дождитесь завершения текущего перед созданием нового
</Callout>

## Когда использовать

Массовое скачивание полезно для:
- Резервного копирования всех работ
- Переноса изображений на другое устройство
- Сохранения изображений перед удалением аккаунта
- Архивирования работ за определённый период
