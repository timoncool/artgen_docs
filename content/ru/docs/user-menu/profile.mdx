# Профиль

Раздел профиля позволяет управлять личными данными и настройками аккаунта.

## Личные данные

### Имя пользователя
Отображается в комментариях и на страницах ваших работ.

### Email
Используется для входа и уведомлений. Можно изменить с подтверждением.

### Аватар
Загрузите изображение для персонализации профиля.

## Безопасность

### Смена пароля
1. Введите текущий пароль
2. Введите новый пароль
3. Подтвердите новый пароль
4. Нажмите "Сохранить"

### Двухфакторная аутентификация
Включите для дополнительной защиты аккаунта.

## Привязанные аккаунты

Вы можете привязать социальные сети для быстрого входа:
- Google
- VK
- Яндекс

## Удаление аккаунта

В разделе "Опасная зона" можно запросить удаление аккаунта. Это действие необратимо.
