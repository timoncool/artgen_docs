# Настройки

Раздел настроек позволяет персонализировать интерфейс и поведение сайта.

## Интерфейс

### Язык
Выберите язык интерфейса: русский или английский.

### Тема
Выберите тёмную или светлую тему оформления.

### Компактный режим
Уменьшает отступы для отображения большего количества контента.

## Генерация

### Настройки по умолчанию
Установите параметры, которые будут применяться к новым генерациям:
- Модель по умолчанию
- Разрешение по умолчанию
- Режим производительности

### Автосохранение
Включите автоматическое сохранение параметров последней генерации.

## Уведомления

### Email-уведомления
- Завершение генерации
- Новости и обновления
- Специальные предложения

### Push-уведомления
Получайте уведомления в браузере о завершении генераций.

## Приватность

### Приватный режим
Ваши генерации не будут отображаться в публичной галерее.

### Скрытие промптов
Другие пользователи не смогут видеть ваши промпты.
