# Настройки

![Настройки](/images/blog/img_033.png)

Раздел настроек позволяет полностью персонализировать работу с ArtGeneration.me — от внешнего вида интерфейса до параметров генерации по умолчанию.

## Настройки интерфейса

### Язык

Выберите язык интерфейса:
- Русский
- English (Английский)

Язык влияет на все элементы интерфейса, но не на контент галереи.

### Тема оформления

**Тёмная тема** — комфортна для глаз при длительной работе, особенно в тёмное время суток. Используется по умолчанию.

**Светлая тема** — классическое оформление с белым фоном.

**Системная** — автоматически переключается в зависимости от настроек вашей операционной системы.

### Компактный режим

Уменьшает отступы и размеры элементов для отображения большего количества контента на экране. Полезно для:
- Небольших экранов
- Просмотра большого количества изображений
- Работы с несколькими окнами

### Размер миниатюр

Настройте размер превью изображений в галерее и истории:
- Маленький — больше изображений на экране
- Средний — баланс между количеством и детализацией
- Большой — лучше видны детали

## Настройки генерации

### Параметры по умолчанию

Установите значения, которые будут автоматически применяться к новым генерациям:

**Модель:**
- Выберите предпочитаемую модель генерации
- Будет выбрана автоматически при открытии генератора

**Разрешение:**
- Установите стандартное разрешение
- Можно изменить для каждой генерации отдельно

**Режим производительности:**
- Быстрый — меньше шагов, быстрее результат
- Качественный — больше шагов, лучше детализация
- Максимальный — максимальное качество

**Количество изображений:**
- Сколько вариантов генерировать за раз (1-4)

### Автосохранение параметров

Когда включено:
- Параметры последней генерации сохраняются
- При следующем входе они восстанавливаются
- Удобно для работы над серией похожих изображений

### Негативный промпт по умолчанию

Установите стандартный негативный промпт, который будет добавляться ко всем генерациям. Полезно для:
- Исключения нежелательных элементов
- Поддержания единого стиля
- Экономии времени

## Уведомления

### Email-уведомления

Настройте, какие письма вы хотите получать:

**Системные:**
- Завершение длительных генераций
- Изменения в аккаунте
- Проблемы с платежами

**Информационные:**
- Новости и обновления платформы
- Новые функции и модели
- Советы по использованию

**Маркетинговые:**
- Специальные предложения
- Скидки и акции
- Партнёрские предложения

### Push-уведомления

Уведомления в браузере:

**Как включить:**
1. Перейдите в настройки уведомлений
2. Нажмите «Включить push-уведомления»
3. Разрешите уведомления в браузере

**Типы уведомлений:**
- Завершение генерации
- Ответы на комментарии
- Новые подписчики

### Telegram-уведомления

Получайте уведомления в Telegram:
1. Привяжите Telegram-аккаунт
2. Выберите типы уведомлений
3. Получайте мгновенные оповещения

## Приватность

### Приватный режим генераций

Когда включён:
- Ваши генерации не появляются в публичной галерее
- Другие пользователи не видят ваши работы
- Вы можете публиковать отдельные работы вручную

### Скрытие промптов

Когда включено:
- Промпты ваших публичных работ скрыты
- Другие пользователи видят только изображение
- Защищает ваши творческие идеи

### Скрытие параметров

Дополнительно можно скрыть:
- Модель генерации
- Стиль
- Технические параметры

### Видимость профиля

**Публичный профиль:**
- Виден всем пользователям
- Отображается в поиске
- Можно подписаться

**Приватный профиль:**
- Виден только вам
- Не отображается в поиске
- Нельзя подписаться

## Данные и хранение

### Экспорт данных

Скачайте все ваши данные:
- Изображения в оригинальном качестве
- Метаданные и параметры генерации
- История использования

### Очистка истории

Удалите старые генерации:
- Выберите период (старше месяца, года и т.д.)
- Подтвердите удаление
- Освободите место в хранилище

### Кэш и cookies

Управление локальными данными:
- Очистка кэша браузера
- Сброс настроек интерфейса
- Выход из всех сессий

## Горячие клавиши

Настройте сочетания клавиш для быстрого доступа к функциям:

| Действие | Клавиша по умолчанию |
|----------|---------------------|
| Генерация | Enter |
| Отмена | Escape |
| Сохранить в избранное | F |
| Скачать | D |
| Полный экран | Space |

Вы можете изменить любое сочетание на удобное для вас.
