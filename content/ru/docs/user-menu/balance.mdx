# Баланс и генерации

Раздел баланса показывает количество доступных генераций и историю их использования.

## Текущий баланс

Отображается количество генераций, доступных для использования:
- **Бесплатные** — ежедневные бесплатные генерации
- **Купленные** — генерации из пакетов
- **По подписке** — генерации в рамках тарифа

## Пополнение баланса

### Покупка пакетов
Выберите пакет генераций и оплатите удобным способом.

### Оформление подписки
Подписка даёт ежемесячный лимит генераций и дополнительные функции.

## История использования

Просматривайте, на что были потрачены генерации:
- Дата и время
- Тип операции (генерация, upscale, face swap)
- Количество потраченных генераций

## Бесплатные генерации

Каждый день вы получаете бесплатные генерации:
- Количество зависит от уровня аккаунта
- Неиспользованные генерации не накапливаются
- Обновление происходит в полночь по МСК
