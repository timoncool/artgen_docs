# Баланс и генерации

Раздел баланса — ваш финансовый центр на ArtGeneration.me. Здесь отображается количество доступных генераций, история использования и возможности пополнения.

## Структура баланса

Ваш баланс состоит из нескольких типов генераций:

### Ежедневные бесплатные генерации

Каждый день вы получаете бесплатные генерации:

| Тариф | Генераций в день |
|-------|------------------|
| Free | 50 |
| PRO | 300 |
| Premium | Безлимит |

**Особенности:**
- Обновляются в полночь по московскому времени
- Неиспользованные генерации не накапливаются
- Сбрасываются каждый день независимо от использования

### Несгораемые генерации

Генерации, которые остаются на балансе до использования:

- Приветственный бонус (100 генераций)
- Купленные пакеты
- Реферальные бонусы
- Промо-акции

**Особенности:**
- Не имеют срока действия
- Используются после исчерпания ежедневных
- Накапливаются без ограничений

### Генерации по подписке

Дополнительные генерации в рамках PRO или Premium подписки:

- Включены в стоимость подписки
- Действуют в течение периода подписки
- Не переносятся на следующий период

## Отображение баланса

В разделе баланса вы видите:

**Общий баланс** — сумма всех доступных генераций

**Детализация:**
- Ежедневные: X из Y (сколько осталось / сколько было)
- Несгораемые: X генераций
- По подписке: X генераций (если применимо)

**Прогресс-бар** — визуальное отображение использования ежедневного лимита

## Стоимость операций

Разные операции расходуют разное количество генераций:

| Операция | Стоимость |
|----------|-----------|
| Стандартная генерация | 1 генерация |
| Генерация в высоком разрешении | 2 генерации |
| Upscale (увеличение) | 1 генерация |
| Face Swap | 1 генерация |
| Inpainting | 1-2 генерации |
| Outpainting | 2 генерации |

Точная стоимость отображается перед выполнением операции.

## Пополнение баланса

### Покупка пакетов генераций

Доступные пакеты:

- **Стартовый** — 100 генераций
- **Базовый** — 300 генераций
- **Продвинутый** — 1000 генераций
- **Профессиональный** — 3000 генераций

Чем больше пакет, тем выгоднее цена за одну генерацию.

### Оформление подписки

Подписка даёт:
- Увеличенный ежедневный лимит
- Доступ к премиум-функциям
- Приоритетную очередь
- Скидки на пакеты генераций

### Способы оплаты

- Банковские карты (Visa, MasterCard, МИР)
- ЮMoney
- СБП (Система быстрых платежей)
- Криптовалюта (по запросу)

## История использования

Детальный лог всех операций с генерациями:

**Информация о каждой записи:**
- Дата и время операции
- Тип операции (генерация, upscale, face swap и т.д.)
- Количество потраченных генераций
- Превью результата (для генераций)

**Фильтрация:**
- По типу операции
- По дате
- По количеству генераций

**Экспорт:**
- Выгрузка истории в CSV
- Детализация для бухгалтерии

## Уведомления о балансе

Система уведомит вас:

- Когда ежедневные генерации заканчиваются
- Когда несгораемый баланс становится низким
- О начислении бонусов
- Об успешном пополнении

Настройте уведомления в разделе «Настройки».

## Советы по экономии

- Используйте режим «Быстрая генерация» для экспериментов
- Сохраняйте удачные промпты для повторного использования
- Генерируйте в стандартном разрешении, увеличивайте только лучшие результаты
- Следите за акциями и скидками на пакеты
