# Реферальная программа

Приглашайте друзей и получайте бонусные генерации.

## Как это работает

1. Получите уникальную реферальную ссылку
2. Поделитесь ссылкой с друзьями
3. Когда друг регистрируется и совершает покупку, вы получаете бонус

## Ваша реферальная ссылка

Скопируйте ссылку из раздела "Реферальная программа" в меню пользователя.

## Бонусы

### Для вас
- Бонусные генерации за каждого приглашённого
- Процент от покупок приглашённых пользователей

### Для друга
- Дополнительные бесплатные генерации при регистрации
- Скидка на первую покупку

## Статистика

Отслеживайте:
- Количество переходов по ссылке
- Количество регистраций
- Заработанные бонусы

## Вывод бонусов

Накопленные бонусы можно:
- Использовать для генераций
- Конвертировать в скидку на подписку
