# Особенности использования весов

При работе с весами важно учитывать несколько нюансов.

## Комбинирование весов

Вы можете применять веса к нескольким элементам одновременно:

```
(красный:1.3) цветок на (зелёном:0.8) фоне, (солнечный:1.2) свет
```

## Группировка элементов

Веса можно применять к группам слов:

```
(красивая девушка с длинными волосами:1.4)
```

## Влияние на качество

### Слишком высокие веса

Значения выше 2.0 могут вызвать:
- Артефакты на изображении
- Перенасыщение цветов
- Искажение форм

### Слишком низкие веса

Значения ниже 0.3 могут:
- Полностью игнорировать элемент
- Создать непредсказуемые результаты

## Совместимость с моделями

Не все модели одинаково обрабатывают веса. Экспериментируйте с разными значениями для достижения оптимального результата.
