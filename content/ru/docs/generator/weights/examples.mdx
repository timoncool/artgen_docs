# Примеры использования весов

Практические примеры применения весов в различных сценариях.

## Портретная фотография

```
(красивая женщина:1.3), (голубые глаза:1.5), длинные волосы, 
(мягкое освещение:1.2), (размытый фон:0.7), профессиональное фото
```

## Пейзаж

```
(горы:1.4) на закате, (оранжевое небо:1.3), озеро на переднем плане,
(туман:0.6) в долине, фотореалистичный стиль
```

## Фэнтези арт

```
(дракон:1.5) в полёте, (огненное дыхание:1.4), 
(средневековый замок:0.8) на фоне, эпическая сцена
```

## Аниме стиль

```
(аниме девушка:1.3), (большие глаза:1.2), розовые волосы,
школьная форма, (цветущая сакура:1.1) на фоне
```

## Натюрморт

```
(ваза с цветами:1.2), (мягкий свет:1.3) из окна,
деревянный стол, (тени:0.8), классический стиль живописи
```
