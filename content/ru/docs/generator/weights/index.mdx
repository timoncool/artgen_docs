# Веса в промптах

Веса позволяют усилить или ослабить влияние определённых элементов в запросе на итоговое изображение.

## Зачем нужны веса

По умолчанию все слова в запросе имеют одинаковый вес. Используя специальный синтаксис, вы можете:

- **Усилить** важные элементы
- **Ослабить** второстепенные детали
- **Точнее контролировать** результат генерации
