# Генератор изображений

Генератор изображений — это основной инструмент ArtGeneration.me, позволяющий создавать уникальные изображения с помощью искусственного интеллекта. Здесь вы можете воплотить любые творческие идеи в визуальную форму.

## Основные возможности

- **Текстовые запросы** — опишите желаемое изображение словами
- **Выбор моделей** — используйте различные AI-модели для разных стилей
- **Настройка параметров** — контролируйте процесс генерации
- **Дополнительные функции** — улучшение, редактирование, Face Swap

## Интерфейс генератора

Интерфейс разделён на три основные области:

1. **Центральная область** — результаты генерации и поле ввода запроса
2. **Левая панель** — история генераций и быстрые действия
3. **Правая панель** — настройки модели, стилей и параметров
