# Советы по запросам

Практические рекомендации для создания эффективных промптов.

## Следуйте структуре

Рекомендуемая структура промпта для лучших результатов:

<Steps>
### Тип изображения
Фотография, рисунок, 3D, иллюстрация

### Объект
Что или кто на изображении

### Внешний вид
Детали и характеристики объекта

### Элементы
Дополнительные объекты и детали

### Локация
Где находится объект

### Эффекты
Освещение, атмосфера, настроение

### Стиль
Художественное направление
</Steps>

## Будьте конкретны

Чем детальнее описание, тем точнее результат:

<Tabs items={['Плохо', 'Хорошо']}>
  <Tab>
    `кот`
  </Tab>
  <Tab>
    `фотография, пушистый рыжий кот, зелёные глаза, лежит на подоконнике, солнечный свет из окна, тёплые тона, уютная атмосфера`
  </Tab>
</Tabs>

## Используйте веса с умом

<Callout type="warning">
**Важно:** Веса работают только в круглых скобках `()`. Квадратные скобки `[]` уменьшают вес.
</Callout>

**Правила работы с весами:**
- Начинайте с небольших изменений (1.1-1.2)
- Увеличивайте постепенно с шагом 0.1-0.2
- Избегайте экстремальных значений (больше 1.8)
- Слишком высокие веса могут вызвать артефакты

**Примеры:**
```
(яркие:1.2) цвета, ((детализированный)) фон
(red:1.4) hair, (blue eyes:0.8)
```

## Используйте описательные прилагательные

Добавляйте качественные характеристики:
- Цвета: яркий, насыщенный, пастельный, неоновый
- Освещение: мягкое, драматичное, контрастное, рассеянное
- Настроение: уютный, мрачный, радостный, загадочный
- Текстуры: гладкий, шероховатый, блестящий, матовый

## Указывайте стиль

Определите художественное направление:
- Фотореалистичный
- Цифровая живопись
- Аниме
- Масляная живопись
- Акварель
- Киберпанк
- Фэнтези-арт

## Добавляйте технические детали

Для качественных результатов укажите:
- Разрешение: 4k, 8k, высокое качество
- Детализация: детальный, чёткий, резкий
- Рендер: octane render, unreal engine
- Камера: 85mm, портретный объектив, широкий угол

## Экспериментируйте

Не бойтесь пробовать разные формулировки. Иногда небольшое изменение в запросе даёт совершенно другой результат.

## Изучайте чужие работы

<Callout type="info">
Смотрите промпты успешных генераций в галерее и учитесь на них. Это отличный способ понять, какие формулировки работают лучше всего.
</Callout>
