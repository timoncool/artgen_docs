# Избранное

Раздел избранного позволяет сохранять понравившиеся изображения для быстрого доступа.

## Добавление в избранное

1. Наведите на изображение
2. Нажмите кнопку с сердечком
3. Изображение добавлено в избранное

## Просмотр избранного

Откройте вкладку "Избранное" на левой панели для просмотра всех сохранённых работ.

## Организация

- Сортировка по дате добавления
- Поиск среди избранного
- Перемещение в папки

## Удаление из избранного

Нажмите на сердечко повторно, чтобы удалить изображение из избранного.

## Синхронизация

Избранное синхронизируется между устройствами при входе в аккаунт.
