# Избранное

Раздел избранного позволяет сохранять понравившиеся изображения для быстрого доступа.

## Счётчик избранного

Для каждого изображения в левой панели отображается счётчик, показывающий сколько пользователей добавили его в избранное.

## Добавление в избранное

Чтобы добавить изображение в избранное, используйте соответствующую иконку при просмотре изображения.

## Работа с избранным

При работе с изображениями в избранном доступны стандартные действия:

- **Удаление** — удалить изображение
- **Предпросмотр** — наведите курсор на иконку лупы для быстрого просмотра
- **Полноэкранный режим** — открыть изображение на весь экран
- **Настройки приватности** — иконка глаза для управления видимостью

<Callout type="info">
  На мобильных устройствах раздел скрыт по умолчанию. Нажмите, чтобы развернуть.
</Callout>
