# Левая панель

Левая панель генератора отображает все ваши созданные изображения и предоставляет инструменты для работы с ними.

## Основные возможности

**Хронологический порядок** — все созданные вами изображения отображаются в хронологическом порядке.

**Поиск по промпту** — функция поиска позволяет находить изображения по тексту промпта.

**Счётчики лайков и избранного** — для каждого изображения отображается количество лайков и добавлений в избранное.

## Поведение на мобильных устройствах

<Callout type="info">
  На мобильных устройствах и экранах с небольшим разрешением левая панель скрыта по умолчанию. Нажмите на неё, чтобы развернуть.
</Callout>

## Взаимодействие с изображениями

При работе с изображениями в левой панели доступны следующие действия:

- **Удаление** — удалить изображение. Пока вы не обновили страницу (в течение часа), у вас есть возможность восстановить удалённое изображение.
- **Предпросмотр** — наведите курсор на иконку лупы, и изображение моментально откроется для предпросмотра с информацией о дате создания и начале запроса.
- **Полноэкранный режим** — открыть изображение на весь экран с функцией приближения.
- **Настройки приватности** — нажмите на иконку глаза для управления видимостью. Открытый глаз означает, что изображение доступно для всех, закрытый — видно только вам.

<Callout type="info">
  Счётчики лайков и избранного помогают избежать случайного удаления популярных работ.
</Callout>
