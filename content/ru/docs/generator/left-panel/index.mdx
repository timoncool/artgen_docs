# Левая панель

Левая панель генератора содержит инструменты для работы с историей и организации ваших изображений.

## Элементы панели

- **История генераций** — все ваши предыдущие генерации
- **Избранное** — сохранённые понравившиеся работы
- **Папки** — организация изображений по категориям
- **Быстрые действия** — часто используемые функции

## Сворачивание панели

Панель можно свернуть для увеличения рабочей области. Нажмите на стрелку в верхней части панели.
