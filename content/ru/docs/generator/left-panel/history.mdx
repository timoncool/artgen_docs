# История генераций

История генераций сохраняет все ваши созданные изображения для быстрого доступа.

## Возможности

- Просмотр всех предыдущих генераций
- Фильтрация по дате
- Поиск по промпту
- Быстрое копирование параметров

## Навигация

История отображается в виде сетки миниатюр. Прокрутите вниз для загрузки более старых генераций.

## Действия с изображениями

При клике на изображение в истории:
- Открывается просмотр
- Доступны все дополнительные действия
- Можно скопировать параметры для новой генерации

## Хранение

История хранится на сервере и доступна с любого устройства при входе в аккаунт.

## Очистка истории

Вы можете удалить отдельные изображения или очистить всю историю в настройках аккаунта.
