# История генераций

![История генераций](/images/blog/img_007.png)

История генераций — это полный архив всех ваших созданных изображений. Каждая генерация автоматически сохраняется, позволяя вам вернуться к любой работе в любой момент.

## Интерфейс истории

![Интерфейс истории](/images/blog/img_008.png)

История отображается в виде сетки миниатюр, отсортированных по дате создания (новые сверху). Каждая миниатюра показывает превью изображения и при наведении отображает дополнительную информацию.

**Бесконечная прокрутка** — при прокрутке вниз автоматически загружаются более старые генерации. Вам не нужно переключать страницы.

**Группировка по дате** — генерации группируются по дням для удобной навигации. Вы увидите разделители с датами между группами изображений.

## Информация о генерации

При наведении на миниатюру отображается:

- Дата и время создания
- Используемая модель
- Начало промпта (первые слова)
- Разрешение изображения

## Поиск в истории

**Поиск по промпту** — введите ключевые слова в поле поиска, чтобы найти генерации с определёнными промптами. Поиск работает по всему тексту промпта.

**Фильтр по дате** — выберите период времени для отображения генераций только за определённые дни, недели или месяцы.

**Фильтр по модели** — отфильтруйте историю, чтобы показать только работы, созданные определённой моделью.

## Действия с изображениями

**Клик на изображение** — открывает полноразмерный просмотр с детальной информацией о генерации.

**Копирование параметров** — нажмите на иконку копирования, чтобы перенести все параметры генерации в текущие настройки. Это позволяет быстро создать вариацию или продолжить работу с теми же настройками.

**Добавление в избранное** — отметьте лучшие работы для быстрого доступа в будущем.

**Перемещение в папку** — организуйте изображения по проектам или темам.

**Удаление** — удалите ненужные генерации. Удаление необратимо.

## Хранение и синхронизация

**Облачное хранение** — вся история хранится на серверах ArtGeneration.me и доступна с любого устройства при входе в ваш аккаунт.

**Автоматическое сохранение** — каждая генерация сохраняется автоматически сразу после создания.

**Лимиты хранения** — количество сохраняемых генераций зависит от вашего тарифного плана. При достижении лимита самые старые генерации могут быть удалены.

## Массовые операции

**Выбор нескольких изображений** — удерживайте Ctrl (Cmd на Mac) и кликайте на изображения для множественного выбора.

**Массовое удаление** — выберите несколько изображений и удалите их одним действием.

**Массовое перемещение** — переместите несколько изображений в папку одновременно.

## Советы по работе с историей

- Регулярно добавляйте лучшие работы в избранное
- Используйте папки для организации по проектам
- Копируйте параметры успешных генераций для создания вариаций
- Используйте поиск для быстрого нахождения нужных работ
