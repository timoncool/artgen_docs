# История генераций

История генераций — это раздел, где отображаются все созданные вами изображения.

## Отображение изображений

Все ваши изображения отображаются в хронологическом порядке — от новых к старым.

## Поиск

Используйте функцию поиска для нахождения изображений по тексту промпта. Просто введите ключевые слова, и система покажет соответствующие генерации.

## Мобильная версия

<Callout type="info">
  На мобильных устройствах и экранах с небольшим разрешением раздел истории скрыт по умолчанию. Нажмите на панель, чтобы развернуть её.
</Callout>

## Действия с изображениями

При работе с изображениями доступны следующие действия:

**Удаление** — удалите ненужные изображения из истории. Пока вы не обновили страницу (в течение часа), у вас есть возможность восстановить удалённое изображение.

**Предпросмотр** — наведите курсор на иконку лупы, и изображение моментально откроется для предпросмотра с информацией о дате создания и начале запроса.

**Полноэкранный режим** — откройте изображение на весь экран для детального просмотра с функцией приближения.

**Настройки приватности** — нажмите на иконку глаза, чтобы управлять видимостью изображения. Открытый глаз означает, что изображение доступно для всех пользователей, закрытый — видно только вам.

## Статистика изображений

Для каждого изображения отображаются счётчики лайков и добавлений в избранное от других пользователей. Эта информация помогает избежать случайного удаления популярных работ.
