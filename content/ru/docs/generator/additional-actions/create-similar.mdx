# Создать похожее

![Создать похожее](/images/blog/blog_image-14.png)

Функция "Создать похожее" генерирует вариации изображения с теми же параметрами, но другим seed.

## Как работает

При нажатии кнопки "Создать похожее":

1. Копируются все параметры исходной генерации
2. Генерируется новый seed
3. Запускается новая генерация

## Результат

Вы получите изображение:
- С тем же промптом
- С той же моделью и стилем
- С теми же техническими параметрами
- Но с другой случайной вариацией

## Когда использовать

- Когда результат почти идеален, но хочется попробовать другие варианты
- Для создания серии похожих изображений
- Для поиска лучшей интерпретации промпта

## Советы

Если вам нравится общее направление, но не конкретный результат, используйте эту функцию несколько раз, чтобы найти идеальный вариант.
