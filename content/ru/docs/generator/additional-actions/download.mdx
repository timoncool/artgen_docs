# Скачать

![Скачать](/images/blog/blog_image-15.png)

Функция скачивания позволяет сохранить изображение на ваше устройство.

## Как скачать

1. Наведите на изображение
2. Нажмите кнопку "Скачать"
3. Файл будет загружен в папку загрузок

## Формат файла

Изображения скачиваются в формате PNG с максимальным качеством.

## Имя файла

Имя файла содержит:
- Идентификатор генерации
- Дату создания
- Расширение .png

## Права на использование

Скачанные изображения можно использовать:
- В личных проектах
- В коммерческих целях (при наличии соответствующей подписки)
- Для публикации с указанием источника

## Советы

- Скачивайте изображения после применения улучшений (upscale, восстановление)
- Сохраняйте понравившиеся работы, так как они могут быть удалены со временем
- Используйте функцию массового скачивания в личном кабинете для множества изображений
