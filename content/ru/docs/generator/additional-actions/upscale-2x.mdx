# Увеличение 2x (Upscale)

![Увеличение 2x](/screens/image-actions-toolbar.png)

Функция увеличения удваивает размер изображения с сохранением качества.

## Как работает

Upscale использует специальные AI-алгоритмы для:
- Увеличения размера изображения в 2 раза
- Добавления деталей
- Сохранения чёткости
- Устранения артефактов масштабирования

## Как использовать

<Steps>
### Наведите на изображение
Выберите изображение, которое хотите увеличить.

### Нажмите кнопку "Увеличение 2x"
Кнопка появится на панели действий.

### Дождитесь обработки
Результат откроется во всплывающем окне.

### Скачайте результат
Увеличенное изображение необходимо скачать вручную из всплывающего окна.
</Steps>

<Callout type="info">
Результат увеличения открывается во всплывающем окне. Чтобы сохранить увеличенное изображение, его нужно скачать вручную — увеличенная версия **не сохраняется автоматически** в вашу галерею.
</Callout>

## Результат

Изображение 512x512 станет 1024x1024 с улучшенной детализацией.

## Когда использовать

- Для печати изображений
- Для использования в качестве обоев
- Для публикации в высоком разрешении
- Когда нужна максимальная детализация

## Стоимость

Upscale расходует генерации из вашего баланса. Количество зависит от размера исходного изображения.

## Ограничения

<Callout type="warning">
- Максимальный размер после увеличения ограничен
- Очень маленькие изображения могут терять качество
- Некоторые артефакты могут усиливаться при увеличении
</Callout>
