# Инпейтинг

Редактируйте определенные части изображения, сохраняя остальное без изменений.

## Как это работает

1. Выберите область для редактирования (маска)
2. Опишите, что должно там появиться
3. Создайте редактирование

## Создание масок

- Инструмент кисти для рисования
- Инструменты выделения для фигур
- Автоматическое обнаружение объектов

## Примеры использования

- Удалите нежелательные объекты
- Добавьте новые элементы
- Исправьте недостатки
- Измените конкретные детали

## Советы

- Делайте маски немного больше, чем область
- Используйте четкие описания
- Соответствуйте освещению и стилю
