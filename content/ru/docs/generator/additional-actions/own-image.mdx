# Своё изображение

Функция "Своё изображение" позволяет использовать сгенерированное или загруженное изображение как основу для новой генерации (img2img).

## Как использовать

1. Наведите на изображение в результатах генерации
2. Нажмите кнопку "Своё изображение"
3. Изображение будет добавлено в панель настроек
4. Настройте силу влияния исходного изображения
5. Введите новый промпт или измените существующий
6. Запустите генерацию

## Сила влияния

Параметр "Сила влияния" определяет, насколько результат будет похож на исходное изображение:

- **0.1-0.3** — минимальные изменения, сохранение структуры
- **0.4-0.6** — умеренные изменения, баланс между оригиналом и промптом
- **0.7-0.9** — значительные изменения, промпт доминирует

## Применение

- Создание вариаций понравившегося изображения
- Изменение стиля существующего изображения
- Добавление или удаление элементов
- Улучшение композиции
