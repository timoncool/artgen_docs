# Негативный промпт

![Негативный промпт](/screens/generator-right-panel.png)

Негативный промпт — это инструмент, который позволяет указать нейросети, чего следует избегать при генерации изображения. Он вводится в поле **«Избегать»** на панели настроек.

## Как работает негативный промпт

Когда вы указываете что-то в поле «Избегать», модель старается «отодвинуться» от этих концепций в процессе генерации. Это не гарантия, что элемент не появится, но значительно снижает вероятность.

Негативный промпт особенно полезен для:
- Устранения типичных артефактов нейросетей
- Исключения нежелательных элементов
- Повышения общего качества изображения
- Коррекции стиля генерации

## Синтаксис весов в негативном промпте

<Callout type="warning">
**Важно:** В негативном промпте также работают веса. Используйте круглые скобки `()` для усиления, квадратные `[]` — для ослабления.
</Callout>

```
(blurry:1.4), low quality, ((deformed hands))
```

## Базовый негативный промпт

В ArtGeneration.me для большинства моделей уже настроен базовый негативный промпт, который помогает избежать самых распространенных проблем. Вы можете дополнить его своими требованиями.

## Эффективные негативные промпты

### Для повышения качества
```
low quality, worst quality, blurry, pixelated,
jpeg artifacts, compression artifacts, noise
```

### Для портретов и людей
```
deformed, ugly, mutilated, disfigured,
extra limbs, extra fingers, missing fingers,
bad anatomy, bad proportions, gross proportions,
malformed limbs, fused fingers, too many fingers,
long neck, cross-eyed, mutated hands
```

### Для фотореализма
```
cartoon, anime, illustration, painting, drawing,
3d render, cgi, unrealistic, artificial
```

### Для аниме и иллюстраций
```
realistic, photo, photograph, 3d,
bad anatomy, bad hands, text, watermark
```

### Универсальный негативный промпт
```
lowres, bad anatomy, bad hands, text, error,
missing fingers, extra digit, fewer digits,
cropped, worst quality, low quality,
normal quality, jpeg artifacts, signature,
watermark, username, blurry
```

## Советы по использованию

<Callout type="warning">
**Не перегружайте негативный промпт.** Слишком длинный негативный промпт может запутать модель и ухудшить результат. Сосредоточьтесь на самых важных исключениях.
</Callout>

**Используйте конкретные термины.** Вместо абстрактного «плохое качество» лучше указать конкретные проблемы: «размытое», «пикселизация», «артефакты сжатия».

**Применяйте умеренные веса.** Начинайте с небольших значений (1.1-1.3) и увеличивайте по необходимости.

**Экспериментируйте.** Разные модели по-разному реагируют на негативные промпты. То, что работает для одной модели, может не работать для другой.

<Callout type="info">
**Язык негативного промпта.** Как и основной промпт, негативный промпт лучше писать на английском языке для большинства моделей.
</Callout>

## Когда негативный промпт не помогает

Негативный промпт не всесилен. Если проблема связана с ограничениями самой модели или неудачным основным промптом, негативный промпт может не помочь. В таких случаях попробуйте:
- Изменить основной промпт
- Выбрать другую модель
- Использовать инпейнтинг для исправления проблемных областей
