# Негативный промпт

Негативный промпт указывает, чего следует избегать в генерации.

## Зачем нужен

Негативный промпт помогает:
- Избежать нежелательных элементов
- Улучшить качество изображения
- Устранить типичные артефакты

## Как использовать

Введите в поле негативного промпта элементы, которые не должны появляться:

```
размытое, низкое качество, деформированные руки, 
лишние конечности, текст, водяной знак
```

## Типичные негативные промпты

### Для портретов
```
деформированное лицо, лишние пальцы, размытые глаза,
асимметричное лицо, неестественная кожа
```

### Для пейзажей
```
люди, текст, логотипы, низкое качество, 
размытый фон, артефакты
```

### Универсальный
```
низкое качество, размытое, пикселизация, 
артефакты сжатия, водяной знак, подпись
```

## Советы

- Не перегружайте негативный промпт
- Используйте конкретные термины
- Экспериментируйте с разными комбинациями
