# Face Swap

Face Swap позволяет заменить лицо на сгенерированном изображении на лицо с другой фотографии.

## Как использовать

1. Сгенерируйте изображение с лицом
2. Откройте раздел "Face Swap"
3. Загрузите фото с желаемым лицом
4. Нажмите "Применить"

## Требования к фото

Для лучших результатов:
- Лицо должно быть чётко видно
- Фронтальный ракурс предпочтительнее
- Хорошее освещение
- Высокое разрешение

## Настройки

- **Сила замены** — насколько точно копировать черты лица
- **Сохранение выражения** — сохранить выражение оригинала или источника

## Ограничения

- Работает лучше с фотореалистичными изображениями
- Сложные ракурсы могут давать артефакты
- Стилизованные изображения (аниме) могут не поддерживаться

## Этика использования

- Используйте только свои фото или фото с разрешением
- Не создавайте контент, нарушающий права других людей
- Соблюдайте правила платформы
