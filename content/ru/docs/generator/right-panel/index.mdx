# Правая панель

Правая панель генератора содержит все настройки для управления процессом генерации изображений.

## Основные разделы

- **Модели** — выбор AI-модели для генерации
- **Стили** — художественные стили и пресеты
- **LoRA** — дополнительные модели для специфических эффектов
- **Исходное изображение** — img2img генерация
- **Face Swap** — замена лиц на изображениях
- **Параметры** — технические настройки генерации

## Сворачивание панели

Панель можно свернуть для увеличения области просмотра результатов.
