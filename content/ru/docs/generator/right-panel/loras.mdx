# LoRA

LoRA (Low-Rank Adaptation) — это дополнительные модели, которые добавляют специфические возможности к основной модели.

## Что такое LoRA

LoRA — это небольшие модели, обученные на специфических данных:
- Определённые персонажи
- Специфические стили
- Особые эффекты
- Конкретные объекты

## Использование LoRA

1. Откройте раздел "LoRA" на правой панели
2. Найдите нужную LoRA
3. Нажмите для добавления
4. Настройте силу влияния

## Сила влияния (Weight)

Параметр силы определяет, насколько сильно LoRA влияет на результат:
- **0.3-0.5** — лёгкое влияние
- **0.6-0.8** — умеренное влияние
- **0.9-1.0** — сильное влияние

## Комбинирование LoRA

Можно использовать несколько LoRA одновременно, но:
- Общая сумма весов не должна превышать 1.5-2.0
- Конфликтующие LoRA могут давать артефакты

## Совместимость

Не все LoRA совместимы со всеми моделями. Проверяйте рекомендации в описании LoRA.
