# LoRA

![LoRA](/images/blog/blog_image-16.png)

LoRA (Low-Rank Adaptation) — это дополнительные модели, которые добавляют специфические возможности к основной модели. В ArtGeneration.me доступна обширная библиотека LoRA для расширения возможностей генерации.

## Что такое LoRA

![Выбор LoRA](https://lh7-us.googleusercontent.com/LoRA_selection-Paflkahe2Vc_YBBoWS9_CLfG1VAWH5SMOAQQmeTn4fJPIb8Q_kWeDFE-l6MvPkG_tNGgJAjv0GW39VI0b2BidpJ75eZ9DAcYynavnuJtUO29hzM_9CbKTP7F6vidosxUKJ6kNogQhUko)

LoRA — это небольшие дообученные модели, которые добавляют к базовой модели новые знания:

- **Персонажи** — конкретные персонажи из игр, фильмов, аниме
- **Стили художников** — уникальные художественные стили
- **Концепции** — специфические визуальные концепции
- **Одежда и аксессуары** — определенные типы одежды, украшений
- **Позы и композиции** — специфические позы и ракурсы

LoRA работают поверх основной модели, добавляя к ней дополнительные «знания». Это позволяет получить результаты, которые невозможно достичь только с помощью промпта.

## Как использовать LoRA

1. Откройте раздел «LoRA» на правой панели генератора
2. Просмотрите доступные LoRA или воспользуйтесь поиском
3. Нажмите на карточку LoRA для добавления
4. Настройте силу влияния с помощью ползунка
5. При необходимости добавьте триггер-слово в промпт

## Триггер-слова

Многие LoRA требуют использования специальных триггер-слов в промпте для активации. Триггер-слово указано в описании каждой LoRA. Без триггер-слова LoRA может работать слабо или не работать вовсе.

Например, если LoRA обучена на персонаже и имеет триггер «1girl, character_name», вам нужно включить эти слова в ваш промпт.

## Сила влияния (Weight)

Параметр силы определяет, насколько сильно LoRA влияет на результат:

- **0.1-0.3** — минимальное влияние, легкий намек на стиль
- **0.4-0.6** — умеренное влияние, баланс между базовой моделью и LoRA
- **0.7-0.9** — сильное влияние, LoRA доминирует
- **1.0** — максимальное влияние

Рекомендуется начинать с значения **0.6-0.7** и корректировать по результатам. Слишком высокое значение может привести к артефактам или «пережаренному» изображению.

## Комбинирование нескольких LoRA

Вы можете использовать несколько LoRA одновременно для комбинирования эффектов. Однако учитывайте:

- Общая сумма весов всех LoRA не должна превышать **1.5-2.0**
- Конфликтующие LoRA (например, два разных стиля) могут давать непредсказуемые результаты
- Начните с одной LoRA, затем добавляйте другие постепенно

## Совместимость с моделями

Не все LoRA совместимы со всеми базовыми моделями:

- LoRA для **SDXL** работают только с SDXL-моделями
- LoRA для **SD 1.5** работают только с SD 1.5 моделями
- LoRA для **FLUX** работают только с FLUX-моделями

Информация о совместимости указана в описании каждой LoRA. Использование несовместимой LoRA может привести к ошибкам или некачественным результатам.

- Комбинируйте LoRA с подходящими стилями для лучших результатов
