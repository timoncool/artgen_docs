# Исходное изображение (Init Image)

Функция исходного изображения позволяет использовать существующее изображение как основу для генерации (img2img).

## Загрузка изображения

1. Откройте раздел "Исходное изображение"
2. Нажмите "Загрузить" или перетащите файл
3. Изображение появится в превью

## Сила влияния (Denoising Strength)

Этот параметр определяет, насколько результат будет отличаться от исходного изображения:

- **0.1-0.3** — минимальные изменения, сохранение структуры
- **0.4-0.6** — умеренные изменения
- **0.7-0.9** — значительные изменения, промпт доминирует

## Применение

- **Изменение стиля** — превратить фото в картину
- **Добавление элементов** — дополнить изображение деталями
- **Улучшение** — повысить качество или детализацию
- **Вариации** — создать похожие изображения

## Маска

Вы можете указать область для изменения с помощью маски, оставив остальную часть изображения нетронутой.

## Советы

- Начните с низкой силы влияния
- Используйте промпт, описывающий желаемые изменения
- Экспериментируйте с разными моделями
