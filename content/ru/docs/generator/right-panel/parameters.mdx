# Параметры генерации

Технические параметры позволяют тонко настроить процесс генерации изображений. Понимание этих параметров поможет вам получать более качественные и предсказуемые результаты.

## Seed (Сид)

Seed — это начальное значение для генератора случайных чисел. Это число определяет «случайность» генерации.

**Как работает:**
- При одинаковом seed и всех остальных параметрах вы получите идентичное изображение
- Случайный seed даёт уникальный результат каждый раз
- Вы можете скопировать seed понравившейся генерации и использовать его для создания вариаций

**Применение:**
- Нашли удачную композицию? Зафиксируйте seed и экспериментируйте с промптом
- Хотите серию похожих изображений? Используйте один seed с небольшими изменениями промпта
- Нужна воспроизводимость? Сохраните seed вместе с остальными параметрами

## CFG Scale (Guidance Scale)

CFG Scale определяет, насколько строго модель следует вашему промпту. Это один из самых важных параметров для контроля результата.

**Значения:**
- **1-4** — Модель имеет большую творческую свободу, может значительно отклоняться от промпта. Результаты могут быть неожиданными, но иногда интересными.
- **5-7** — Баланс между следованием промпту и творческой свободой. Рекомендуется для большинства задач.
- **8-12** — Модель строго следует промпту. Хорошо для точных запросов, но может снизить «живость» изображения.
- **13-20** — Очень строгое следование. Может привести к артефактам, перенасыщенности или «пережаренному» виду.

**Рекомендация:** Начните с **7** и корректируйте по результатам.

## Шаги генерации (Steps)

Количество шагов определяет, сколько итераций пройдёт модель при создании изображения. Больше шагов — больше деталей, но и больше времени.

**Значения:**
- **15-25** — Быстрая генерация, базовое качество. Подходит для тестирования идей.
- **25-35** — Оптимальный баланс качества и скорости. Рекомендуется для большинства задач.
- **35-50** — Высокое качество, больше деталей. Для финальных работ.
- **50+** — Максимальное качество, но значительно дольше. Не всегда оправдано.

**Важно:** После определённого количества шагов качество перестаёт улучшаться. Для большинства моделей это 30-40 шагов.

## Сэмплер (Sampler)

Сэмплер — это алгоритм, который определяет, как модель «движется» от шума к финальному изображению. Разные сэмплеры дают разные результаты.

**Популярные сэмплеры:**
- **Euler** — Быстрый и стабильный. Хороший выбор по умолчанию.
- **Euler a** — Вариант Euler с дополнительной случайностью. Более «живые» результаты.
- **DPM++ 2M** — Отличная детализация, стабильные результаты.
- **DPM++ 2M Karras** — Улучшенная версия с более плавными переходами.
- **DPM++ SDE** — Добавляет случайность, хорош для органических текстур.
- **DDIM** — Стабильный и предсказуемый. Хорош для воспроизводимости.

**Совет:** Если не знаете, какой выбрать — используйте **DPM++ 2M Karras** или **Euler a**.

## Взаимосвязь параметров

Параметры влияют друг на друга:
- Высокий CFG Scale может потребовать больше шагов для качественного результата
- Некоторые сэмплеры работают лучше с определённым количеством шагов
- Разные модели имеют свои оптимальные настройки

## Сохранение настроек

Когда вы нашли идеальную комбинацию параметров для определённого типа изображений, сохраните их как пресет для быстрого доступа в будущем.
