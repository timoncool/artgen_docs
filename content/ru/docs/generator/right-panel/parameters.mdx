# Параметры генерации

Технические параметры позволяют тонко настроить процесс генерации.

## Основные параметры

### Seed

Начальное значение для генератора случайных чисел. Одинаковый seed с теми же параметрами даёт идентичный результат.

- **Случайный** — каждая генерация уникальна
- **Фиксированный** — воспроизводимый результат

### CFG Scale (Guidance Scale)

Определяет, насколько строго модель следует промпту:
- **1-5** — творческая свобода, может отклоняться от промпта
- **7-10** — баланс между точностью и качеством
- **12-20** — строгое следование промпту, возможны артефакты

### Шаги (Steps)

Количество итераций генерации:
- **20-30** — быстрая генерация, базовое качество
- **30-50** — оптимальный баланс
- **50-100** — максимальное качество, долгая генерация

### Сэмплер (Sampler)

Алгоритм генерации. Разные сэмплеры дают разные результаты:
- **Euler** — быстрый, хорошее качество
- **DPM++** — детализированный результат
- **DDIM** — стабильный, предсказуемый

## Сохранение настроек

Вы можете сохранить набор параметров как пресет для быстрого доступа.
