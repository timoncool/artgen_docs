# Режим производительности

Режим производительности позволяет выбрать баланс между скоростью и качеством генерации.

## Доступные режимы

### Набросок (Draft)
- Быстрая генерация
- Базовое качество
- Идеально для тестирования идей
- Минимальный расход генераций

### Эскиз (Sketch)
- Средняя скорость
- Хорошее качество
- Подходит для большинства задач
- Умеренный расход генераций

### Шедевр (Masterpiece)
- Медленная генерация
- Максимальное качество
- Для финальных работ
- Повышенный расход генераций

## Когда использовать

- **Набросок** — поиск идей, тестирование промптов
- **Эскиз** — повседневная работа
- **Шедевр** — важные проекты, печать

## Влияние на параметры

Режим автоматически настраивает:
- Количество шагов
- CFG Scale
- Сэмплер

## Советы

Начните с режима "Набросок" для поиска идеального промпта, затем переключитесь на "Шедевр" для финальной генерации.
