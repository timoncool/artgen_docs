# Поиск и категории

Галерея предоставляет мощные инструменты для поиска и фильтрации изображений.

## Поиск

Используйте строку поиска для нахождения изображений по:

- **Ключевым словам** — описание или теги изображения
- **Промптам** — текстовые запросы, использованные при генерации
- **Авторам** — имя пользователя, создавшего изображение

## Фильтры

Дополнительные фильтры помогут сузить результаты:

- **Дата создания** — новые, за неделю, за месяц
- **Популярность** — по количеству лайков
- **Модель** — фильтр по используемой модели генерации
- **Стиль** — фильтр по применённому стилю
