# Детали генерации

На странице просмотра изображения отображаются все параметры, использованные при генерации. Эта информация позволяет понять, как было создано изображение, и использовать эти знания для собственных работ.

## Промпт (Prompt)

Основной текстовый запрос, который описывает желаемое изображение. Промпт — это «инструкция» для нейросети.

**Что можно узнать из промпта:**
- Какие ключевые слова использовал автор
- Как структурирован запрос
- Какие детали были указаны явно

**Копирование промпта** — нажмите на иконку копирования рядом с промптом, чтобы скопировать его в буфер обмена. Это удобно для изучения и создания собственных вариаций.

## Негативный промпт (Negative Prompt)

Указывает, чего следует избегать в генерации. Негативный промпт помогает исключить нежелательные элементы и артефакты.

Изучение негативных промптов опытных пользователей поможет вам понять, какие исключения эффективны для разных типов изображений.

## Модель (Model)

Название модели, использованной для генерации. Разные модели имеют разные сильные стороны:

- Некоторые модели лучше для фотореализма
- Другие оптимизированы для аниме или иллюстраций
- Есть модели для специфических стилей и тематик

Зная модель, вы можете попробовать создать похожие работы с теми же настройками.

## Стиль (Style)

Если автор использовал один из предустановленных стилей ArtGeneration.me, он будет указан здесь. Стили добавляют специфические художественные характеристики к изображению.

## Технические параметры

**Разрешение** — размер изображения в пикселях (например, 1024x1024). Влияет на детализацию и пропорции.

**Шаги (Steps)** — количество итераций генерации. Больше шагов обычно означает более детализированный результат.

**CFG Scale** — насколько строго модель следовала промпту. Низкие значения дают больше творческой свободы, высокие — более точное следование запросу.

**Сэмплер (Sampler)** — алгоритм генерации. Разные сэмплеры дают разные результаты даже при одинаковых параметрах.

**Seed** — начальное значение генератора случайных чисел. С одинаковым seed и параметрами можно воспроизвести идентичный результат.

## LoRA и дополнительные модели

Если при генерации использовались LoRA или другие дополнительные модели, они также будут указаны с информацией о силе влияния (weight).

## Использование параметров

**Кнопка «Создать версию»** — автоматически копирует все параметры в генератор, позволяя создать собственную вариацию.

**Копирование отдельных параметров** — каждый параметр можно скопировать отдельно, нажав на соответствующую иконку.

**Изучение и обучение** — анализ параметров успешных работ — отличный способ улучшить свои навыки в создании промптов и настройке генерации.
