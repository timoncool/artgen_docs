# Детали генерации

На странице просмотра изображения отображаются все параметры, использованные при генерации.

## Основные параметры

- **Промпт** — текстовый запрос, описывающий изображение
- **Негативный промпт** — что исключить из генерации
- **Модель** — какая модель использовалась
- **Стиль** — применённый художественный стиль

## Технические параметры

- **Разрешение** — размер изображения в пикселях
- **Шаги** — количество шагов генерации
- **CFG Scale** — сила следования промпту
- **Сэмплер** — алгоритм генерации
- **Seed** — начальное значение для воспроизводимости

## Использование параметров

Вы можете скопировать любые параметры для использования в своих генерациях, нажав на соответствующую кнопку копирования.
