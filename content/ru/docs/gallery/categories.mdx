# Категории галереи

Галерея организована по категориям для удобной навигации.

## Основные категории

- **Все** — все изображения без фильтрации
- **Портреты** — изображения людей и персонажей
- **Пейзажи** — природа и окружающая среда
- **Фэнтези** — фантастические миры и существа
- **Sci-Fi** — научная фантастика
- **Аниме** — изображения в стиле аниме
- **Абстракция** — абстрактное искусство
- **Архитектура** — здания и сооружения

## Сортировка

Внутри каждой категории доступна сортировка:

- **Новые** — по дате создания (сначала новые)
- **Популярные** — по количеству лайков
- **Обсуждаемые** — по количеству комментариев
