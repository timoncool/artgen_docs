# Просмотр изображения

![Просмотр изображения](/screens/image-actions-toolbar.png)

При клике на изображение в галерее открывается полноэкранный режим просмотра. Здесь вы можете детально изучить работу, узнать параметры генерации и взаимодействовать с изображением.

## Интерфейс просмотра

Страница просмотра изображения состоит из нескольких основных областей:

**Область изображения** — центральная часть экрана с изображением в высоком разрешении. Изображение масштабируется для оптимального отображения на вашем экране.

**Панель информации** — справа или снизу (в зависимости от устройства) расположена панель с детальной информацией о работе.

**Панель действий** — кнопки для взаимодействия с изображением: лайк, избранное, скачивание, создание версии.

## Информация об изображении

На странице просмотра отображается:

**Автор** — аватар, имя пользователя и ссылка на профиль создателя. Вы можете перейти в профиль, чтобы увидеть другие работы этого автора.

**Дата создания** — когда было создано изображение.

**Статистика** — количество просмотров, лайков и добавлений в избранное.

**Промпт** — полный текстовый запрос, использованный для генерации. Вы можете скопировать промпт одним кликом.

**Параметры генерации** — модель, стиль, разрешение, seed и другие технические параметры.

## Действия с изображением

**Лайк** — отметьте изображение как понравившееся. Это помогает автору и влияет на рейтинг работы.

**Избранное** — добавьте изображение в свою коллекцию для быстрого доступа в будущем.

**Скачать** — загрузите изображение в полном разрешении на своё устройство.

**Создать версию** — перейдите в генератор с параметрами этого изображения для создания собственной вариации.

**Поделиться** — скопируйте ссылку на изображение для отправки друзьям.

## Навигация между изображениями

<Callout type="info">
  Используйте клавиши стрелок на клавиатуре для быстрой навигации между изображениями.
</Callout>

**Стрелки на экране** — кнопки слева и справа от изображения для перехода к предыдущему или следующему изображению в галерее.

**Клавиатура** — используйте стрелки влево/вправо на клавиатуре для быстрой навигации.

**Свайп** — на мобильных устройствах проведите пальцем влево или вправо для перехода между изображениями.

## Закрытие просмотра

Вернуться в галерею можно несколькими способами:

- Нажмите кнопку **X** в углу экрана
- Нажмите клавишу **Escape** на клавиатуре
- Кликните на затемнённую область вне изображения

## Похожие изображения

Под основным изображением отображается секция с похожими работами. Это помогает найти вдохновение и изучить различные интерпретации похожих тем.

## Масштабирование

Для детального изучения изображения:
- Кликните на изображение для увеличения
- Используйте колёсико мыши для масштабирования
- На мобильных устройствах используйте жест «щипок» для увеличения
