# Создать свою версию

Кнопка "Создать свою версию" позволяет использовать понравившееся изображение как основу для собственной генерации.

## Как это работает

1. Нажмите кнопку "Создать свою версию" на странице просмотра
2. Вы будете перенаправлены в генератор
3. Все параметры оригинального изображения будут автоматически заполнены
4. Вы можете изменить любые параметры по своему усмотрению
5. Запустите генерацию

## Что копируется

- Промпт (текстовый запрос)
- Негативный промпт
- Модель
- Стиль
- Технические параметры (опционально)

## Этика использования

- Уважайте авторские права оригинального создателя
- Используйте функцию для обучения и вдохновения
- Создавайте уникальные вариации, а не точные копии
