# Просмотр галереи

Галерея отображает изображения в виде адаптивной сетки.

## Режимы отображения

- **Сетка** — компактное отображение множества изображений
- **Список** — подробное отображение с информацией

## Навигация

- **Бесконечная прокрутка** — новые изображения загружаются автоматически
- **Пагинация** — переход между страницами (опционально)

## Адаптивность

Галерея автоматически подстраивается под размер экрана:

- **Десктоп** — 4-6 колонок
- **Планшет** — 3-4 колонки
- **Мобильный** — 2 колонки
