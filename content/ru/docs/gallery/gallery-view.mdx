# Просмотр галереи

![Просмотр галереи](/screens/gallery-grid.png)

Галерея ArtGeneration.me отображает изображения в виде адаптивной сетки, оптимизированной для удобного просмотра на любых устройствах.

## Интерфейс галереи

Главная страница галереи состоит из нескольких элементов:

**Панель навигации** — в верхней части расположены категории и фильтры для быстрого доступа к нужным разделам.

**Строка поиска** — позволяет искать изображения по ключевым словам, промптам или авторам.

**Сетка изображений** — основная область с карточками изображений, организованными в адаптивную сетку.

## Карточки изображений

![Карточки с датами](/screens/gallery-thumbnails-with-dates.png)

Каждая карточка отображает:
- Миниатюру изображения
- Дату создания
- Иконку для множественного выбора

## Массовый выбор изображений

![Массовый выбор](/screens/gallery-bulk-select.png)

Вы можете выбрать несколько изображений для групповых операций:

<Steps>
### Активируйте режим выбора
Нажмите на чекбокс в углу карточки изображения.

### Выберите нужные изображения
Отметьте все изображения, с которыми хотите работать.

### Используйте панель действий
Выберите нужное действие в панели сверху.
</Steps>

В панели массового выбора отображается:
- Количество выбранных изображений
- Выпадающий список действий
- Кнопка отмены выбора

### Доступные действия

![Меню действий](/screens/gallery-bulk-delete-menu.png)

<Callout type="warning">
  Удаление изображений необратимо. Убедитесь, что выбрали правильные файлы.
</Callout>

- **Поменять видимость** — сделать изображения публичными или приватными
- **Удалить** — удалить выбранные изображения

## Пагинация

![Пагинация](/screens/gallery-pagination.png)

Навигация по страницам галереи:
- Номера страниц
- Стрелки вперёд/назад
- Индикатор текущей страницы

## Режимы отображения

**Сетка (Grid)** — компактное отображение множества изображений. Оптимально для быстрого просмотра.

**Masonry** — изображения отображаются с сохранением оригинальных пропорций.

## Навигация по галерее

**Бесконечная прокрутка** — при прокрутке вниз автоматически загружаются новые изображения.

**Быстрый возврат наверх** — кнопка в правом нижнем углу для возврата к началу.

## Адаптивность

Галерея автоматически подстраивается под размер экрана:

- **Десктоп (1920px+)** — 5-6 колонок
- **Ноутбук (1200-1920px)** — 4-5 колонок
- **Планшет (768-1200px)** — 3-4 колонки
- **Мобильный (до 768px)** — 2 колонки
