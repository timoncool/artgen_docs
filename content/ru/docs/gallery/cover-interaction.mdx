# Взаимодействие с обложкой

При наведении на обложку изображения в галерее появляются интерактивные элементы.

## Элементы обложки

- **Превью изображения** — уменьшенная версия работы
- **Информация об авторе** — имя пользователя и аватар
- **Счётчик лайков** — количество отметок "нравится"
- **Кнопка избранного** — добавление в избранное

## Действия при наведении

При наведении курсора на обложку:

1. Появляется затемнение с информацией
2. Отображаются кнопки действий
3. Показывается краткое описание (если есть)

## Клик по обложке

Нажатие на обложку открывает полноэкранный просмотр изображения с детальной информацией.
