# Инструменты редактора

![Инструменты редактора](/images/blog/img_024.png)

Редактор ArtGeneration.me предоставляет широкий набор инструментов для работы с изображениями — от классических инструментов рисования до продвинутых AI-функций.

## Инструменты рисования

### Кисть (B)

Основной инструмент для рисования и создания масок. Используется как для художественного рисования, так и для выделения областей при работе с AI-инструментами.

**Параметры кисти:**
- **Размер** — диаметр кисти в пикселях (1-1000)
- **Жёсткость** — чёткость края (0-100%). Мягкая кисть создаёт плавные переходы
- **Прозрачность** — непрозрачность мазка (0-100%)
- **Нажим** — интенсивность цвета при рисовании
- **Цвет** — выберите из палитры или используйте пипетку

**Режимы кисти:**
- Нормальный — стандартное рисование
- Умножение — затемнение
- Экран — осветление
- Перекрытие — контраст

### Ластик (E)

Удаляет нарисованные элементы или создаёт прозрачные области на слоях с альфа-каналом.

**Параметры:**
- Размер и жёсткость аналогичны кисти
- Режим — стирание до прозрачности или до цвета фона

### Заливка (G)

Заполняет область выбранным цветом. Работает по принципу «волшебной палочки» — заливает смежные пиксели похожего цвета.

**Параметры:**
- **Допуск** — насколько похожие цвета будут залиты (0-255)
- **Смежные пиксели** — заливать только соседние области или все подходящие
- **Сглаживание** — плавные края заливки

## Инструменты выделения

### Прямоугольное выделение (M)

Создаёт выделение прямоугольной или квадратной формы.

**Модификаторы:**
- Shift — квадратное выделение
- Alt — выделение из центра
- Shift+Alt — квадрат из центра

### Эллиптическое выделение

Создаёт овальное или круглое выделение.

**Модификаторы:**
- Shift — круглое выделение
- Alt — выделение из центра

### Лассо (L)

Свободное выделение произвольной формы. Рисуйте контур мышью, и область внутри будет выделена.

**Варианты:**
- Обычное лассо — свободное рисование
- Полигональное лассо — выделение прямыми линиями
- Магнитное лассо — автоматическое прилипание к краям объектов

### Волшебная палочка (W)

Автоматическое выделение по цвету. Кликните на область, и все пиксели похожего цвета будут выделены.

**Параметры:**
- **Допуск** — диапазон похожих цветов (0-255)
- **Смежные** — выделять только соседние пиксели
- **Сглаживание** — плавные края выделения

### Операции с выделением

- **Shift + клик** — добавить к выделению
- **Alt + клик** — вычесть из выделения
- **Shift + Alt + клик** — пересечение выделений
- **Ctrl+D** — снять выделение
- **Ctrl+Shift+I** — инвертировать выделение

## Инструменты AI

### Inpainting (перерисовка)

Перерисовывает выделенную область с помощью AI на основе вашего промпта.

**Как использовать:**
1. Выделите область для перерисовки (кистью или инструментами выделения)
2. Введите промпт — опишите, что должно появиться
3. Настройте силу воздействия (денойзинг)
4. Нажмите «Генерировать»

**Параметры:**
- **Промпт** — описание желаемого результата
- **Денойзинг** — сила изменений (0.5-1.0). Выше = больше изменений
- **Маска** — размытие краёв для плавного перехода

### Outpainting (расширение)

Расширяет изображение за его текущие границы, дорисовывая недостающие части.

**Как использовать:**
1. Увеличьте размер холста в нужном направлении
2. Выделите пустую область
3. Введите промпт (опционально) или оставьте пустым для автоматического продолжения
4. Нажмите «Генерировать»

### Удаление объектов

Автоматически удаляет выделенные объекты и заполняет область подходящим фоном.

**Как использовать:**
1. Выделите объект для удаления
2. Нажмите «Удалить объект»
3. AI проанализирует окружение и заполнит область

**Советы:**
- Выделяйте объект с небольшим запасом
- Для сложных фонов может потребоваться несколько попыток
- Используйте inpainting для доработки результата

## Инструменты трансформации

### Перемещение (V)

Перемещает выделенную область или слой.

### Трансформация (Ctrl+T)

Изменяет размер, поворот и перспективу:
- Перетаскивание углов — масштабирование
- Shift — пропорциональное масштабирование
- Перетаскивание за пределами — поворот
- Ctrl + угол — искажение перспективы

## Настройки инструментов

Каждый инструмент имеет контекстную панель настроек справа. Общие параметры:

- **Размер** — размер инструмента
- **Прозрачность** — непрозрачность воздействия
- **Режим наложения** — способ смешивания с существующими пикселями
- **Сглаживание** — плавность краёв
