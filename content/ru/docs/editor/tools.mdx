# Инструменты редактора

Редактор предоставляет набор инструментов для работы с изображениями.

## Инструменты рисования

### Кисть
Основной инструмент для рисования и создания масок.
- Настройка размера
- Настройка жёсткости
- Выбор цвета

### Ластик
Удаление нарисованных элементов или создание прозрачных областей.

### Заливка
Заполнение области выбранным цветом.

## Инструменты выделения

### Прямоугольное выделение
Выделение прямоугольной области.

### Лассо
Свободное выделение произвольной формы.

### Волшебная палочка
Автоматическое выделение по цвету.

## Инструменты AI

### Inpainting
Перерисовка выделенной области с помощью AI.

### Outpainting
Расширение изображения за его границы.

### Удаление объектов
Автоматическое удаление выделенных объектов с заполнением фона.

## Настройки инструментов

Каждый инструмент имеет свои параметры на правой панели:
- Размер
- Прозрачность
- Режим наложения
- Дополнительные опции
