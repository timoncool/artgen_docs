# Интерфейс редактора

Интерфейс редактора разделён на несколько функциональных областей.

## Основные области

### Холст
Центральная область для работы с изображением. Поддерживает:
- Масштабирование (колесо мыши)
- Перемещение (зажатый пробел + перетаскивание)
- Поворот холста

### Панель инструментов
Левая панель с инструментами рисования и выделения.

### Панель настроек
Правая панель с параметрами выбранного инструмента.

### Панель слоёв
Нижняя или правая панель для управления слоями.

## Горячие клавиши

- **Ctrl+Z** — отмена действия
- **Ctrl+Y** — повтор действия
- **B** — кисть
- **E** — ластик
- **M** — выделение
- **Space** — перемещение холста

## Настройка интерфейса

Вы можете настроить расположение панелей и их видимость в меню "Вид".
