# Интерфейс редактора

![Интерфейс редактора](/images/blog/img_022.png)

Интерфейс редактора изображений разработан для удобной и эффективной работы. Он разделён на несколько функциональных областей, каждая из которых отвечает за определённые задачи.

## Структура интерфейса

### Холст (центральная область)

Холст — это основная рабочая область, где отображается редактируемое изображение. Здесь вы выполняете все операции рисования, выделения и редактирования.

**Навигация по холсту:**
- Масштабирование — прокрутка колеса мыши или Ctrl + колесо
- Перемещение — зажмите пробел и перетаскивайте мышью
- Сброс масштаба — двойной клик на инструменте «Рука» или нажмите «0»
- Подогнать под экран — нажмите «F» или используйте меню «Вид»

**Отображение:**
- Сетка — включите для точного позиционирования
- Направляющие — добавьте для выравнивания элементов
- Линейки — отображаются по краям холста

### Панель инструментов (слева)

Вертикальная панель с основными инструментами редактирования:

- Инструменты выделения (прямоугольник, лассо, волшебная палочка)
- Инструменты рисования (кисть, ластик, заливка)
- Инструменты AI (inpainting, outpainting, удаление объектов)
- Инструменты трансформации (перемещение, масштаб, поворот)

Каждый инструмент имеет иконку и горячую клавишу для быстрого доступа.

### Панель настроек (справа)

Контекстная панель, содержимое которой меняется в зависимости от выбранного инструмента:

- Для кисти: размер, жёсткость, прозрачность, цвет
- Для выделения: режим (добавить, вычесть), растушёвка
- Для AI-инструментов: промпт, сила воздействия, параметры модели

### Панель слоёв (справа внизу)

Управление слоями изображения:

- Список всех слоёв с миниатюрами
- Кнопки создания, удаления, группировки
- Настройки прозрачности и режима наложения
- Блокировка и видимость слоёв

### Верхняя панель

Меню и дополнительные функции:

- Файл — открытие, сохранение, экспорт
- Редактирование — отмена, повтор, трансформации
- Вид — масштаб, сетка, направляющие
- Фильтры — эффекты и коррекция

## Горячие клавиши

| Действие | Клавиша |
|----------|---------|
| Отмена | Ctrl+Z |
| Повтор | Ctrl+Y / Ctrl+Shift+Z |
| Кисть | B |
| Ластик | E |
| Выделение | M |
| Лассо | L |
| Волшебная палочка | W |
| Перемещение | V |
| Рука (навигация) | H / Пробел |
| Масштаб | Z |
| Сохранить | Ctrl+S |
| Экспорт | Ctrl+Shift+E |

## Настройка интерфейса

**Изменение расположения панелей:**
- Перетаскивайте панели за заголовок
- Прикрепляйте к разным сторонам экрана
- Сворачивайте неиспользуемые панели

**Настройка отображения:**
- Меню «Вид» → «Настройки интерфейса»
- Выберите тёмную или светлую тему
- Настройте размер элементов интерфейса

**Сохранение рабочего пространства:**
- Настройте интерфейс под свои задачи
- Сохраните как пресет для быстрого переключения
