# Экспорт результата

![Экспорт результата](/images/blog/img_023.png)

После завершения редактирования важно правильно сохранить результат. Выбор формата и параметров экспорта влияет на качество изображения и размер файла.

## Форматы экспорта

### PNG (Portable Network Graphics)

Лучший выбор для изображений с прозрачностью или когда важно сохранить максимальное качество.

**Характеристики:**
- Сжатие без потерь — качество не ухудшается
- Поддержка прозрачности (альфа-канал)
- Поддержка до 16 бит на канал
- Относительно большой размер файла

**Когда использовать:**
- Изображения с прозрачным фоном
- Графика для веб-сайтов (логотипы, иконки)
- Промежуточное сохранение для дальнейшего редактирования
- Когда качество важнее размера файла

### JPEG (Joint Photographic Experts Group)

Оптимальный формат для фотографий и изображений без прозрачности.

**Характеристики:**
- Сжатие с потерями — качество зависит от настроек
- Не поддерживает прозрачность
- Маленький размер файла
- 8 бит на канал

**Когда использовать:**
- Фотографии и реалистичные изображения
- Публикация в социальных сетях
- Отправка по email
- Когда важен размер файла

**Настройка качества:**
- 90-100% — максимальное качество, большой файл
- 70-89% — хорошее качество, умеренный размер
- 50-69% — заметные артефакты, маленький файл
- Ниже 50% — значительная потеря качества

### WebP

Современный формат от Google, сочетающий преимущества PNG и JPEG.

**Характеристики:**
- Сжатие с потерями и без потерь
- Поддержка прозрачности
- Отличное соотношение качество/размер
- Поддерживается всеми современными браузерами

**Когда использовать:**
- Изображения для веб-сайтов
- Когда нужна прозрачность и маленький размер
- Современные проекты без требований к совместимости

## Процесс экспорта

### Стандартный экспорт

1. Откройте меню «Файл» → «Экспорт» или нажмите Ctrl+Shift+E
2. Выберите формат из выпадающего списка
3. Настройте параметры качества
4. Укажите имя файла и папку сохранения
5. Нажмите «Сохранить»

### Быстрое сохранение

- **Ctrl+S** — сохранить в формате PNG с настройками по умолчанию
- **Ctrl+Shift+S** — сохранить как (выбор формата и места)

### Экспорт выделенной области

1. Создайте выделение нужной области
2. Меню «Файл» → «Экспортировать выделение»
3. Будет сохранена только выделенная часть изображения

## Параметры экспорта

### Качество

Для форматов с потерями (JPEG, WebP с потерями):
- Ползунок от 1% до 100%
- Предпросмотр размера файла
- Визуальное сравнение качества

### Масштаб

Изменение размера при экспорте:
- 100% — оригинальный размер
- 50% — уменьшение вдвое
- 200% — увеличение вдвое
- Произвольный размер в пикселях

**Методы масштабирования:**
- Бикубический — лучшее качество для уменьшения
- Билинейный — быстрый, хорошее качество
- Ближайший сосед — для пиксельной графики

### Цветовой профиль

- **sRGB** — стандартный профиль для веба и большинства устройств
- **Adobe RGB** — расширенный цветовой охват для печати
- **Без профиля** — минимальный размер файла

### Метаданные

Выберите, какую информацию включить в файл:
- EXIF — данные о камере (если есть)
- Цветовой профиль
- Информация об авторе

## Сохранение в галерею ArtGeneration

Помимо локального сохранения, вы можете сохранить результат в галерею платформы:

1. Нажмите «Сохранить в галерею» на панели инструментов
2. Добавьте название и описание
3. Выберите категорию
4. Настройте приватность (публичное/приватное)
5. Нажмите «Опубликовать»

**Преимущества:**
- Доступ с любого устройства
- Возможность поделиться ссылкой
- Сохранение параметров генерации
- Резервное копирование работ

## Советы по экспорту

- Сохраняйте рабочий файл со слоями отдельно от финального экспорта
- Для веба используйте WebP или оптимизированный PNG
- Для печати экспортируйте в максимальном качестве
- Проверяйте размер файла перед отправкой
- Используйте понятные имена файлов с датой или версией
