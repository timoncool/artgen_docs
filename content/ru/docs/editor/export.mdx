# Экспорт результата

После завершения редактирования вы можете сохранить результат в различных форматах.

## Форматы экспорта

### PNG
- Поддержка прозрачности
- Без потери качества
- Большой размер файла

### JPEG
- Меньший размер файла
- Настраиваемое качество
- Без прозрачности

### WebP
- Современный формат
- Хорошее сжатие
- Поддержка прозрачности

## Как экспортировать

1. Нажмите "Файл" → "Экспорт" или Ctrl+Shift+E
2. Выберите формат
3. Настройте параметры качества
4. Выберите место сохранения
5. Нажмите "Сохранить"

## Параметры экспорта

- **Качество** — для JPEG и WebP (1-100%)
- **Масштаб** — изменение размера при экспорте
- **Цветовой профиль** — sRGB, Adobe RGB

## Быстрое сохранение

Используйте Ctrl+S для быстрого сохранения в формате PNG.

## Сохранение в галерею

Вы также можете сохранить результат в галерею ArtGeneration для дальнейшего использования.
