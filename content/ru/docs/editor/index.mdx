# Редактор изображений

Редактор изображений позволяет дорабатывать сгенерированные изображения с помощью различных инструментов.

## Возможности редактора

- **Inpainting** — перерисовка выделенных областей
- **Outpainting** — расширение границ изображения
- **Ретушь** — удаление нежелательных элементов
- **Коррекция** — настройка цвета и контраста
- **Слои** — работа с несколькими элементами

## Как открыть редактор

1. Сгенерируйте или выберите изображение
2. Нажмите кнопку "Редактировать"
3. Изображение откроется в редакторе
