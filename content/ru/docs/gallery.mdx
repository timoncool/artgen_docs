# Главная Галерея

Галерея — это раздел, где вы можете просматривать работы других пользователей и находить вдохновение для своих генераций.

## Просмотр и взаимодействие с изображением

При клике на любое изображение в галерее открывается окно просмотра с полной информацией о работе.

### Информация об изображении

В окне просмотра вы увидите:

- **Промпт** — текстовое описание, использованное для генерации
- **Негативный промпт** — что было исключено из генерации
- **Модель** — какая модель использовалась
- **Стиль** — применённый стиль
- **Seed** — зерно генерации для воспроизведения результата
- **Автор** — профиль пользователя, создавшего изображение

### Действия с изображением

import { Callout } from '@/components/mdx'

<Callout type="info">
Нажмите на изображение, чтобы увидеть параметры генерации и использовать их для создания похожих работ.
</Callout>

При просмотре изображения доступны следующие действия:

- **Лайк** — отметьте понравившуюся работу
- **Избранное** — добавьте в коллекцию избранного
- **Создать свою версию** — откройте генератор с настройками этого изображения
- **Скачать** — сохраните изображение на устройство
- **Поделиться** — получите ссылку для отправки друзьям

## Категории и фильтры

### Категории изображений

![Вкладки галереи](/images/blog/img_030.png)

Галерея разделена на тематические категории:

- **Все** — все публичные работы
- **Новое** — недавно созданные изображения
- **Популярное** — работы с наибольшим количеством лайков
- **Избранное** — работы, добавленные в избранное сообществом

### Фильтрация по модели

Вы можете фильтровать работы по использованной модели генерации, чтобы найти изображения в определённом стиле.

### Поиск по тегам

Используйте теги для поиска изображений по конкретным темам или стилям.

## Личная галерея

![Личная галерея пользователя](/images/blog/img_031.png)

После авторизации вам доступна личная галерея со всеми вашими генерациями.

### Организация изображений

В личной галерее вы можете:

- Создавать папки для организации работ
- Перемещать изображения между папками
- Удалять ненужные генерации
- Делать изображения публичными или приватными

### Папки по датам

Система автоматически группирует ваши изображения по датам создания для удобной навигации.

### Массовые операции с изображениями

Выделите несколько изображений для выполнения массовых операций:

- Массовое перемещение в папку
- Массовое удаление
- Массовое скачивание

## Навигация и поиск

### Бесконечная прокрутка

Галерея использует бесконечную прокрутку — новые изображения загружаются автоматически при прокрутке вниз.

### Быстрый переход

Используйте кнопку "Наверх" для быстрого возврата к началу галереи.

## Действия с изображениями

### Активные элементы на изображении

При наведении на изображение в галерее появляются активные элементы:

- **Удаление** — позволяет удалить изображение. Изображения удаляются не сразу, пока вы не обновили страницу (в течение часа) у вас есть возможность восстановить изображение.
- **Лайк** — быстрая отметка понравившейся работы
- **Избранное** — добавление в коллекцию
- **Меню** — дополнительные действия

### Массовое скачивание изображений

<Callout type="info">
Функция массового скачивания позволяет загрузить несколько изображений одним архивом.
</Callout>

#### Как использовать функцию:

1. Выделите нужные изображения, кликая на них с зажатой клавишей Ctrl (или Cmd на Mac)
2. Нажмите кнопку "Скачать выбранные"
3. Дождитесь формирования архива
4. Архив автоматически скачается на ваше устройство

## Социальные функции

### Подписки

Вы можете подписаться на других пользователей, чтобы следить за их новыми работами.

### Комментарии

Оставляйте комментарии под понравившимися работами и общайтесь с авторами.

### Репосты

Делитесь понравившимися работами в социальных сетях прямо из галереи.
