# Галерея

Галерея — это раздел, где вы можете просматривать работы других пользователей и находить вдохновение для своих генераций.

## Типы галерей

### Общая галерея
Показывает все публичные работы пользователей платформы. Здесь вы можете:
- Просматривать новые работы
- Фильтровать по категориям
- Искать по параметрам

### Личная галерея
Ваши сгенерированные изображения сохраняются в личной галерее. Доступна после авторизации.

### Галерея по категориям
Работы сгруппированы по тематическим категориям для удобного поиска.

## Функции галереи

import { Callout } from 'nextra/components'

<Callout type="info">
Нажмите на изображение, чтобы увидеть параметры генерации и использовать их для создания похожих работ.

### Просмотр параметров
При клике на изображение вы увидите:
- Использованный промпт
- Модель генерации
- Стиль
- Seed (зерно генерации)
- Другие параметры

### Копирование настроек
Вы можете скопировать настройки понравившегося изображения и использовать их в генераторе.

## Фильтрация и поиск

### По модели
Фильтруйте работы по использованной модели генерации.

### По цвету
Ищите изображения по преобладающему цвету.

### По seed
Найдите вариации изображений с одинаковым seed.

## Организация работ

### Папки
Создавайте папки для организации своих изображений по проектам или темам.

### Пакетные операции
Выделяйте несколько изображений для массового перемещения или удаления.
