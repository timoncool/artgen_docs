# Главная страница

Главная страница ArtGeneration.me — это ваш стартовый пункт для погружения в мир генерации изображений с помощью искусственного интеллекта. Здесь вы найдёте все необходимые инструменты для начала работы.

## Основные элементы

На главной странице расположены:

- **Навигационное меню** — быстрый доступ ко всем разделам сайта
- **Счётчик генераций** — отображает количество доступных генераций
- **Кнопка "Начать творить"** — переход к генератору изображений
