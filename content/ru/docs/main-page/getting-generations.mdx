# Получение генераций

![Получение генераций](/screens/daily-reward-popup.png)

Генерации — это основная валюта на платформе ArtGeneration.me. Каждая генерация позволяет создать одно изображение с помощью искусственного интеллекта.

## Способы получения генераций

### Бесплатные генерации

- **Ежедневные генерации** — каждый день вы получаете определённое количество бесплатных генераций
- **Приветственный бонус** — новые пользователи получают 100 несгораемых генераций при регистрации

<Callout type="info">
  Приветственный бонус в 100 генераций — отличный способ познакомиться с возможностями сервиса без каких-либо затрат!
</Callout>

### Платные генерации

- **Подписка PRO** — 300 генераций в день
- **Покупка пакетов** — дополнительные генерации можно приобрести отдельно

## Типы генераций

1. **Ежедневные** — обновляются каждый день, не накапливаются
2. **Несгораемые** — сохраняются до использования, не имеют срока действия

<Callout type="warning">
  Ежедневные генерации не накапливаются! Обязательно забирайте их каждый день, чтобы не потерять.
</Callout>
