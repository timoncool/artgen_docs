# Редактор изображений

Редактор позволяет модифицировать сгенерированные изображения без необходимости использования сторонних программ. Это мощный инструмент для финальной обработки ваших генераций.

## Как открыть редактор

Есть несколько способов открыть изображение в редакторе:

1. **Из галереи генератора** — наведите на изображение и нажмите кнопку "В редактор"
2. **Из основной галереи** — откройте изображение и выберите "Редактировать"
3. **Напрямую** — перейдите в раздел "Редактор" в навигации и загрузите изображение

## Основные инструменты

### Обрезка (Crop)

Обрежьте изображение до нужного размера или соотношения сторон. Доступны предустановленные соотношения:
- Свободная обрезка
- 1:1 (квадрат)
- 16:9 (широкоформатное)
- 9:16 (вертикальное)
- 4:3 (классическое)

### Масштабирование

Измените размер изображения с сохранением пропорций. Полезно для подготовки изображений к публикации в социальных сетях.

### Поворот

Поверните изображение на любой угол. Доступны быстрые повороты на 90° и точная настройка угла.

### Отражение

Отразите изображение по горизонтали или вертикали. Часто используется для исправления композиции.

### Доступные настройки

- **Яркость** — осветление или затемнение изображения
- **Контраст** — усиление или ослабление разницы между светлыми и тёмными областями
- **Насыщенность** — интенсивность цветов
- **Экспозиция** — общая освещённость
- **Тени** — детализация в тёмных областях
- **Света** — детализация в светлых областях
- **Температура** — тёплые или холодные тона
- **Оттенок** — сдвиг цветовой гаммы

### Художественные фильтры

- Размытие (Blur)
- Резкость (Sharpen)
- Виньетирование
- Зернистость (Grain)
- Чёрно-белое

## Рисование и маски

### Кисть

Рисуйте поверх изображения различными кистями:
- Настройка размера кисти
- Настройка жёсткости краёв
- Выбор цвета
- Настройка прозрачности

### Маска для инпейнтинга

Создавайте маски для выборочного редактирования:

1. Выберите инструмент "Маска"
2. Закрасьте область, которую хотите изменить
3. Нажмите "Отправить в генератор"
4. Напишите промпт для новой области
5. Запустите генерацию

<Callout type="warning">
Для качественного инпейнтинга рекомендуется закрашивать область с небольшим запасом за пределы объекта, который хотите изменить.
</Callout>

## Текст и надписи

Добавляйте текст на изображения:
- Выбор шрифта
- Настройка размера
- Выбор цвета текста и обводки
- Позиционирование

## Слои

Редактор поддерживает работу со слоями:
- Добавление новых слоев
- Изменение порядка слоев
- Настройка прозрачности слоя
- Режимы наложения

## Сохранение и экспорт

После редактирования вы можете:

- **Сохранить в галерею** — изображение появится в вашей личной галерее
- **Скачать** — сохранить на устройство в формате PNG или JPEG
- **Отправить в генератор** — использовать как основу для новой генерации

### Форматы экспорта

- **PNG** — без потери качества, поддержка прозрачности
- **JPEG** — меньший размер файла, настройка качества

## Интеграция с генератором

Отредактированное изображение можно использовать как Init Image для дальнейшей генерации. Это позволяет создавать сложные композиции итеративно:

1. Сгенерируйте базовое изображение
2. Отредактируйте его в редакторе
3. Отправьте обратно в генератор
4. Добавьте новые элементы через инпейнтинг
5. Повторите процесс до достижения желаемого результата

## Горячие клавиши

- **Ctrl+Z** — отменить последнее действие
- **Ctrl+Y** — повторить отменённое действие
- **Ctrl+S** — сохранить изображение
- **Space + перетаскивание** — перемещение по изображению
- **Колесо мыши** — масштабирование
