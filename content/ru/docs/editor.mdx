# Редактор изображений

Редактор позволяет модифицировать сгенерированные изображения без необходимости использования сторонних программ.

## Основные инструменты

### Обрезка (Crop)
Обрежьте изображение до нужного размера или соотношения сторон.

### Масштабирование
Измените размер изображения с сохранением пропорций.

### Поворот
Поверните изображение на любой угол.

### Отражение
Отразите изображение по горизонтали или вертикали.

## Фильтры и эффекты

import { Callout } from 'nextra/components'

<Callout type="info">
Фильтры применяются неразрушающе — вы всегда можете вернуться к оригиналу.
</Callout>

### Доступные фильтры
- Яркость и контраст
- Насыщенность
- Размытие
- Резкость
- Виньетирование

## Рисование

### Кисть
Рисуйте поверх изображения различными кистями.

### Маска
Создавайте маски для выборочного редактирования или inpainting.

## Сохранение

После редактирования вы можете:
- Сохранить как новое изображение
- Заменить оригинал
- Скачать на устройство

## Интеграция с генератором

Отредактированное изображение можно использовать как Init Image для дальнейшей генерации, что позволяет создавать сложные композиции итеративно.
