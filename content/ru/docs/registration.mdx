# Регистрация на ArtGeneration.me

Для использования всех возможностей платформы необходимо зарегистрироваться.

## Способы регистрации

### Через социальные сети
Самый быстрый способ — авторизация через социальные сети:
- **ВКонтакте** — для пользователей из России
- **Яндекс** — для пользователей из России

### Через email
Вы также можете зарегистрироваться, используя адрес электронной почты.

## Процесс регистрации

import { Steps } from 'nextra/components'

<Steps>
### Шаг 1
Перейдите на [главную страницу](https://artgeneration.me) и нажмите кнопку «Войти»

### Шаг 2
Выберите способ авторизации: социальная сеть или email

### Шаг 3
Подтвердите доступ к вашему аккаунту (для социальных сетей) или введите код подтверждения (для email)

### Шаг 4
Готово! Теперь вы можете пользоваться всеми функциями платформы
</Steps>

## После регистрации

После успешной регистрации вам будут доступны:
- Генерация изображений
- Сохранение работ в личной галерее
- Создание папок для организации изображений
- Доступ к истории генераций
- Возможность приобретения подписки

## Бесплатный тариф

Новые пользователи получают бесплатные генерации для ознакомления с платформой. Подробнее о тарифах читайте в разделе [Оплата и подписки](/docs/payments).
