# Генератор изображений

Генератор — это основной инструмент платформы для создания изображений с помощью искусственного интеллекта.

## Основные параметры

### Промпт (Prompt)
Текстовое описание желаемого изображения. Чем подробнее описание, тем точнее результат.

import { Callout } from 'nextra/components'

<Callout type="warning">
Пишите промпты на английском языке для лучших результатов. Большинство моделей обучены на англоязычных данных.
</Callout>

### Негативный промпт
Описание того, чего НЕ должно быть на изображении. Помогает избежать нежелательных элементов.

### Модель
Выбор модели генерации влияет на стиль и качество результата. Доступны различные модели для разных задач.

### Стиль
Дополнительные стилевые модификаторы, которые применяются к генерации.

## Продвинутые настройки

### Разрешение
Выберите размер выходного изображения. Доступны различные соотношения сторон.

### Режим производительности
- **Набросок** — быстрая генерация, низкое качество
- **Эскиз** — баланс скорости и качества
- **Шедевр** — максимальное качество, дольше генерация

### LoRA
Low-Rank Adaptation модели для тонкой настройки стиля генерации.

### Seed
Зерно генерации. Одинаковый seed с одинаковыми параметрами даст похожий результат.

## Специальные функции

### Init Image (img2img)
Загрузите изображение как основу для генерации. Полезно для:
- Стилизации фотографий
- Создания вариаций
- Дорисовки элементов

### Face Swap
Замена лица на сгенерированном изображении. Требует подписки PRO или MAX.

### Upscale
Увеличение разрешения готового изображения с сохранением качества.

### Background Remove
Удаление фона с изображения.

## Процесс генерации

Введите промпт — описание желаемого изображения

### Шаг 2
Выберите модель и стиль

### Шаг 3
Настройте дополнительные параметры (опционально)

Нажмите кнопку «Создать»

### Шаг 5
Дождитесь завершения генерации (обычно 10-30 секунд)
</Steps>

## Советы по промптам

### Структура хорошего промпта
1. **Основной объект** — что изображено
2. **Детали** — особенности объекта
3. **Окружение** — фон, обстановка
4. **Стиль** — художественное направление
5. **Качество** — технические параметры

### Пример
```
a beautiful woman with long red hair, wearing a white dress, 
standing in a flower garden, soft sunlight, 
oil painting style, highly detailed, 8k resolution
```
