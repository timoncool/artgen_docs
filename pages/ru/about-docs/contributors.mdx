## Как попасть в список авторов

Если вы внесли вклад в документацию (исправили ошибку, добавили новый раздел, улучшили описание), вы можете добавить себя в список авторов. Просто отредактируйте эту страницу и добавьте свою информацию по шаблону ниже.

## Как внести свой вклад

1. **Найдите ошибку или неточность** в документации
2. **Отредактируйте страницу** через GitHub (внизу каждой страницы есть ссылка)
3. **Создайте Pull Request** с вашими изменениями
4. **Добавьте себя** в список авторов на этой странице

Подробная инструкция: [Как редактировать документацию](/ru/contributing)
