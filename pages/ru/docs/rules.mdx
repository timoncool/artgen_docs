# Правила использования

Используя платформу ArtGeneration.me, вы соглашаетесь соблюдать следующие правила.

## Запрещённый контент

import { Callout } from 'nextra/components'

<Callout type="error">
Нарушение правил может привести к блокировке аккаунта без возврата средств.
</Callout>

### Категорически запрещено генерировать:
- Детскую порнографию или сексуализацию несовершеннолетних
- Реалистичные изображения насилия и жестокости
- Контент, разжигающий ненависть по признаку расы, национальности, религии
- Изображения реальных людей в компрометирующих ситуациях без их согласия
- Контент, нарушающий авторские права
- Инструкции по созданию оружия или наркотиков

### Ограниченный контент:
- NSFW контент разрешён только в приватном режиме
- Изображения публичных личностей должны быть явно художественными

## Использование сервиса

### Разрешено:
- Генерация изображений для личного использования
- Коммерческое использование (при наличии платной подписки)
- Публикация работ с указанием, что они созданы с помощью ИИ

### Запрещено:
- Автоматизированный доступ к сервису без разрешения
- Попытки обхода ограничений и защитных механизмов
- Перепродажа доступа к сервису
- Использование для спама или мошенничества

## Авторские права

### Ваши права:
- Вы сохраняете права на созданные вами промпты
- При платной подписке вы получаете права на коммерческое использование изображений

### Права платформы:
- Платформа может использовать публичные изображения для улучшения сервиса
- Приватные изображения не используются без вашего согласия

## Модерация

Все публичные изображения проходят модерацию. Изображения, нарушающие правила, будут удалены, а аккаунт может быть заблокирован.

## Изменение правил

Платформа оставляет за собой право изменять правила. Пользователи будут уведомлены о существенных изменениях.
