# Главная Галерея

![Главная галерея ArtGeneration.me](/screens/gallery-home.png)

Галерея — это сердце сообщества ArtGeneration.me, где собраны лучшие работы пользователей. Здесь вы можете найти вдохновение, изучить техники других художников и открыть для себя новые стили.

## Навигация и категории

![Шапка галереи](/screens/gallery-header-full.png)

В верхней части галереи расположена строка навигации и категории. Вы можете выбрать интересующую категорию из списка:

**Доступные категории:**
- Фэнтезийное искусство
- Аниме
- Персонажи
- Пейзаж
- Киберпанк
- Космос
- Поп-арт
- Картины
- И многие другие

## Фильтры галереи

![Фильтры галереи](/screens/gallery-filters-bar.png)

Вы можете фильтровать изображения по:
- **Свежие** — самые новые работы
- **Интересное** — выборка интересных работ
- **Популярное** — работы с наибольшим количеством лайков
- **Лучшее** — топ-работы

## Фильтр по категориям и моделям

![Фильтр категорий](/screens/category-filter-dialog.png)

В диалоге фильтра доступны вкладки:
- **Категории** — выбор по тематике
- **Модели** — фильтрация по использованной модели

Во вкладке "Модели" можно выбрать конкретную модель генерации для фильтрации результатов.

## Просмотр галереи

![Сетка галереи](/screens/gallery-grid.png)

Галерея отображается в виде сетки изображений. Изображения автоматически подгружаются по мере прокрутки.

![Пагинация](/screens/gallery-pagination.png)

Навигация по страницам галереи с номерами страниц и стрелками.

## Просмотр изображения

![Карточка изображения](/screens/image-view-card.png)

При клике на изображение открывается карточка с деталями: название, описание, счётчик лайков.

![Детали изображения](/screens/image-details-with-prompt.png)

В расширенном просмотре отображается боковая панель с промптом и кнопкой «Создать свою версию».

## Метаданные изображения

![Метаданные](/screens/image-metadata-panel.png)

В панели метаданных отображается:
- Использованная модель
- Разрешение
- Метод генерации
- Использованные стили
- Цветовая палитра
