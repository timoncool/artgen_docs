# Элементы просмотра

![Элементы просмотра](/screens/image-view-card.png)

Страница просмотра изображения содержит множество информативных элементов.

## Верхняя панель

- **Название работы** — если указано автором
- **Дата создания** — когда было сгенерировано изображение
- **Кнопка закрытия** — возврат к галерее

## Боковая панель

- **Аватар автора** — фото профиля
- **Имя автора** — ссылка на профиль
- **Кнопка подписки** — подписаться на автора
- **Статистика** — лайки, просмотры, комментарии

## Нижняя панель

- **Кнопки действий** — лайк, избранное, скачать, поделиться
- **Кнопка "Создать версию"** — использовать как основу для своей генерации

<Callout type="info">
  Кнопка «Создать версию» автоматически копирует все параметры генерации в редактор для создания собственной вариации.
</Callout>
