# Взаимодействие с обложкой

![Взаимодействие с обложкой](/screens/gallery-thumbnails-with-dates.png)

Каждое изображение в галерее представлено в виде карточки с обложкой. При взаимодействии с карточкой доступны различные действия и информация.

## Элементы карточки изображения

**Превью изображения** — уменьшенная версия работы, оптимизированная для быстрой загрузки. Превью сохраняет пропорции оригинала.

**Информация об авторе** — в нижней части карточки отображается аватар и имя пользователя, создавшего изображение.

**Счётчик лайков** — показывает количество пользователей, которым понравилась эта работа.

**Индикатор NSFW** — если изображение содержит контент для взрослых, это отмечается специальным индикатором.

## Действия при наведении

При наведении курсора на карточку изображения появляется оверлей с дополнительными элементами:

**Кнопка лайка** — нажмите, чтобы поставить или убрать лайк. Требуется авторизация.

**Кнопка избранного** — добавьте изображение в свою коллекцию избранного для быстрого доступа в будущем.

**Кнопка «Создать версию»** — быстрый переход к генератору с параметрами этого изображения.

**Краткая информация** — отображается часть промпта или описания изображения.

## Клик по карточке

Нажатие на карточку открывает полноэкранный режим просмотра с детальной информацией:
- Изображение в полном размере
- Полный промпт и параметры генерации
- Профиль автора
- Похожие изображения
- Возможность скачивания

## Контекстное меню

Правый клик на карточке открывает контекстное меню с дополнительными действиями:
- Открыть в новой вкладке
- Скопировать ссылку на изображение
- Пожаловаться на контент

## Мобильная версия

На мобильных устройствах взаимодействие адаптировано:
- Долгое нажатие вызывает меню действий
- Свайп влево/вправо для навигации
- Двойное нажатие для лайка
