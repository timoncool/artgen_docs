# Правила сообщества

Правила взаимодействия между пользователями платформы.

## Уважительное общение

- Будьте вежливы в комментариях
- Не оскорбляйте других пользователей
- Конструктивно критикуйте работы
- Помогайте новичкам

## Запрещённое поведение

### Харассмент
- Преследование других пользователей
- Нежелательные сообщения
- Угрозы и запугивание

### Спам
- Массовая рассылка сообщений
- Реклама без разрешения
- Накрутка лайков и просмотров

### Мошенничество
- Обман других пользователей
- Продажа чужих работ как своих
- Фишинг и кража данных

## Сообщение о нарушениях

Если вы стали свидетелем нарушения правил:
1. Нажмите кнопку "Пожаловаться"
2. Выберите тип нарушения
3. Добавьте комментарий при необходимости

## Последствия нарушений

- Предупреждение
- Временная блокировка
- Постоянная блокировка
- Удаление аккаунта
