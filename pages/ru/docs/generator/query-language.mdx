# Язык запросов

Текстовый запрос (промпт) — это основной способ описать желаемое изображение для AI. От качества запроса напрямую зависит результат генерации.

## Основы написания запросов

### Структура запроса

Хороший запрос обычно содержит:

<Steps>
### Основной объект
Что изображено: человек, пейзаж, предмет.

### Детали
Характеристики объекта: цвет, размер, материал.

### Окружение
Где находится объект: фон, освещение.

### Стиль
Художественное направление: реализм, аниме, живопись.
</Steps>

### Примеры запросов

**Простой запрос:**
```
красивый закат над морем
```

**Детальный запрос:**
```
красивый закат над морем, оранжевое небо, отражение солнца в воде, 
силуэты пальм на переднем плане, фотореалистичный стиль, 8k качество
```

## Язык запросов

<Callout type="info">
Запросы можно писать на русском или английском языке. Английский язык часто даёт более точные результаты, так как большинство моделей обучены на англоязычных данных.
</Callout>

## Негативный промпт

Негативный промпт указывает, чего следует избегать в изображении:

```
размытое, низкое качество, деформированные руки, лишние конечности
```
