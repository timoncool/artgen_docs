# Скачать

![Скачать](/screens/download-menu.png)

Функция скачивания позволяет сохранить изображение на ваше устройство в различных форматах.

## Доступные форматы

При скачивании вы можете выбрать формат:

- **WebP** — современный формат с отличным сжатием, рекомендуется для веба
- **PNG** — высокое качество без потерь, идеально для дальнейшего редактирования
- **JPG** — универсальный формат с хорошим сжатием

![Форматы скачивания](/screens/download-menu2.png)

## Как скачать

1. Наведите на изображение
2. Нажмите кнопку скачивания
3. Выберите нужный формат
4. Файл будет загружен в папку загрузок

## Массовое скачивание

![Скачать мои изображения](/screens/download-my-images-dropdown.png)

В настройках профиля доступна функция массового скачивания изображений:

- **Сегодня** — все изображения за сегодня
- **Вчера** — все изображения за вчера
- **Неделя** — за последнюю неделю
- **Месяц** — за последний месяц
- **3 месяца** / **Полгода** / **Год** — за соответствующий период
- **Всё время** — все ваши изображения

![Выбор периода](/screens/download-period-options.png)

Количество изображений для скачивания отображается рядом с каждым периодом.

## Права на использование

Скачанные изображения можно использовать:
- В личных проектах
- В коммерческих целях (при наличии соответствующей подписки)
- Для публикации с указанием источника

## Советы

- Скачивайте изображения после применения улучшений (upscale, восстановление)
- Сохраняйте понравившиеся работы локально
- Используйте функцию массового скачивания для резервного копирования
- Выбирайте PNG для редактирования, WebP для публикации в интернете
