# Восстановление + Увеличение

![Восстановление + Увеличение](/screens/image-actions-toolbar.png)

Комбинированная функция, объединяющая восстановление лица и увеличение разрешения.

## Преимущества

Выполняет две операции за один раз:
1. Улучшает качество лиц
2. Увеличивает разрешение в 2 раза

## Когда использовать

- Портретные фотографии для печати
- Изображения с людьми в высоком разрешении
- Когда нужно и улучшить лица, и увеличить размер

## Порядок обработки

1. Сначала восстанавливаются лица
2. Затем изображение увеличивается

Такой порядок обеспечивает лучшее качество, чем обратная последовательность.

## Стоимость

Расходует больше генераций, чем каждая операция по отдельности, но меньше, чем обе операции последовательно.

## Результат

Вы получаете изображение:
- С улучшенными лицами
- В увеличенном разрешении
- С сохранением общего качества
