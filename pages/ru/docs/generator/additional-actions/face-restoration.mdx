# Восстановление лица

![Восстановление лица](/screens/face-swap-panel.png)

Функция восстановления лица улучшает качество лиц на сгенерированных изображениях.

## Проблема

AI-генерация иногда создаёт лица с:
- Размытыми чертами
- Асимметрией
- Неестественными деталями
- Артефактами

## Решение

Восстановление лица использует специализированные алгоритмы для:
- Улучшения чёткости черт лица
- Исправления асимметрии
- Добавления реалистичных деталей
- Устранения артефактов

## Как использовать

1. Наведите на изображение с лицом
2. Нажмите кнопку "Восстановление лица"
3. Дождитесь обработки
4. Получите улучшенный результат

## Когда применять

- Портретные изображения
- Групповые фото
- Любые изображения с людьми
- Когда лица выглядят неестественно

## Ограничения

- Работает лучше на крупных лицах
- Может изменить некоторые черты
- Не всегда улучшает стилизованные изображения (аниме, арт)
