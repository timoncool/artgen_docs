# Увеличение 2x (Upscale)

![Upscale 2x](/screens/download-menu.png)

Функция увеличения позволяет повысить разрешение изображения в 2 раза с сохранением качества.

## Как работает

Upscale использует специальные AI-алгоритмы для:
- Увеличения размера изображения
- Добавления деталей
- Сохранения чёткости
- Устранения артефактов масштабирования

## Результат

Изображение 512x512 станет 1024x1024 с улучшенной детализацией.

## Когда использовать

- Для печати изображений
- Для использования в качестве обоев
- Для публикации в высоком разрешении
- Когда нужна максимальная детализация

## Стоимость

Upscale расходует генерации из вашего баланса. Количество зависит от размера исходного изображения.

## Ограничения

- Максимальный размер после увеличения ограничен
- Очень маленькие изображения могут терять качество
- Некоторые артефакты могут усиливаться при увеличении
