# Копировать ссылку

![Копировать ссылку](/images/blog/img_013.png)

Функция копирования ссылки позволяет получить прямой URL на изображение.

## Как использовать

1. Наведите на изображение
2. Нажмите кнопку "Копировать ссылку"
3. Ссылка скопирована в буфер обмена

## Применение

- Поделиться изображением в социальных сетях
- Отправить ссылку в мессенджере
- Вставить в документ или презентацию
- Сохранить для последующего доступа

## Формат ссылки

Ссылка ведёт на страницу просмотра изображения на ArtGeneration.me, где доступны:
- Полноразмерное изображение
- Информация о параметрах генерации
- Возможность создать свою версию

## Приватность

Если изображение создано в приватном режиме, ссылка будет доступна только вам.
