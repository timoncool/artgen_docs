# Модели

![Выбор модели](/screens/model-selection-overlay.png)

Модель (она же чекпоинт) — самая важная часть системы генерации изображений. Существует несколько типов моделей:

- **SDXL** — высококачественная архитектура для создания детализированных изображений
- **Flux** — новейшая архитектура с глубоким пониманием промптов и улучшенным следованием инструкциям
- **RunWare / GE / Custom** — специализированные модели от различных провайдеров
- **Редакционные модели** — модели для inpainting, outpainting, face-swap и стилизации

На основе базовых моделей сообщество по всему миру обучает свои собственные *узкоспециализированные модели* для работы в разных стилях.

Модель — это что-то вроде операционной системы. Если сам ArtGeneration.me — это компьютер, системный блок, то модель — это его операционная система. Всё, что вы сможете сгенерировать, зависит от модели.

Разные модели ориентированы на разный результат:
- Генерировать аниме лучше на аниме-модели
- Фотореалистичные изображения — на фото-модели

## Доступные разрешения

Большинство современных моделей поддерживают высокие разрешения:
- **Стандартные SDXL модели**: от 1024×1024 до 1536×1536
- **Модели с поддержкой 4K**: до 1536×1536 и выше (например, 1472×704 для широкоформатных)
- Для ещё более высокого качества используйте функцию **upscale x2/x4**

## Категории моделей

![Фильтры моделей](/screens/model-filter-tags.png)

На моделях вы заметите различные метки, они могут многое рассказать:

- **HIT** — Самые популярные модели получают такой значок
- **NEW** — Новые модели помечены так
- **GE** — Наш собственный сервис-провайдер. Модели с этой меткой показывают лучшее быстродействие, качество и дополнительные функции
- **RUNWARE** — Высокопроизводительный провайдер с быстрой генерацией
- **POLLI** — Провайдер с дополнительными функциями
- **YANDEX ART** — Модели от Яндекса
- **SDXL** — Базовая архитектура модели
- **FLUX** — Самая новая базовая архитектура с улучшенным пониманием промптов
- **ART** — На такой модели будет хорошо получаться генерировать различные иллюстрации
- **PHOTO** — Такая модель лучше подойдёт для стиля фотореализма
- **ANIME** — Лучший выбор для создания иллюстраций в стиле японских мультфильмов
- **NSFW** — Модель, ориентированная на создание контента для взрослых без цензуры
- **3D** — Модель для создания персонажей в стиле 3D, как Pixar

## Индикаторы моделей

Так как моделей много и сервис-провайдеров, где они запускаются, всего несколько, некоторые модели могут быть загружены больше, чем другие. Поэтому на моделях есть три индикатора:

- **Зелёный** — Модель работает стабильно, быстро и без ошибок
- **Жёлтый** — Модель нагружена, время генерации может быть дольше, чем обычно
- **Красный** — Модель перегружена, время генерации высокое

Красный индикатор — это не приговор. Вы можете попробовать использовать модель, и возможно всё пройдёт хорошо. Это лишь статистическая информация для ориентира.

## Специальные модели для редактирования

Помимо моделей для генерации изображений, доступны специализированные модели:

- **Inpainting** — для редактирования отдельных областей изображения
- **Outpainting** — для расширения изображения за его границы
- **Face Swap** — для замены лиц на изображениях
- **Стилизация** — для применения художественных стилей к фотографиям
