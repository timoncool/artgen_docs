# Параметры генерации

![Параметры генерации](/screens/generator-right-panel.png)

Технические параметры позволяют тонко настроить процесс генерации изображений. Понимание этих параметров поможет вам получать более качественные и предсказуемые результаты.

## Шум (Seed)

![Шум](/screens/seed-setting.png)

Шум (Seed) — это начальное значение для генератора случайных чисел. Это число определяет «случайность» генерации.

**Как работает:**
- При одинаковом значении шума и всех остальных параметрах вы получите идентичное изображение
- Случайный шум даёт уникальный результат каждый раз
- Вы можете скопировать значение шума понравившейся генерации и использовать его для создания вариаций
- Нажмите на иконку кубика для генерации случайного значения

**Применение:**
- Нашли удачную композицию? Зафиксируйте шум и экспериментируйте с промптом
- Хотите серию похожих изображений? Используйте одно значение шума с небольшими изменениями промпта
- Нужна воспроизводимость? Сохраните значение шума вместе с остальными параметрами

## Количество изображений

![Количество изображений](/screens/num-images-selection.png)

Вы можете генерировать от 1 до 4 изображений за один запрос. Это удобно для быстрого сравнения вариантов. Стоимость генерации указана рядом с каждой кнопкой.

## Приватный режим

![Приватный режим](/screens/private-mode-toggle.png)

Включите «Приватный режим», чтобы ваши изображения не отображались в публичной галерее и были видны только вам.

## CFG Scale (Guidance Scale)

CFG Scale определяет, насколько строго модель следует вашему промпту. Это один из самых важных параметров для контроля результата.

**Значения:**
- **1-4** — Модель имеет большую творческую свободу, может значительно отклоняться от промпта
- **5-7** — Баланс между следованием промпту и творческой свободой. Рекомендуется для большинства задач
- **8-12** — Модель строго следует промпту
- **13-20** — Очень строгое следование. Может привести к артефактам

**Рекомендация:** Начните с **7** и корректируйте по результатам.

## Шаги генерации (Steps)

Количество шагов определяет, сколько итераций пройдёт модель при создании изображения. Больше шагов — больше деталей, но и больше времени.

**Значения:**
- **15-25** — Быстрая генерация, базовое качество
- **25-35** — Оптимальный баланс качества и скорости
- **35-50** — Высокое качество для финальных работ

**Важно:** После определённого количества шагов качество перестаёт улучшаться. Для большинства моделей это 30-40 шагов.

## Сэмплер (Sampler)

Сэмплер — это алгоритм, который определяет, как модель «движется» от шума к финальному изображению.

**Популярные сэмплеры:**
- **Euler** — Быстрый и стабильный
- **Euler a** — Более «живые» результаты
- **DPM++ 2M** — Отличная детализация
- **DPM++ 2M Karras** — Улучшенная версия с плавными переходами

**Совет:** Если не знаете, какой выбрать — используйте **DPM++ 2M Karras** или **Euler a**.

## Взаимосвязь параметров

Параметры влияют друг на друга:
- Высокий CFG Scale может потребовать больше шагов
- Некоторые сэмплеры работают лучше с определённым количеством шагов
- Разные модели имеют свои оптимальные настройки
