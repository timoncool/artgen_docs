# История генераций

![История генераций](/screens/history-panel.png)

История генераций — это полный архив всех ваших созданных изображений. Каждая генерация автоматически сохраняется, позволяя вам вернуться к любой работе в любой момент.

## Интерфейс истории

![Полная история](/screens/history-panel-full.png)

История отображается в виде сетки миниатюр, отсортированных по дате создания (новые сверху). Каждая миниатюра показывает превью изображения.

**Бесконечная прокрутка** — при прокрутке вниз автоматически загружаются более старые генерации.

**Регулировка размера** — используйте ползунок для изменения размера миниатюр.

**Поиск** — введите ключевые слова для поиска по промптам.

## Массовый выбор

![Массовый выбор](/screens/history-multiselect.png)

Вы можете выбрать несколько изображений для групповых операций:

- Установите флажки на нужных изображениях
- Используйте иконки действий в верхней части панели
- Доступно: удаление, открытие, закрытие выбора

## Действия с изображениями

![Панель действий](/screens/image-actions-toolbar.png)

При наведении на изображение доступны быстрые действия:

- **Вариации** — создать похожие изображения
- **Увеличение** — применить upscale
- **Кадрирование** — обрезать изображение
- **Копирование ссылки** — получить ссылку на изображение
- **Скачивание** — сохранить на устройство
- **Отправка в редактор** — открыть в редакторе

## Предпросмотр

![Предпросмотр](/screens/generator-preview-overlay.png)

Нажмите на изображение, чтобы открыть его в режиме предпросмотра с увеличением.

![Просмотр нескольких](/screens/multiple-preview-panel.png)

Можно просматривать несколько изображений одновременно в рабочей области.

## Хранение и синхронизация

**Облачное хранение** — вся история хранится на серверах ArtGeneration.me и доступна с любого устройства.

**Автоматическое сохранение** — каждая генерация сохраняется автоматически.

## Советы по работе с историей

- Регулярно добавляйте лучшие работы в избранное
- Используйте папки для организации по проектам
- Копируйте параметры успешных генераций для создания вариаций
- Используйте поиск для быстрого нахождения нужных работ
