# Левая панель

Левая панель генератора — это ваш центр управления созданными изображениями. Здесь вы найдёте историю всех генераций, избранные работы и инструменты для организации контента.

## Расположение и интерфейс

Левая панель расположена слева от основной рабочей области генератора. Она занимает примерно 20-25% ширины экрана на десктопе и может быть свёрнута для увеличения рабочего пространства.

**Вкладки панели** — в верхней части панели расположены вкладки для переключения между разделами: История, Избранное, Папки.

**Область контента** — основная часть панели отображает содержимое выбранной вкладки в виде сетки миниатюр.

**Панель действий** — внизу расположены кнопки быстрых действий и фильтры.

## Основные разделы

**История генераций** — полный архив всех ваших генераций, отсортированный по дате. Здесь вы можете найти любое ранее созданное изображение и использовать его параметры для новых работ.

**Избранное** — коллекция изображений, которые вы отметили как понравившиеся. Удобно для хранения лучших работ и референсов.

**Папки** — система организации изображений по проектам, темам или любым другим критериям. Помогает поддерживать порядок при большом количестве генераций.

## Сворачивание панели

Для увеличения рабочей области вы можете свернуть левую панель:

<Steps>
### Нажмите на стрелку

Кликните на стрелку в верхней части панели.

### Панель свернётся

Панель свернётся до узкой полоски с иконками.

### Разверните при необходимости

Для разворачивания нажмите на свёрнутую панель или используйте ту же горячую клавишу.
</Steps>

<Callout type="info">
  Используйте горячую клавишу `[` для быстрого сворачивания и разворачивания панели.
</Callout>

## Работа с изображениями

При наведении на миниатюру в любом разделе появляются быстрые действия:

- **Просмотр** — открыть изображение в полном размере
- **Копировать параметры** — скопировать все настройки генерации
- **Добавить в избранное** — отметить изображение
- **Переместить в папку** — организовать изображение
- **Удалить** — удалить изображение из истории

## Поиск и фильтрация

В каждом разделе доступны инструменты поиска:

- **Поиск по промпту** — найдите изображения по ключевым словам
- **Фильтр по дате** — отобразите генерации за определённый период
- **Фильтр по модели** — покажите работы, созданные определённой моделью
