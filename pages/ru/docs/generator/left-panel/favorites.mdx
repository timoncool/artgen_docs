# Избранное

![Избранное](/screens/history-panel-full.png)

Раздел «Избранное» — это ваша персональная коллекция лучших работ. Здесь вы храните изображения, к которым хотите иметь быстрый доступ: удачные генерации, референсы для будущих проектов, вдохновляющие работы.

## Зачем использовать избранное

**Быстрый доступ** — не нужно искать удачные генерации в длинной истории. Все лучшие работы собраны в одном месте.

**Референсы** — сохраняйте изображения, стиль или композицию которых хотите использовать в будущем.

**Портфолио** — формируйте коллекцию своих лучших работ для демонстрации.

**Изучение** — сохраняйте работы других авторов из галереи для анализа их техник.

## Добавление в избранное

**Из истории генераций:**
1. Наведите курсор на изображение в истории
2. Нажмите на иконку сердечка
3. Изображение мгновенно добавляется в избранное

**Из просмотра изображения:**
1. Откройте изображение в полноразмерном режиме
2. Нажмите кнопку «Добавить в избранное» на панели действий

**Из галереи:**
1. Найдите понравившееся изображение в публичной галерее
2. Нажмите на сердечко для добавления в избранное

## Просмотр избранного

Перейдите на вкладку «Избранное» на левой панели генератора. Изображения отображаются в виде сетки миниатюр, отсортированных по дате добавления.

## Работа с избранным

**Поиск** — используйте поле поиска для нахождения изображений по ключевым словам из промпта.

**Сортировка** — измените порядок отображения: по дате добавления, по дате создания, по модели.

**Фильтрация** — отфильтруйте избранное по модели, стилю или другим параметрам.

## Действия с изображениями

При работе с изображениями в избранном доступны все стандартные действия:

- Просмотр в полном размере
- Копирование параметров генерации
- Перемещение в папку
- Скачивание изображения
- Удаление из избранного

## Удаление из избранного

Чтобы удалить изображение из избранного:
- Нажмите на сердечко повторно (оно станет пустым)
- Или используйте контекстное меню и выберите «Удалить из избранного»

Удаление из избранного не удаляет само изображение — оно остаётся в истории генераций.

## Синхронизация

Избранное автоматически синхронизируется между всеми вашими устройствами. Добавьте изображение в избранное на компьютере — оно появится в избранном на телефоне и планшете.

## Лимиты

Количество изображений в избранном может быть ограничено в зависимости от вашего тарифного плана. Проверьте текущие лимиты в настройках аккаунта.

## Советы

- Добавляйте в избранное сразу после удачной генерации
- Используйте избранное как источник референсов для новых работ
- Регулярно просматривайте избранное для вдохновения
- Организуйте лучшие работы из избранного в тематические папки
