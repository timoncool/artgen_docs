# Часто задаваемые вопросы

Ответы на самые популярные вопросы о работе с ArtGeneration.

## Что такое ArtGeneration?

ArtGeneration — это онлайн-сервис для создания изображений с помощью искусственного интеллекта. Вы описываете желаемое изображение текстом, а нейросеть создаёт его для вас.

## Как начать работу?

1. Зарегистрируйтесь на сайте
2. Получите приветственные бесплатные генерации
3. Введите описание желаемого изображения
4. Нажмите "Создать"

## Сколько это стоит?

- Базовые функции доступны бесплатно
- Ежедневно начисляются бесплатные генерации
- Для расширенных возможностей доступны подписки и пакеты

## Какие форматы поддерживаются?

Изображения создаются в формате PNG с различными разрешениями от 512x512 до 1024x1024 и выше.
