If you've contributed to the documentation (fixed an error, added a new section, improved descriptions), you can add yourself to the contributors list. Simply edit this page and add your information using the template below.

## How to Contribute

1. **Find an error or inaccuracy** in the documentation
2. **Edit the page** via GitHub (there's a link at the bottom of each page)
3. **Create a Pull Request** with your changes
4. **Add yourself** to the contributors list on this page

Detailed guide: [How to Edit Documentation](/en/about-docs/contributing)
