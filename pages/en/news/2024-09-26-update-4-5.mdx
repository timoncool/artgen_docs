# Update 4.5

**Date:** September 26, 2024

## 1. Design Update

Updated design of all site sections. Top bar is now transparent and slightly rounded, providing cleaner and more responsive interface.

## 2. Column Settings

Platform now allows users to customize column count in generator, useful for efficiently managing multiple images.

## 3. Layer System

Styles and "Own Image" are now organized as layers that can be collapsed, expanded or temporarily disabled via layer icons.

## 4. Performance

Fixed bugs and implemented speed improvements.
