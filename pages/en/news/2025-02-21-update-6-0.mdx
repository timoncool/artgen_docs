# Update 6.0

**Date:** February 21, 2025

![Bulk operations](/blog/update-6.0-1.png)

## 1. Bulk Image Operations

Bulk file operations available in personal cabinet. Select them with checkboxes and delete using convenient popup panel. Additional file operations planned for future releases.

## 2. Date Folders

Now all your images are available for viewing in folders by generation dates. You can view or delete entire folders at once. Pagination and bulk actions available.

## 3. Bulk Image Download

New option in settings allows requesting image archives for specific periods. Processing time depends on image count. Download links are sent to email and remain active for 72 hours. Only one active archive at a time.

## 4. Dedicated Image Editor

Image editor got a separate page at artgeneration.me/editor. Allows quick image editing — upload from device or select previously generated, then download or save to gallery.

![Style combining](/blog/update-6.0-2.png)

## 5. Expanded Style Library

We've collected probably all unique styles created by humanity — 307 total. Each style includes 5 preview variations for accurate visualization. Named categories and better organization coming soon.

## 6. Combine Up to Three Styles

Users can combine up to three styles simultaneously for creative experiments. Warning: excessive combinations may produce unpredictable results.

## 7. New AI Models

- YiffyMix v52 XL
- Pony Diffusion V6 XL
- Pony Realism v2.2
- Interior Scene XL
- Big Lust XL v1.6
- STOIQO Afrodite FLUX
- Flux Mystic

## 8. Outdated Model Removal

Technology has long been considered outdated — version 1.5 models were removed. XL equivalents now replace old model options.

## 9. Interface Improvements

Numerous small bug fixes and performance improvements across the platform.
