# Update 7.0

**Date:** July 7, 2025

![International version](/blog/update-7.0-1.png)

## 1. International Website Version

New platform version launched at [artgeneration.co](https://artgeneration.co/) for users outside Russia. Provides more reliable communication and international card payment option.

These are separate servers with different databases and legal entities, account and history transfer is not possible.

![Localization](/blog/update-7.0-2.png)

## 2. Interface Localization

Language selection option added regardless of server geography. Current support: Russian and English with expansion plans.

![Shapes menu](/blog/update-7.0-3.png)

## 3. New "Shapes" Menu in Editor

Complex shape icons for adding to works. Features: color change, size change, shadow addition. Function supports use as design element and as base for generation via "Own Image" option.

## 4. Technical Improvements

Many small bug fixes and performance improvements.
