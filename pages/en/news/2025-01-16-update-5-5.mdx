# Update 5.5

**Date:** January 16, 2025

![Creative modes](/blog/update-5.5-1.png)

## 1. Creative Modes

Feature allows users to choose mode based on priorities:

- **Sketch Mode** — image will be generated several times faster, but with lower quality
- **Masterpiece Mode** — maximum quality, but generation takes more time

![Image editor](/blog/update-5.5-2.png)

## 2. Image Editor

Built-in editor added for post-processing. Functionality includes:

- Drawing
- Adding text
- Color adjustment
- Applying filters
- Creating sketches using geometric shapes for subsequent generation

## 3. Bug Fixes

Fixed many old bugs and created a few new ones. Report discovered issues in our [Telegram community](https://t.me/ArtGenerationMe).
