# Search and Categories

![Search and Categories](/images/blog/img_002.png)

The ArtGeneration.me gallery provides powerful tools for searching and filtering images. You can easily find inspiration, explore other users' work, or find specific styles and techniques.

## Search Bar

At the top of the gallery is a search bar that allows you to search for images by various criteria:

**By keywords** - enter a description of what you want to find. Search works on prompts, tags, and image descriptions.

**By author name** - enter a username to find all their works in the gallery.

**By model** - specify the model name to see works created with it.

## Gallery Categories

The gallery is organized by categories for easy navigation:

- **All** - all public images without filtering
- **New** - recently uploaded works
- **Popular** - images with the most likes
- **Favorites** - works you've added to favorites (requires authorization)

## Sorting

Within each category, sorting options are available:

- **By date** - newest first or oldest first
- **By popularity** - by number of likes
- **By views** - by number of views

## Filters

Additional filters help narrow down search results:

**Time period:**
- Today
- This week
- This month
- All time

**By generation model:**
- SDXL models
- SD 1.5 models
- FLUX models
- Specific models by name

**By style:**
- Photorealism
- Anime
- Illustration
- And other styles

## Search Tips

- Use English terms for more accurate search
- Combine multiple filters for precise results
- Save works you like to favorites for quick access
- Study prompts of found images for inspiration
