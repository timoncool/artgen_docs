# Поиск и категории

![Search and Categories](/screens/category-filter-dialog.png)

Галерея ArtGeneration.me предоставляет мощные инструменты для поиска и фильтрации изображений. Вы можете легко найти вдохновение, изучить работы других пользователей или найти конкретные стили и техники.

## Строка поиска

В верхней части галереи расположена строка поиска, которая позволяет искать изображения по различным критериям:

**По ключевым словам** — введите описание того, что хотите найти. Поиск работает по промптам, тегам и описаниям изображений.

**По имени автора** — введите имя пользователя, чтобы найти все его работы в галерее.

**По модели** — укажите название модели, чтобы увидеть работы, созданные с её помощью.

## Категории галереи

Галерея организована по категориям для удобной навигации:

- **Все** — все публичные изображения без фильтрации
- **Новые** — последние загруженные работы
- **Популярные** — изображения с наибольшим количеством лайков
- **Избранное** — работы, добавленные вами в избранное (требуется авторизация)

## Сортировка

Внутри каждой категории доступны варианты сортировки:

- **По дате** — сначала новые или сначала старые
- **По популярности** — по количеству лайков
- **По просмотрам** — по количеству просмотров

## Фильтры

Дополнительные фильтры помогут сузить результаты поиска:

**Период времени:**
- За сегодня
- За неделю
- За месяц
- За всё время

**По модели генерации:**
- SDXL модели
- SD 1.5 модели
- FLUX модели
- Конкретные модели по названию

**По стилю:**
- Фотореализм
- Аниме
- Иллюстрация
- И другие стили

## Советы по поиску

- Используйте английские термины для более точного поиска
- Комбинируйте несколько фильтров для точного результата
- Сохраняйте понравившиеся работы в избранное для быстрого доступа
- Изучайте промпты найденных изображений для вдохновения
