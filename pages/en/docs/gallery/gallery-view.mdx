# Просмотр галереи

![Gallery View](/screens/gallery-grid.png)

Галерея ArtGeneration.me отображает изображения в виде адаптивной сетки, оптимизированной для удобного просмотра на любых устройствах.

## Интерфейс галереи

Главная страница галереи состоит из нескольких элементов:

**Панель навигации** — в верхней части расположены категории и фильтры для быстрого доступа к нужным разделам.

**Строка поиска** — позволяет искать изображения по ключевым словам, промптам или авторам.

**Сетка изображений** — основная область с карточками изображений, организованными в адаптивную сетку.

**Боковая панель** — на десктопе может отображаться панель с дополнительными фильтрами и категориями.

## Режимы отображения

**Сетка (Grid)** — компактное отображение множества изображений. Оптимально для быстрого просмотра и поиска вдохновения.

**Masonry** — изображения отображаются с сохранением оригинальных пропорций, создавая динамичную раскладку.

## Навигация по галерее

**Бесконечная прокрутка** — при прокрутке вниз автоматически загружаются новые изображения. Это позволяет непрерывно просматривать контент без необходимости переключения страниц.

**Быстрый возврат наверх** — кнопка в правом нижнем углу позволяет мгновенно вернуться к началу галереи.

**Клавиатурная навигация** — используйте стрелки для перемещения между изображениями в режиме просмотра.

## Адаптивность

Галерея автоматически подстраивается под размер экрана:

**Десктоп (1920px+)** — 5-6 колонок, максимальное количество изображений на экране.

**Ноутбук (1200-1920px)** — 4-5 колонок, комфортный просмотр на средних экранах.

**Планшет (768-1200px)** — 3-4 колонки, оптимизировано для сенсорного управления.

**Мобильный (до 768px)** — 2 колонки, удобный просмотр одной рукой.

## Загрузка изображений

Изображения загружаются постепенно с использованием ленивой загрузки (lazy loading). Это обеспечивает быструю начальную загрузку страницы и экономит трафик.

Превью изображений оптимизированы для быстрой загрузки, а полноразмерные версии загружаются только при открытии детального просмотра.

## Кэширование

Просмотренные изображения кэшируются в браузере, что ускоряет повторный просмотр и снижает нагрузку на сеть.
