# Cover Interaction

![Cover Interaction](/images/blog/img_025.png)

Each image in the gallery is presented as a card with a cover. Various actions and information are available when interacting with the card.

## Image Card Elements

**Image preview** - a reduced version of the work, optimized for fast loading. The preview maintains the original proportions.

**Author information** - at the bottom of the card, the avatar and name of the user who created the image are displayed.

**Like counter** - shows the number of users who liked this work.

**NSFW indicator** - if the image contains adult content, it is marked with a special indicator.

## Hover Actions

When hovering over an image card, an overlay with additional elements appears:

**Like button** - click to add or remove a like. Authorization required.

**Favorites button** - add the image to your favorites collection for quick access in the future.

**"Create version" button** - quick transition to the generator with this image's parameters.

**Brief information** - part of the prompt or image description is displayed.

## Clicking on a Card

Clicking on a card opens full-screen viewing mode with detailed information:
- Full-size image
- Complete prompt and generation parameters
- Author profile
- Similar images
- Download option

## Context Menu

Right-clicking on a card opens a context menu with additional actions:
- Open in new tab
- Copy image link
- Report content

## Mobile Version

On mobile devices, interaction is adapted:
- Long press brings up the action menu
- Swipe left/right for navigation
- Double tap to like
