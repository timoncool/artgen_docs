# Базовый синтаксис весов

Веса задаются с помощью скобок и числовых значений.

## Синтаксис

### Усиление элемента

Используйте круглые скобки с числом больше 1:

```
(красный:1.5) цветок
```

Это усилит влияние слова "красный" в 1.5 раза.

### Ослабление элемента

Используйте число меньше 1:

```
(фон:0.5) с деревьями
```

Это ослабит влияние фона в 2 раза.

## Диапазон значений

- **1.0** — стандартный вес (по умолчанию)
- **1.1 - 1.5** — умеренное усиление
- **1.5 - 2.0** — сильное усиление
- **0.5 - 0.9** — ослабление
- **Выше 2.0** — может привести к артефактам

## Альтернативный синтаксис

Некоторые модели поддерживают упрощённый синтаксис:

```
((важный элемент)) — двойные скобки = вес 1.1
(((очень важный))) — тройные скобки = вес 1.21
```
