# Prompt Writing

A prompt is a text description of the image you want to create.

## Prompt Basics

A good prompt contains:
- Main subject
- Environment or background
- Style and mood
- Details and attributes

## Prompt Structure

Start with the main subject, then add details:

```
A beautiful sunset over the ocean, golden light, dramatic clouds, photorealistic style
```

## Language

You can write prompts in English or Russian. English often gives better results as models are primarily trained on English data.

## Prompt Length

- Minimum: 3-5 words
- Optimal: 20-50 words
- Maximum: up to 500 characters

## Special Characters

Use commas to separate concepts and parentheses for grouping.
