# Режим производительности

![Performance Mode](/screens/creative-mode-options.png)

Режим производительности — это удобный способ быстро настроить баланс между скоростью генерации и качеством результата. Вместо ручной настройки множества параметров вы можете выбрать один из предустановленных режимов.

## Доступные режимы

### Набросок (Draft)

Самый быстрый режим для быстрого тестирования идей.

**Характеристики:**
- Минимальное количество шагов генерации
- Быстрое время выполнения
- Базовое качество изображения
- Минимальный расход генераций

**Когда использовать:**
- Поиск удачной композиции
- Тестирование различных промптов
- Проверка работы LoRA или стилей
- Быстрый набросок идеи

### Эскиз (Sketch)

Сбалансированный режим для повседневной работы.

**Характеристики:**
- Среднее количество шагов
- Хорошее качество изображения
- Умеренное время генерации
- Оптимальный расход генераций

**Когда использовать:**
- Большинство повседневных задач
- Создание контента для социальных сетей
- Работа над серией изображений
- Когда нужен баланс качества и скорости

### Шедевр (Masterpiece)

Максимальное качество для финальных работ.

**Характеристики:**
- Максимальное количество шагов
- Наивысшее качество изображения
- Увеличенное время генерации
- Повышенный расход генераций

**Когда использовать:**
- Финальные версии работ
- Изображения для печати
- Важные проекты
- Когда качество важнее скорости

## Как режим влияет на параметры

При выборе режима автоматически настраиваются:

- **Количество шагов (Steps)** — от минимального в «Наброске» до максимального в «Шедевре»
- **CFG Scale** — оптимизируется под выбранный режим
- **Сэмплер** — выбирается наиболее подходящий алгоритм
- **Другие технические параметры** — настраиваются для достижения лучшего результата

## Рекомендуемый рабочий процесс

1. **Начните с «Наброска»** — быстро протестируйте несколько вариантов промпта
2. **Найдите удачную идею** — определите лучшую композицию и стиль
3. **Переключитесь на «Эскиз»** — уточните детали и параметры
4. **Финализируйте в «Шедевре»** — создайте финальную версию в максимальном качестве

Этот подход экономит генерации и время, позволяя быстро итерировать на этапе поиска идеи.

## Ручная настройка

Если предустановленные режимы не подходят для вашей задачи, вы всегда можете настроить параметры вручную. Режим производительности — это удобный способ быстрого старта, но не ограничение.
