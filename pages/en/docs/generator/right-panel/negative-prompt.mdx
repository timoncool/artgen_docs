# Negative Prompt

![Negative Prompt](/images/blog/img_034.png)

Negative prompt specifies what you don't want in the image.

## Purpose

Use negative prompt to:
- Remove unwanted elements
- Improve quality
- Avoid common artifacts

## Common Exclusions

### Quality
```
blurry, low quality, pixelated, jpeg artifacts
```

### Anatomy
```
extra fingers, deformed hands, bad anatomy
```

### Style
```
cartoon, anime, 3d render
```

## Best Practices

- Be specific
- Don't overload with exclusions
- Focus on common problems

## Default Negative Prompt

A default negative prompt is applied automatically. You can customize or disable it.
