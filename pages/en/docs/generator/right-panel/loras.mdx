# LoRA

![LoRA](/images/blog/img_002.png)

LoRA (Low-Rank Adaptation) are additional adapters that modify model behavior.

## What is LoRA?

LoRA allows fine-tuning generation for specific:
- Characters
- Styles
- Concepts
- Objects

## Using LoRA

1. Select LoRA from the list
2. Adjust strength (0.1-1.0)
3. Generate image

## Combining LoRA

You can use multiple LoRA simultaneously, but be careful with compatibility.

## LoRA Strength

- **0.3-0.5** — subtle effect
- **0.6-0.8** — moderate effect
- **0.9-1.0** — strong effect

## Finding LoRA

Browse the LoRA library by category or search by name.
