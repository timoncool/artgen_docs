# Папки

![Folders](/screens/history-panel.png)

Система папок позволяет организовать ваши изображения по любым критериям: проектам, темам, стилям, клиентам. Это особенно полезно при большом количестве генераций.

## Зачем нужны папки

**Организация по проектам** — создайте отдельную папку для каждого проекта и храните все связанные генерации вместе.

**Тематическая сортировка** — группируйте изображения по темам: портреты, пейзажи, абстракции и т.д.

**Рабочий процесс** — создайте папки «В работе», «На доработку», «Готово» для отслеживания статуса.

**Клиентские проекты** — если вы работаете с клиентами, создавайте папки для каждого заказа.

## Создание папки

1. Перейдите на вкладку «Папки» на левой панели
2. Нажмите кнопку «+» или «Создать папку»
3. Введите название папки
4. Нажмите Enter или кнопку подтверждения

Папка создана и готова к использованию.

## Добавление изображений в папку

**Перетаскивание (Drag & Drop):**
- Захватите изображение мышью
- Перетащите его на нужную папку
- Отпустите для добавления

**Контекстное меню:**
1. Нажмите правой кнопкой мыши на изображение
2. Выберите «Переместить в папку»
3. Выберите целевую папку из списка

**Из просмотра изображения:**
1. Откройте изображение в полноразмерном режиме
2. Нажмите на иконку папки
3. Выберите папку назначения

## Структура папок

**Вложенные папки** — создавайте подпапки для более детальной организации. Например: «Проект А» → «Концепты» → «Версия 1».

**Иерархия** — папки отображаются в виде дерева. Нажмите на стрелку рядом с папкой, чтобы развернуть или свернуть вложенные папки.

## Управление папками

**Переименование:**
- Двойной клик на названии папки
- Или правый клик → «Переименовать»
- Введите новое название и нажмите Enter

**Удаление:**
- Правый клик на папке → «Удалить»
- Подтвердите удаление
- Изображения из папки не удаляются — они остаются в истории

**Перемещение:**
- Перетащите папку в другую папку для создания вложенности
- Или используйте контекстное меню

## Просмотр содержимого

Кликните на папку, чтобы увидеть все изображения внутри. Изображения отображаются в виде сетки миниатюр.

**Счётчик** — рядом с названием папки отображается количество изображений внутри.

**Рекурсивный просмотр** — при просмотре папки можно включить отображение изображений из всех вложенных папок.

## Массовые операции

**Выбор нескольких изображений:**
- Удерживайте Ctrl (Cmd на Mac) и кликайте на изображения
- Или используйте Shift для выбора диапазона

**Массовое перемещение:**
- Выберите несколько изображений
- Перетащите их в папку или используйте контекстное меню

## Синхронизация

Структура папок и их содержимое синхронизируются между устройствами. Организуйте изображения на компьютере — изменения отобразятся на всех устройствах.

## Советы по организации

- Создавайте папки заранее перед началом нового проекта
- Используйте понятные и описательные названия
- Не создавайте слишком глубокую вложенность (2-3 уровня достаточно)
- Регулярно сортируйте новые генерации по папкам
- Удаляйте пустые и неиспользуемые папки
