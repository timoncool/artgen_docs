# Быстрые действия

![Quick Actions](/screens/image-actions-toolbar.png)

Быстрые действия — это набор часто используемых функций, доступных одним кликом. Они ускоряют рабочий процесс и позволяют выполнять типичные операции без навигации по меню.

## Расположение

Панель быстрых действий расположена в нижней части левой панели генератора. Она всегда видна и доступна независимо от выбранной вкладки (История, Избранное, Папки).

## Основные действия

**Генерировать** — запускает генерацию с текущими параметрами. Это основная кнопка для создания изображений. Эквивалент горячей клавиши Ctrl+Enter.

**Новая генерация** — очищает все текущие параметры и начинает с чистого листа. Полезно, когда вы хотите начать совершенно новый проект без влияния предыдущих настроек.

**Повторить** — запускает генерацию с теми же параметрами, но с новым seed. Это создаёт вариацию последнего изображения с сохранением всех настроек.

**Случайный промпт** — генерирует случайную идею для промпта. Полезно для вдохновения или когда вы не знаете, что создать.

## Дополнительные действия

**Загрузить изображение** — открывает диалог выбора файла для загрузки изображения. Используется для режима img2img, когда вы хотите создать вариацию существующего изображения.

**Очистить промпт** — удаляет текст из поля промпта, оставляя остальные настройки без изменений.

**Сбросить настройки** — возвращает все параметры к значениям по умолчанию.

## Горячие клавиши

Для максимальной эффективности используйте горячие клавиши:

| Действие | Windows/Linux | Mac |
|----------|---------------|-----|
| Генерировать | Ctrl+Enter | Cmd+Enter |
| Новая генерация | Ctrl+N | Cmd+N |
| Повторить | Ctrl+R | Cmd+R |
| Загрузить изображение | Ctrl+O | Cmd+O |
| Очистить промпт | Ctrl+Backspace | Cmd+Backspace |

## Контекстные действия

Некоторые быстрые действия появляются в зависимости от контекста:

**После генерации** — появляются кнопки «Повторить с новым seed», «Увеличить разрешение», «Сохранить в избранное».

**При выборе изображения** — появляются кнопки «Использовать параметры», «Создать вариацию», «Удалить».

**При загрузке изображения** — появляются кнопки для настройки режима img2img.

## Настройка панели

В настройках генератора вы можете:

- Выбрать, какие действия отображать на панели
- Изменить порядок кнопок
- Настроить размер кнопок
- Включить или отключить подписи

## Советы по использованию

- Запомните основные горячие клавиши для ускорения работы
- Используйте «Повторить» для быстрого создания вариаций
- «Случайный промпт» — отличный способ найти вдохновение
- Настройте панель под свой рабочий процесс
