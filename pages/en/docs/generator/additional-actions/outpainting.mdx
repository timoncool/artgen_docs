# Outpainting

Extend image boundaries beyond the original frame.

## How It Works

The AI analyzes the existing image and generates content that seamlessly continues beyond the edges.

## Extension Directions

- Left
- Right
- Top
- Bottom
- All directions

## Use Cases

- Create wider panoramas
- Add more context
- Change aspect ratio
- Create backgrounds

## Tips

- Extend gradually for best results
- Provide context in the prompt
- Check edge blending quality
