# Дополнительные действия

После генерации изображения доступен ряд дополнительных действий для улучшения и работы с результатом.

## Доступные действия

При наведении на сгенерированное изображение появляется панель с кнопками действий:

- **Своё изображение** — использовать как основу для новой генерации
- **Создать похожее** — сгенерировать вариацию
- **Увеличение 2x** — улучшить разрешение
- **Восстановление лица** — улучшить качество лиц
- **Расширение границ** — добавить контент за пределами изображения
- **Копировать ссылку** — получить ссылку на изображение
- **Скачать** — сохранить на устройство
- **Полноэкранный режим** — просмотр в полном размере
