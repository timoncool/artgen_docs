# Your Image

![Your Image](/screens/own-image-panel.png)

The "Your Image" function allows you to use a generated or uploaded image as a basis for a new generation (img2img).

## How to Use

1. Hover over an image in the generation results
2. Click the "Your Image" button
3. The image will be added to the settings panel
4. Adjust the influence strength of the source image
5. Enter a new prompt or modify the existing one
6. Start generation

## Influence Strength

The "Influence Strength" parameter determines how similar the result will be to the source image:

- **0.1-0.3** — minimal changes, structure preserved
- **0.4-0.6** — moderate changes, balance between original and prompt
- **0.7-0.9** — significant changes, prompt dominates

## Applications

- Creating variations of a liked image
- Changing the style of an existing image
- Adding or removing elements
- Improving composition
