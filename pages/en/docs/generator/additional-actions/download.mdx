# Download

![Download](/screens/image-actions-toolbar.png)

The download function allows you to save an image to your device.

## How to Download

1. Hover over an image
2. Click the "Download" button
3. The file will be downloaded to your downloads folder

## File Format

Images are downloaded in PNG format with maximum quality.

## File Name

The file name contains:
- Generation identifier
- Creation date
- .png extension

## Usage Rights

Downloaded images can be used:
- In personal projects
- For commercial purposes (with appropriate subscription)
- For publication with source attribution

## Tips

- Download images after applying enhancements (upscale, restoration)
- Save liked works as they may be deleted over time
- Use the bulk download function in your account for multiple images
