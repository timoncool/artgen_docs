# Fullscreen Mode

![Fullscreen Mode](/screens/single-image-view.png)

Fullscreen mode allows you to view an image at maximum size.

## How to Activate

1. Hover over an image
2. Click the fullscreen button (or double-click the image)
3. The image will open in fullscreen

## Features

In fullscreen mode you can access:
- Maximum resolution viewing
- Zoom in/out
- Pan across the image
- Navigate between images

## Controls

- **Arrows** — go to next/previous image
- **+/-** or mouse wheel — zoom
- **Drag** — pan across zoomed image
- **Escape** or click outside image — exit

## When to Use

- For detailed result examination
- For quality check before downloading
- For showcasing works
- For comparing details of different generations
