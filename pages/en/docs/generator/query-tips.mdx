# Советы по запросам

Практические рекомендации для создания эффективных промптов.

## Будьте конкретны

Чем детальнее описание, тем точнее результат:

**Плохо:** `кот`

**Хорошо:** `пушистый рыжий кот с зелёными глазами, сидит на подоконнике, солнечный свет, уютная атмосфера`

## Используйте описательные прилагательные

Добавляйте качественные характеристики:
- Цвета: яркий, насыщенный, пастельный
- Освещение: мягкое, драматичное, контрастное
- Настроение: уютный, мрачный, радостный

## Указывайте стиль

Определите художественное направление:
- Фотореалистичный
- Цифровая живопись
- Аниме
- Масляная живопись
- Акварель

## Добавляйте технические детали

Для качественных результатов укажите:
- Разрешение: 4k, 8k, высокое качество
- Детализация: детальный, чёткий, резкий
- Рендер: octane render, unreal engine

## Экспериментируйте

Не бойтесь пробовать разные формулировки. Иногда небольшое изменение в запросе даёт совершенно другой результат.

## Изучайте чужие работы

Смотрите промпты успешных генераций в галерее и учитесь на них.
