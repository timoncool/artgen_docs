## Update 6.0

**Date:** February 21, 2025

![Bulk Operations](/blog/update-6.0-1.png)

### 1. Bulk Image Operations

Bulk file operations are available in personal cabinet. Select them with checkboxes and delete using a convenient popup panel. Additional file operations planned for future releases.

### 2. Date Folders

Now all your images are available for viewing in folders with generation dates. You can view or delete entire folders at once. Pagination and bulk actions available.

### 3. Bulk Image Downloads

New settings option allows requesting image archives for specific periods. Processing time depends on image quantity. Download links are sent to email and remain active for 72 hours. Only one active archive allowed at a time.

### 4. Dedicated Image Editor

The image editor got a separate page at artgeneration.me/editor. Allows quick image editing — upload from device or select previously generated, then download or save to gallery.

![Style Combination](/blog/update-6.0-2.png)

We've collected probably all unique styles created by humanity — 307 in total. Each style includes 5 preview variations for accurate visualization. Named categories and better organization coming soon.
