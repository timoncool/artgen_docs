# ArtGeneration.me - Документация

Документация для платформы [ArtGeneration.me](https://artgeneration.me) - сервиса генерации изображений с помощью искусственного интеллекта.

## Как редактировать страницы

Любой пользователь может предложить изменения в документацию! Это помогает поддерживать информацию актуальной.

### Способ 1: Редактирование через GitHub (простой)

1. Откройте страницу документации, которую хотите отредактировать
2. Нажмите ссылку "Редактировать страницу на GitHub" внизу страницы
3. GitHub откроет редактор файла
4. Внесите изменения в текст (используется формат MDX/Markdown)
5. Нажмите "Commit changes" и опишите что вы изменили
6. GitHub автоматически создаст Pull Request с вашими изменениями
7. После проверки изменения будут опубликованы

### Способ 2: Локальное редактирование (для разработчиков)

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/timoncool/artgen_docs.git
   cd artgen_docs
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите локальный сервер:
   ```bash
   npm run dev
   ```

4. Откройте http://localhost:3000 в браузере

5. Редактируйте файлы в папке `pages/`:
   - `pages/ru/` - русская версия
   - `pages/en/` - английская версия

6. Создайте Pull Request с вашими изменениями

## Структура проекта

```
pages/
├── ru/                      # Русская версия
│   ├── docs/                # Руководство пользователя
│   │   ├── generator/       # Генератор изображений
│   │   ├── gallery/         # Галерея
│   │   ├── editor/          # Редактор
│   │   ├── faq.mdx          # Часто задаваемые вопросы
│   │   └── rules.mdx        # Правила сервиса
│   ├── news/                # Новости (автозагрузка из папки)
│   │   ├── index.jsx        # Лента новостей
│   │   └── *.mdx            # Отдельные новости
│   └── about-docs/          # Работа над документацией
│       ├── contributing.mdx # Как редактировать
│       └── contributors.mdx # Авторы
└── en/                      # English version (аналогичная структура)
```

## Формат файлов

Страницы написаны в формате MDX (Markdown + JSX). Основные элементы:

```mdx
# Заголовок страницы

Обычный текст параграфа.

## Подзаголовок

- Элемент списка
- Еще один элемент

![Описание изображения](/images/example.png)

[Текст ссылки](https://example.com)
```

## Добавление изображений

1. Добавьте изображение в папку `public/screens/` или `public/blog/`
2. Используйте в MDX файле:
   ```mdx
   ![Описание](/screens/your-image.png)
   ```
