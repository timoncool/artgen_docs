1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/vitalii-siziakov/artgen_docs.git
   cd artgen_docs
   ```

2. Установите зависимости:
   ```bash
   yarn install
   ```

3. Копировать env
   ```bash
   cp .env.example .env
   ```

4. Запустите локальный сервер:
   ```bash
   yarn dev
   ```
